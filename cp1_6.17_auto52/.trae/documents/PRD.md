## 1. 产品概述

3D蛋白质分子对接与折叠模拟应用，用于生物教学和科普场景，帮助学生直观理解蛋白质折叠和分子对接过程。通过交互式3D可视化，用户可拖拽配体分子与受体蛋白进行对接模拟，实时观察空间匹配机制。

目标用户：生物教师、学生、科普爱好者。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 浏览和使用所有功能 |

### 2.2 功能模块

1. **主场景页面**：3D分子渲染、交互控制、对接模拟
2. **分子信息面板**：实时显示分子参数、对接结果
3. **底部工具条**：分子选择、温度调节、对接控制

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-----------|----------------|
| 主场景页面 | 3D渲染模块 | Three.js实时渲染5000原子球棍模型，支持旋转缩放 |
| 主场景页面 | 分子拖拽模块 | 鼠标拖拽配体分子，实时碰撞检测 |
| 主场景页面 | 对接动画模块 | 距离小于2.5单位触发对接动画，蓝色光环脉冲 |
| 主场景页面 | 温度模拟模块 | 0-100温度调节，控制振动幅度和旋转速度 |
| 信息面板 | 参数显示模块 | 原子数、键数、分子量、结合能、关键残基 |
| 工具条 | 分子选择器 | 下拉切换预设分子对（至少2组） |
| 工具条 | 温度滑块 | 0-100温度调节，实时显示数值 |
| 工具条 | 对接按钮 | 一键触发对接 |
| 工具条 | 重置视角按钮 | 恢复初始视角 |

## 3. 核心流程

用户打开应用 → 默认加载新冠病毒刺突蛋白受体和小分子配体 → 鼠标拖拽配体靠近受体 → 碰撞检测实时计算距离 → 距离<2.5单位触发对接 → 配体弹入结合位点 → 蓝色光环脉冲动画 → 显示结合能和关键残基 → 调节温度观察振动变化

## 4. 用户界面设计

### 4.1 设计风格

- 主背景：#0D0D0D 深黑色
- 辅背景：#2C2C2C 深灰色
- 强调色：#FF7043 橙红色、#43A047 绿色
- 文字：白色、#C0C0C0 浅灰色
- 字体：等宽数字字体用于数据显示，现代无衬线字体用于普通文本
- 圆角：主元素6px，面板12px
- 过渡动画：0.2秒平滑过渡

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|-----------|----------|
| 主场景 | 3D画布 | 全屏渲染，深色背景，分子球棍模型 |
| 右侧面板 | 信息面板 | 半透明深灰#1E1E2E，80%不透明度，圆角12px，宽度280px |
| 底部工具条 | 控制条 | 高度60px，背景#2C2C2C 80%不透明度，两端圆角8px |
| 工具条 | 分子下拉选择器 | 宽度160px，高度36px，背景#3A3A3A |
| 工具条 | 温度滑块 | 宽度200px，轨道高度6px，滑块直径16px，颜色#FF7043 |
| 工具条 | 对接按钮 | 宽度100px，高度36px，线性渐变#43A047到#66BB6A |
| 右上角 | 重置按钮 | 圆形直径40px，背景#FF7043，悬停放大至48px |

### 4.3 响应式设计

- **大屏幕（>1200px）：固定宽度居中，左侧75%场景，右侧25%面板
- **中等屏幕（768-1200px）：左侧65%场景，右侧35%面板
- **小屏幕（<768px）：右侧面板折叠为底部可收起抽屉

### 4.4 3D场景设计

- **环境**：深色背景，点光源+环境光组合
- **灯光**：双点光源（白色+淡蓝色），环境光（弱白色）
- **相机**：PerspectiveCamera，初始距离原点8单位
- **原子样式**：球棍模型，原子球按范德华半径，半透明圆柱键
- **后处理**：抗锯齿，无额外后处理以保证性能
- **性能预算**：帧率≥45FPS，内存≤300MB
