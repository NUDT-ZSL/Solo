## 1. 产品概述

折纸工坊是一款交互式纸艺网格应用，让用户通过折叠、旋转和平移虚拟纸张来创作折纸图案，并支持导出为PDF打印模板。目标用户为折纸爱好者、手工达人和创意设计者，通过数字化方式预览和规划折纸作品。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 使用所有折纸设计与导出功能 |

### 2.2 功能模块
1. **主画布页面**：纸艺网格画布、折叠操作、旋转平移控制
2. **工具栏**：选择工具、折叠工具、旋转工具、导出按钮
3. **历史面板**：折叠历史列表、状态回退、纸张统计

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主画布页面 | 纸艺网格画布 | 400x400像素正方形纸张，网格线渲染，折痕绘制，折叠动画 |
| 主画布页面 | 折叠操作 | 点击两个格点绘制红色虚线折痕，点击第三格点执行折叠 |
| 主画布页面 | 旋转控制 | 滑块0-360度旋转纸张，旋转时显示淡蓝色描边 |
| 主画布页面 | 平移控制 | 水平/垂直滑块控制偏移，阴影方向同步变化 |
| 工具栏 | 工具选择 | 选择/折叠/旋转三种工具模式，改变光标样式 |
| 工具栏 | 导出按钮 | 生成PDF，弹出纸张尺寸设置模态框 |
| 历史面板 | 折叠历史 | 显示折叠序号和时间，点击回退并播放逆向动画 |
| 历史面板 | 统计信息 | 显示当前面积和总折叠次数 |

## 3. 核心流程

用户打开应用后，默认看到居中的正方形纸张和网格。选择折叠工具后，依次点击两个格点绘制折痕，点击第三个格点完成折叠操作。可通过旋转和平移滑块调整纸张视角。每次折叠自动记录到历史列表，点击历史项可回退到对应状态。完成后点击导出按钮生成PDF打印模板。

## 4. 用户界面设计

### 4.1 设计风格
- 主背景色：浅灰色 #f5f6fa
- 卡片背景：纯白色 #ffffff，圆角10px，阴影 0 2px 10px rgba(0,0,0,0.08)
- 纸张：白色 #ffffff，5px阴影 0 2px 12px rgba(0,0,0,0.15)
- 网格线：浅灰 #d3d3d3（细线每20px，粗线每100px）
- 折痕线：红色 #e74c3c，2px宽虚线
- 旋转描边：淡蓝 #3498db，1px
- 导出按钮：蓝绿 #1abc9c，白色文字 #ffffff，圆角6px
- 当前状态高亮：#ecf0f1
- 按钮微放缩动画：0.15s ease-out，scale 1.02
- 折叠动画：0.4s cubic-bezier(0.4, 0, 0.2, 1)
- 字体：选择具有手工/纸艺感的字体

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主画布页面 | 纸艺网格画布 | 居中Canvas，网格线，折痕线，折叠多边形 |
| 工具栏 | 左侧工具面板 | 220px宽白色卡片，工具按钮，滑块，导出按钮 |
| 历史面板 | 右侧历史面板 | 300px宽白色卡片，历史列表，统计信息 |

### 4.3 响应式
- 桌面优先设计，三栏布局（工具栏220px + 画布 + 面板300px）
- 页面宽度 < 900px 时切换为上下布局
- 工具栏变为顶部水平导航栏
- 状态面板变为底部可折叠面板
