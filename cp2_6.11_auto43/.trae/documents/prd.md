## 1. 产品概述

情绪轨迹图是一个帮助用户记录、可视化和分享每日情绪的全栈Web应用。用户每天选择一种颜色和一段文字描述自己的心情，系统自动生成由彩色圆点和贝塞尔曲线构成的沉浸式情绪轨迹图，形成随时间流淌的抽象情绪河流。同时支持社交互动——朋友可以给轨迹添加「回声」评论，以微光动画附着在对应日期上。

- 解决数字日记缺乏可视化表达和社交互动的问题
- 目标用户：希望通过视觉化方式记录和分享情绪的年轻群体

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，默认用户 | 记录情绪、查看轨迹、分享轨迹、添加回声 |
| 访客 | 通过分享链接 | 查看轨迹、添加回声 |

### 2.2 功能模块
1. **首页**：30天情绪轨迹时间轴、添加今日心情、轨迹交互
2. **个人主页**：用户信息、情绪统计面板（饼图+折线图）
3. **分享页面**：轨迹卡片生成与回声添加

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 情绪轨迹时间轴 | 横向滚动时间轴，Canvas绘制贝塞尔曲线连接的情绪圆点，左右箭头平滑滚动 |
| 首页 | 添加心情浮窗 | 点击空白处或浮动按钮弹出模态框，包含12色选择器和50字文字输入 |
| 首页 | 圆点详情卡片 | 点击圆点弹出详情卡片，含日期、文字、光晕动画，支持编辑和删除 |
| 首页 | 分享按钮 | 点击纸飞机图标生成分享卡片，复制链接到剪贴板 |
| 个人主页 | 用户信息区 | 头像、用户名、记录天数 |
| 个人主页 | 统计面板 | 毛玻璃卡片，包含月度情绪分布饼图和周情绪趋势折线图 |
| 分享页面 | 轨迹查看 | 访客查看完整30天轨迹，可添加回声评论 |

## 3. 核心流程

用户每日打开应用，在首页看到最近30天的情绪轨迹。点击浮动按钮或空白处创建当日情绪记录（选颜色+写文字），记录后轨迹实时更新。点击任意圆点查看详情，可编辑或删除。点击分享按钮生成卡片并复制链接。朋友打开链接后可查看轨迹并添加回声，回声以小圆点+波纹动画附着在对应日期旁。

## 4. 用户界面设计

### 4.1 设计风格
- 主色：暖杏色（#FADDC6）到淡薰衣草（#E8D0F0）全屏渐变
- 辅助色：12种预设情绪色圆环
- 毛玻璃效果：backdrop-filter: blur(12px)，半透明白色边框
- 字体：中文使用系统默认，英文使用特色字体
- 布局：横向时间轴为主视觉，卡片式模态交互
- 圆角风格：统一12-16px圆角
- 动画：贝塞尔曲线平滑过渡，脉冲动画，波纹动画，光晕扩散

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 时间轴区域 | 全屏渐变背景、毛玻璃卡片(圆角16px、模糊12px)、Canvas轨迹图(贝塞尔曲线+圆点)、左右箭头、浮动添加按钮(56px) |
| 首页 | 创建模态框 | 居中400px宽、毛玻璃背景、12色选择器(脉冲动画)、文字输入(50字实时计数)、保存按钮(悬停变色) |
| 首页 | 详情卡片 | 定位在圆点上方20px、三角箭头、毛玻璃(模糊10px)、光晕扩散动画0.5s、编辑/删除按钮 |
| 首页 | 分享卡片 | 800×500px、渐变背景、完整轨迹、水印、复制链接toast |
| 个人主页 | 用户信息 | 圆形头像(48px)、用户名+天数 |
| 个人主页 | 统计面板 | 毛玻璃卡片(280px宽)、饼图(12色)、折线图(渐变线) |
| 分享页面 | 回声添加 | 文本输入+颜色选择、8px小圆点+波纹动画0.6s |

### 4.3 响应式设计
- 桌面优先设计，时间轴横向滚动适配宽屏
- 移动端时间轴保持横向滚动，模态框自适应宽度
- 触屏支持：拖拽圆点、点击交互

### 4.4 性能要求
- 首次加载 ≤ 3秒（3G模拟）
- 轨迹绘制帧率 ≥ 30fps
- 分享卡片生成 ≤ 1秒
- CSS GPU加速：will-change、transform、backdrop-filter使用合成层
