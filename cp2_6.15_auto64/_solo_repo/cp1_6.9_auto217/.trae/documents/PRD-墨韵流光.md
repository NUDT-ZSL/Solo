## 1. 产品概述

「墨韵流光」是一款融合中国传统水墨画与数字流光特效的交互式Web绘图应用。用户可通过鼠标或触控笔在画布上创作，笔触呈现水墨晕染扩散效果，同时伴随流动彩色光痕，最终形成动态水墨画卷。

- 核心价值：将东方传统水墨艺术与现代数字视觉效果结合，提供沉浸式创作体验
- 目标用户：艺术爱好者、设计师、对东方美学感兴趣的普通用户

## 2. 核心功能

### 2.1 功能模块
1. **画布绘制模块**：水墨晕染笔触、实时绘制、扩散算法
2. **流光粒子模块**：彩色粒子生成、沿轨迹流动、拖尾效果
3. **工具控制模块**：笔刷大小调节、颜色选择器、清除画布
4. **历史记录模块**：撤销/重做操作（最多20步）
5. **导出模块**：PNG图片导出（1920x1080分辨率）

### 2.2 功能详情

| 模块名称 | 子功能 | 功能描述 |
|---------|--------|----------|
| 水墨绘制 | 笔触绘制 | 鼠标/触控笔拖拽时实时绘制线条 |
| 水墨绘制 | 晕染扩散 | 线条边缘不规则向外扩散5-15像素，持续约1秒 |
| 流光粒子 | 粒子生成 | 绘制线条时释放彩色粒子串 |
| 流光粒子 | 渐变流动 | 粒子颜色从主色渐变至互补色，速度60单位/秒 |
| 流光粒子 | 拖尾效果 | 粒子拖尾约20像素，大小随机2-5像素 |
| 工具栏 | 笔刷滑块 | 1-50像素调节，默认10像素 |
| 工具栏 | 颜色选择 | 8种传统中国色预设 |
| 工具栏 | 清除按钮 | 圆形扩散动画清除画布 |
| 历史记录 | 撤销 | Ctrl+Z，同步回退线条和粒子，淡出0.3秒 |
| 历史记录 | 重做 | Ctrl+Shift+Z，最多20步 |
| 导出 | PNG导出 | 1920x1080分辨率，进度动画提示 |

## 3. 核心流程

## 4. 用户界面设计

### 4.1 设计风格
- **设计理念**：深色极简东方美学，禅意留白
- **主色调**：背景渐变 `#0a0a0a` → `#1a1a1a`（黑至深灰）
- **强调色**：金色渐变 `#d4a853` → `#b8860b`（导出按钮）
- **传统色板**：墨黑、朱砂、藤黄、石青、石绿、胭脂、赭石、钛白
- **控件样式**：圆角8px，磨砂玻璃 `rgba(255,255,255,0.05)`，边框 `1px rgba(255,255,255,0.2)`
- **悬停效果**：背景变亮至 `rgba(255,255,255,0.1)`，平滑过渡
- **字体**：衬线字体体现东方韵味，按钮使用现代无衬线

### 4.2 页面设计概述

| 页面区域 | 模块名称 | UI元素与风格 |
|---------|---------|-------------|
| 背景层 | 渐变背景 | 黑到深灰线性渐变，营造深夜意境 |
| 主体区 | Canvas画布 | 居中90%宽度，全屏高度，半透明边框 |
| 左侧栏 | 垂直工具栏 | 毛玻璃效果，垂直排列，间距均匀 |
| 左侧栏 | 笔刷滑块 | 自定义样式滑块，数值显示 |
| 左侧栏 | 颜色选择器 | 圆形色块网格，选中高亮描边 |
| 左侧栏 | 清除按钮 | 磨砂背景，悬停变亮 |
| 右上角 | 导出按钮 | 金色渐变+微光扫光动画 |
| 弹出层 | 导出提示 | 半透明遮罩+进度动画 |

### 4.3 响应式设计
- **桌面端（默认）**：左侧垂直工具栏，画布占90%宽度
- **移动端**：全屏竖版布局，工具栏移至底部横向排列，控件间距自适应缩小
- **触控优化**：增大点击区域，支持触控笔压感（如可用）

### 4.4 动效设计
- 导出按钮：金色流光周期性横向扫过
- 画布清除：圆形波纹从中心扩散至边缘
- 撤销/重做：0.3秒透明度淡出淡入
- 控件悬停：0.2s平滑过渡，背景亮度提升
