## 1. 产品概述

WireSplitter 是一款面向 UI/UX 设计师的线框图智能分割与原型生成工具，解决设计师在手绘或制作低保真线框图后需手动拆分并编码原型、导致迭代反馈周期过长的痛点。

- 目标用户：UI/UX 设计师、产品经理、前端开发者
- 核心价值：将线框图截图自动识别为独立 UI 组件，一键生成可交互 HTML 原型，将线框图到原型的转化时间从数小时缩短至秒级

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 设计师 | 直接访问 | 上传、分割、预览、调整、导出 |
| 产品经理 | 直接访问 | 上传、预览、导出 |

### 2.2 功能模块

1. **上传页**: 图像上传区域、拖拽上传、缩略图预览
2. **工作台页**: 左侧原始线框图参考 + 右侧交互原型预览 + 底部 JSON 结构树

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 上传页 | 图像上传区 | 支持 PNG/JPEG 拖拽上传与点击选择，校验分辨率≥1200×800px，显示缩略图 |
| 工作台页 | 参考图面板 | 左侧显示原始线框图，可折叠，折叠后仅保留右侧预览 |
| 工作台页 | 原型预览面板 | 右侧渲染可交互 HTML 原型，组件悬停高亮+标签，点击脉冲动画，支持手动调整 |
| 工作台页 | 组件调整工具 | 拖拽角点调整组件大小，右键菜单（重新识别/合并/拆分/替换类型） |
| 工作台页 | JSON 结构树面板 | 底部折叠面板展示组件树 JSON，支持导出 JSON 文件 |

## 3. 核心流程

用户打开应用 → 上传线框图截图 → 系统基于 Canvas 像素分析自动识别组件边界 → 生成组件结构树 → 进入工作台预览模式 → 用户可手动修正分割结果 → 导出 JSON 结构

```mermaid
flowchart TD
    "A[用户上传线框图]" --> "B[Canvas 像素分析]"
    "B" --> "C[识别矩形/圆形/文本/容器边界]"
    "C" --> "D[构建组件结构树]"
    "D" --> "E[进入工作台预览模式]"
    "E" --> "F{用户是否需修正?}"
    "F" --> "是" --> "G[拖拽调整/右键菜单操作]"
    "G" --> "H[同步更新结构树]"
    "H" --> "F"
    "F" --> "否" --> "I[导出 JSON]"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主题色：#4A90D9（主题蓝）
- 深色主题：背景 #1E1E2E，卡片 #2A2A3E，主文本 #E0E0E0
- 组件边框：1px solid #ddd，圆角 8px
- 字体：系统字体栈，标签 12px
- 布局：左右分栏（50%/50%），可折叠左侧面板
- 动画：过渡 200ms ease-out，面板展开/折叠 300ms ease-in-out，脉冲高亮 500ms

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 上传页 | 图像上传区 | 深色卡片居中，虚线边框拖拽区，缩略图预览，分辨率提示 |
| 工作台页 | 参考图面板 | 左侧50%宽度，顶部工具栏含折叠按钮，线框图缩放适配显示 |
| 工作台页 | 原型预览面板 | 右侧50%宽度，组件div包裹，悬停蓝色边框+右上角标签，点击脉冲光晕 |
| 工作台页 | 调整手柄 | 四角8px直径小圆点，拖拽时半透明预览 |
| 工作台页 | 右键菜单 | 浮动菜单：重新识别/合并到上一个/拆分为左右两个/替换类型 |
| 工作台页 | JSON面板 | 底部折叠面板，深色代码区域，导出按钮 |

### 4.3 响应式适配

- 桌面端（≥768px）：左右分栏布局
- 移动端（<768px）：上下叠放布局
- 左侧面板可折叠，折叠后右侧预览占满宽度

### 4.4 组件类型定义

| 组件类型 | 识别特征 | 预览渲染 |
|----------|----------|----------|
| button | 小矩形、圆角 | 带文字的按钮样式 |
| input | 窄长矩形、底部有线 | 输入框样式 |
| card | 较大矩形、有内边距 | 卡片容器样式 |
| navbar | 顶部横向矩形 | 导航栏样式 |
| container | 大面积矩形、包含子组件 | 虚线边框容器 |
| text | 文本区域 | 文字段落样式 |
| image | 矩形内有图标或图形 | 图片占位样式 |
