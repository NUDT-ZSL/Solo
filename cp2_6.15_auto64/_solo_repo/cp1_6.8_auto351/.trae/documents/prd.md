## 1. 产品概述

「浮光绘影」是一款交互式光影绘画 Web 应用，用户在虚拟画布上用鼠标拖拽生成动态光痕，光痕随时间缓慢衰减消失，拖拽速度越快光痕越亮越粗，并带有渐变色彩和粒子拖尾效果。

- 面向创意工作者、数字艺术爱好者和休闲用户，提供即开即用的沉浸式光影绘画体验
- 以极致视觉表现力和流畅交互体验为核心竞争力，打造浏览器中的数字光影艺术工具

## 2. 核心功能

### 2.1 用户角色

无需角色区分，所有用户均为匿名创作者。

### 2.2 功能模块

1. **主画布页面**：全屏 Canvas 光影画布、动态光痕绘制、粒子拖尾系统、光痕衰减动画
2. **控制面板**：画笔颜色选择器、画笔大小滑块、光痕衰减速度滑块、清空画布按钮、录制导出视频按钮

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主画布页面 | 光痕绘制 | 鼠标按下拖拽时生成光痕，速度越快光痕越亮越粗，颜色从暖色（红橙）渐变到冷色（蓝紫） |
| 主画布页面 | 粒子拖尾 | 拖拽时在光痕周围生成随机偏移的粒子，粒子逐渐消散 |
| 主画布页面 | 光痕衰减 | 停止拖拽后光痕自动衰减消失，衰减过程中颜色变暗并缩小 |
| 主画布页面 | Canvas 渲染引擎 | requestAnimationFrame 驱动的 60fps 渲染循环，管理光痕和粒子生命周期 |
| 控制面板 | 画笔颜色选择器 | 选择光痕的基础色调 |
| 控制面板 | 画笔大小滑块 | 调节光痕的基础宽度 |
| 控制面板 | 衰减速度滑块 | 调节光痕消失的快慢 |
| 控制面板 | 清空画布按钮 | 一键清除所有光痕和粒子 |
| 控制面板 | 录制按钮 | 开始/停止录制画布内容，导出为视频文件 |

## 3. 核心流程

用户打开应用 → 看到深色画布 → 在画布上按住鼠标拖拽 → 生成动态光痕（颜色渐变、粒子拖尾）→ 松开鼠标 → 光痕开始衰减 → 衰减完毕光痕消失 → 用户可调整控制面板参数 → 可录制并导出视频

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深灰到纯黑渐变背景（#0a0a0a → #000000），光痕以暖色到冷色渐变（红橙 → 蓝紫）
- **按钮风格**：圆角半透明按钮，悬停时带柔和光晕和微缩放动画
- **字体**：使用思源黑体/Noto Sans SC 作为 UI 字体，轻量优雅
- **布局风格**：画布占满全屏主体区域，控制面板悬浮在右下角
- **图标风格**：极简线性图标，低饱和度白色

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 主画布页面 | Canvas 画布 | 全屏黑色画布，占满视口，光痕在画布上绘制 |
| 主画布页面 | 控制面板 | 半透明毛玻璃效果卡片，悬浮右下角，包含滑块、颜色选择器、按钮 |
| 控制面板 | 画笔颜色选择器 | 圆形色盘或预设色板 |
| 控制面板 | 画笔大小滑块 | 自定义样式的范围滑块 |
| 控制面板 | 衰减速度滑块 | 自定义样式的范围滑块 |
| 控制面板 | 清空画布按钮 | 圆角半透明按钮，悬停光晕 |
| 控制面板 | 录制按钮 | 圆角半透明按钮，录制中显示红色脉冲指示 |

### 4.3 响应式设计

- 桌面端优先，画布自适应窗口大小
- 控制面板在移动端可收起为图标按钮
- 触屏设备支持触摸拖拽绘画

### 4.4 动效设计

- 光痕生成：从中心向外辐射，亮度与速度正相关
- 光痕衰减：颜色变暗、宽度缩小、透明度降低直至消失
- 粒子系统：从光痕中心向四周扩散，逐渐缩小消散
- 控制面板：淡入淡出切换，按钮悬停光晕 + 微缩放
- 录制指示：红色脉冲圆点动画
