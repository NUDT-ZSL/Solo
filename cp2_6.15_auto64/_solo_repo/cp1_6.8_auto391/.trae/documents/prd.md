## 1. 产品概述

「情绪色谱日记」是一款将情绪视觉化的全栈日记应用，用户每天记录带情绪标签的日记，系统根据情绪关键词自动生成抽象艺术色块，让心情变成可回望的色彩记忆。
- 解决传统日记缺乏情绪可视化、难以回顾心境变化的问题
- 面向注重情绪管理与自我觉察的年轻用户群体，提供疗愈系日记体验

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 简易注册/登录 | 创建、编辑、删除日记，查看历史与情绪曲线 |

### 2.2 功能模块
1. **日记页面**：日记撰写/编辑、情绪关键词选择与强度调节、实时情绪色块预览
2. **历史页面**：日记时间线列表、按情绪类型筛选、心情曲线图、色块缩略图展示

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 日记页面 | 情绪选择器 | 预设情绪关键词（快乐、悲伤、平静、焦虑、愤怒、感恩等），支持自定义强度滑块（1-10） |
| 日记页面 | 日记编辑区 | 富文本输入区域，支持日期选择，自动保存草稿 |
| 日记页面 | 色块预览 | 根据情绪类型和强度实时生成抽象渐变色块动画，作为日记卡片背景 |
| 历史页面 | 情绪筛选栏 | 胶囊按钮组筛选不同情绪类型，带平滑过渡动画 |
| 历史页面 | 日记列表 | 毛玻璃卡片列表，每张卡片显示日期、情绪标签、色块缩略图、日记摘要 |
| 历史页面 | 心情曲线 | 折线图展示近期情绪强度变化趋势，悬停显示详情 |

## 3. 核心流程

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：浅灰到白色渐变背景（#f5f5f7 → #ffffff），情绪色块采用柔和渐变
- **情绪色谱映射**：快乐→暖橙黄、悲伤→靛蓝紫、平静→薄荷绿、焦虑→琥珀橙、愤怒→深红、感恩→玫瑰金
- **按钮风格**：圆角胶囊按钮，微光悬停效果（subtle glow），过渡动画 300ms
- **字体**：正文使用 Noto Sans SC，标题使用 Playfair Display，营造柔和高雅感
- **布局风格**：单栏居中布局，日记卡片为毛玻璃圆角面板（backdrop-blur）
- **导航栏**：半透明磨砂质感，固定顶部
- **动效**：色块渐变带轻微呼吸动画，筛选过渡使用 CSS transition + opacity/transform

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 日记页面 | 情绪选择器 | 胶囊按钮组，选中态发光，柔和渐变底色 |
| 日记页面 | 强度滑块 | 自定义圆角滑块，轨道随情绪色渐变 |
| 日记页面 | 日记编辑区 | 毛玻璃圆角文本域，半透明白色背景 |
| 日记页面 | 色块预览 | Canvas 绘制抽象渐变色块，CSS 动画呼吸效果 |
| 历史页面 | 筛选栏 | 胶囊按钮组，激活态带色块指示 |
| 历史页面 | 日记卡片 | 毛玻璃圆角面板，左侧色块缩略条，右侧摘要 |
| 历史页面 | 心情曲线 | SVG 折线图，渐变填充区域，圆点标记 |

### 4.3 响应式适配
- 桌面端（≥1024px）：双栏布局，左侧日记编辑 + 右侧色块预览；历史页面三列卡片
- 平板端（768-1023px）：单栏布局，色块预览叠在编辑区上方；历史页面双列卡片
- 手机端（<768px）：全宽单栏，紧凑间距；历史页面单列卡片，心情曲线全宽
- 触控优化：按钮最小触控区域 44px，滑块加大触控范围

### 4.4 性能目标
- 交互响应流畅，帧率稳定在 60fps
- 色块动画使用 CSS transform + opacity 优先触发 GPU 加速
- 长列表使用虚拟滚动避免卡顿
