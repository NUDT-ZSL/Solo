## 1. 产品概述

「气味漂流瓶」是一个匿名气味笔记分享平台，用户将气味记忆封入漂流瓶，在虚拟"气味海洋"中随机漂流，通过共鸣机制实现情感连接。
- 解决现代人气味记忆无处分享的孤独感，让气味记忆在陌生人之间流动和共鸣
- 目标用户为喜欢记录生活细节、追求感官体验的年轻人群体，具有治愈与社交双重价值

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 匿名用户 | 无需注册，浏览器本地标识 | 投放漂流瓶、浏览海洋、共鸣操作、查看我的瓶子 |

### 2.2 功能模块
1. **气味海洋首页**: 漂流瓶海洋渲染、投放入口、瓶子交互（悬停/点击/共鸣）
2. **我的漂流瓶页**: 已发送瓶子列表、已共鸣瓶子列表

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 气味海洋 | 投放漂流瓶 | 输入150字气味笔记、选择气味标签、提交生成漂流瓶 |
| 气味海洋 | 漂流瓶海洋 | Canvas渲染大量半透明漂流瓶，缓慢漂浮动画，呼吸光晕效果 |
| 气味海洋 | 瓶子交互 | 鼠标悬停放大显示标签前两字，点击触发水花粒子+弹出毛玻璃详情卡 |
| 气味海洋 | 共鸣操作 | 点击共鸣按钮，用户漂流瓶飞向目标瓶融合，颜色渐变，共鸣数+1 |
| 我的漂流瓶 | 已发送列表 | 网格布局展示用户投放的所有漂流瓶 |
| 我的漂流瓶 | 已共鸣列表 | 网格布局展示用户共鸣过的漂流瓶，含共鸣数 |

## 3. 核心流程

用户进入气味海洋首页 → 看到大量漂流瓶漂浮 → 点击底部"+"投放 → 输入气味笔记并选择标签 → 提交后瓶子落入海洋 → 浏览海洋中其他瓶子 → 点击瓶子查看详情 → 点击共鸣 → 飞行融合动画 → 共鸣数增加 → 切换到"我的漂流瓶"查看历史

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深蓝(#0a0e27)到紫黑(#1a0a2e)渐变背景，营造深海夜空氛围
- 强调色：各气味标签对应颜色（泥土棕#8B6914、樟木黄#C4A35A、海风蓝#4A9BD9等）
- 瓶子样式：发光半透明瓶形，带呼吸光晕动画（脉动glow），缓动漂浮（正弦曲线运动）
- 按钮样式：圆角毛玻璃按钮，带微光边框
- 字体：标题用"ZCOOL KuaiLe"或"Ma Shan Zheng"等有手写感的中文艺术字体，正文用"Noto Sans SC"保证可读性
- 布局：全屏Canvas海洋 + 底部固定导航 + 浮层弹窗
- 图标/风格：漂流瓶为发光玻璃瓶轮廓，水花用粒子飞溅，海洋用波纹光效

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 气味海洋 | 海洋背景 | 深蓝紫黑渐变，Canvas全屏渲染 |
| 气味海洋 | 漂流瓶群 | 半透明发光瓶形图标，呼吸光晕，缓动漂浮，标签色着色 |
| 气味海洋 | 投放入口 | 底部中央"+"按钮，毛玻璃圆形，脉动动画引导 |
| 气味海洋 | 投放表单 | 居中弹窗，毛玻璃卡片，文本域+标签选择+提交按钮 |
| 气味海洋 | 详情弹窗 | 毛玻璃半透明卡片，气味笔记全文+标签+时间+共鸣按钮 |
| 气味海洋 | 水花粒子 | 点击瓶子时从瓶口溅射半透明水珠粒子，扩散渐隐 |
| 气味海洋 | 共鸣飞瓶 | 用户瓶子飞向目标瓶的轨迹动画，融合后颜色渐变 |
| 我的漂流瓶 | 瓶子网格 | 2列网格，每个瓶子毛玻璃卡片样式，显示标签色+笔记摘要+共鸣数 |
| 全局 | 底部导航 | 固定底部半透明栏，两个tab：气味海洋/我的漂流瓶 |

### 4.3 响应式设计
- 桌面优先设计，移动端自适应
- 海洋Canvas自适应全屏尺寸
- 移动端瓶子交互改为触摸点击（无hover状态），详情卡底部弹出
- 我的漂流瓶网格在移动端改为单列
- 触摸优化：按钮最小44px点击区域

### 4.4 动画与性能
- 漂浮动画：每瓶独立正弦曲线运动，随机频率和振幅，60fps
- 呼吸光晕：CSS box-shadow脉动或Canvas radialGradient脉动
- 水花粒子：点击时生成20-30个粒子，物理抛射+重力+渐隐
- 共鸣飞行：贝塞尔曲线路径，1.5s缓入缓出
- 帧率目标：60fps，Canvas批量渲染，离屏瓶子不绘制
