## 1. 产品概述

「灵感网格」是一个匿名创意碰撞平台，让用户在霓虹都市风的网格画布上发布简短创意，通过「启发」机制实现创意之间的碰撞与融合，激发灵感火花。

- 目标用户：创意工作者、设计师、写作者、任何需要灵感碰撞的人
- 核心价值：通过匿名、可视化的创意碰撞机制，打破思维壁垒，让灵感在人群中自由流动

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 匿名用户 | 无需注册 | 发布创意、浏览网格、启发他人创意、查看热度榜 |

### 2.2 功能模块

1. **网格画布页**：创意卡片网格展示、粒子背景、卡片浮动动画、融合连线动画
2. **热度榜页**：排名卡片列表、排名徽章、热度指示器

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 网格画布页 | 粒子背景 | 缓慢流动的粒子系统，增强视觉深度层次感 |
| 网格画布页 | 创意网格 | 随机散布的创意卡片，荧光色毛玻璃质感，呼吸浮动动画，响应式布局（桌面4列/平板2列/手机1列） |
| 网格画布页 | 详情弹窗 | 点击卡片弹出毛玻璃详情弹窗，缓动滑入动画，显示完整创意内容和被启发次数，提供「启发」按钮 |
| 网格画布页 | 融合动画 | 点击启发后，彩色发光线条从当前创意飞向目标创意并融合，持续1.5秒，完成后卡片扩大1.3倍并更新启发次数 |
| 网格画布页 | 发布弹窗 | 毛玻璃输入框，最多50字创意输入，发布后卡片随机出现在网格中 |
| 网格画布页 | 底部导航栏 | 固定导航栏，包含「发布创意」和「热度榜」按钮 |
| 热度榜页 | 排名列表 | 前10名创意卡片，按被启发次数降序排列，带排名徽章 |
| 热度榜页 | 返回按钮 | 返回网格画布页 |

## 3. 核心流程

用户打开平台 → 看到霓虹都市风网格画布，创意卡片随机散布，缓慢浮动 → 点击底部「发布创意」→ 弹出毛玻璃输入框 → 输入创意（最多50字）→ 发布 → 新卡片以动画出现在网格中 → 点击任意卡片 → 弹出详情弹窗 → 点击「启发」按钮 → 彩色发光线条从用户创意飞向目标卡片 → 融合完成，目标卡片变大变亮，启发次数+1 → 点击底部「热度榜」→ 查看被启发次数最多的前10名创意

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深紫(#0D0221)到深蓝(#0A1628)渐变背景
- 辅助色：荧光粉(#FF2D95)、荧光蓝(#00D4FF)、荧光绿(#39FF14)半透明毛玻璃
- 卡片风格：半透明毛玻璃质感(frosted glass)，荧光色渐变边框，圆角16px
- 按钮风格：荧光色发光按钮，hover时光晕增强
- 字体：标题使用 Orbitron（科幻感），正文使用 Noto Sans SC
- 布局风格：网格散布式，固定底部导航栏
- 动画：卡片呼吸浮动、粒子流动、融合飞线、缓动滑入
- 图标：霓虹风格线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 网格画布页 | 粒子背景 | 深紫深蓝渐变背景，半透明粒子缓慢流动，CSS动画实现 |
| 网格画布页 | 创意网格 | 毛玻璃卡片，荧光色渐变边框，呼吸浮动动画(translateY ±6px, 3s)，随机位置散布 |
| 网格画布页 | 详情弹窗 | 毛玻璃遮罩+弹窗，缓动滑入(transform + opacity, 0.3s)，启发按钮荧光发光 |
| 网格画布页 | 融合动画 | 彩色发光SVG线条飞向目标，1.5秒持续，目标卡片scale(1.3)脉冲 |
| 网格画布页 | 发布弹窗 | 毛玻璃输入框，居中弹出，textarea限50字，发布按钮荧光发光 |
| 网格画布页 | 底部导航栏 | 固定底部，毛玻璃背景，两个荧光按钮，霓虹图标 |
| 热度榜页 | 排名卡片列表 | 纵向列表，卡片带排名徽章(金/银/铜荧光色)，热度数字荧光显示 |

### 4.3 响应式设计

- 桌面端(≥1024px)：网格每行4张卡片，粒子密度高
- 平板端(768px-1023px)：网格每行2张卡片，粒子密度中
- 手机端(<768px)：网格每行1张卡片，粒子密度低，触控优化

### 4.4 性能目标

- 所有交互动画帧率保持60fps
- 使用CSS动画优先，减少JavaScript计算
- 粒子系统使用Canvas渲染，避免DOM节点过多
- 融合动画使用requestAnimationFrame驱动
