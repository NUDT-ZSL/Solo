## 1. 产品概述

智能音乐播放器是一款基于 Web 技术的沉浸式音乐播放应用，融合实时音波可视化、3D 专辑封面展示与拖拽式播放列表管理，为用户提供桌面级音乐播放体验。目标用户为音乐爱好者与视觉体验追求者，核心价值在于将音频播放与视觉艺术深度融合。

## 2. 核心功能

### 2.1 功能模块

1. **播放器页面**：播放控制（播放/暂停、上一首/下一首、进度条、音量）、3D专辑封面旋转卡片、实时音波可视化、键盘快捷键
2. **播放列表面板**：歌曲列表展示、拖拽排序、当前播放高亮

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 播放器页面 | 播放控制 | 播放/暂停切换、上一首/下一首、进度拖拽、音量0-100%调节、当前时间/总时长显示 |
| 播放器页面 | 3D专辑封面 | 当前播放歌曲封面以CSS rotateY 3D旋转展示，悬停暂停旋转并放大1.1倍显示详情浮层 |
| 播放器页面 | 音波可视化 | Canvas绘制64条频率条形图，蓝→红渐变，60帧更新，弹性缓动动画 |
| 播放器页面 | 键盘快捷键 | 空格播放/暂停，左右箭头快进/快退5秒，上下箭头调节音量 |
| 播放列表面板 | 歌曲列表 | 展示6+首虚拟歌曲（标题、艺术家、封面占位图、时长），拖拽排序，当前项高亮+左边缘发光条 |
| 播放列表面板 | 淡入淡出 | 歌曲切换时音频淡入淡出效果 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题 #1a1a2e，辅色 #16213e 和 #0f3460
- **强调色**：霓虹蓝 #00d4ff 与紫色 #a855f7 渐变
- **按钮样式**：圆角8px，悬停时背景发光扩散动画（box-shadow过渡0.3s）
- **字体**：系统无衬线字体
- **布局风格**：桌面端左右并排、平板端上下结构、手机端底部迷你播放器
- **滑条样式**：渐变填充的进度条与音量条
- **列表高亮**：当前播放项左边缘发光条

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 播放器页面 | 播放控制区 | 深色背景、霓虹蓝紫渐变按钮、发光hover效果、渐变进度条、渐变音量条、时间显示 |
| 播放器页面 | 3D封面区 | CSS 3D旋转卡片、悬停放大1.1倍、歌曲详情浮层 |
| 播放器页面 | 音波可视化区 | Canvas 64条频率条、底部蓝→顶部红渐变、弹性动画 |
| 播放列表面板 | 歌曲列表 | 深色卡片、当前项高亮+左边缘发光条、拖拽手柄、封面缩略图 |

### 4.3 响应式设计

- **桌面端（≥1024px）**：播放器与列表左右并排布局
- **平板端（768-1023px）**：上下结构，播放器在上列表在下
- **手机端（<768px）**：底部固定迷你播放器，点击展开全屏模式

### 4.4 性能指标

- 音波可视化 FPS 保持 55-60
- 歌曲切换响应时间 < 200ms
- 拖拽排序无明显卡顿
