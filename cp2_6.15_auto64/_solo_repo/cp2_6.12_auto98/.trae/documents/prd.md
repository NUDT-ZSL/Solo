## 1. 产品概述

故事织机是一个多人异步故事接龙与AI续写辅助的在线创作平台，旨在解决写作爱好者灵感枯竭和缺乏互动反馈的痛点。
- 目标用户：写作爱好者、创意写作者、文学社群成员
- 核心价值：通过多人协作接龙激发创作灵感，借助AI续写建议突破写作瓶颈，可视化故事意象增强沉浸感

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览公开房间和故事 |
| 创作者 | 输入昵称即可加入 | 创建房间、加入房间、提交段落、使用AI续写、导出故事 |

### 2.2 功能模块
1. **房间大厅页**：房间列表、创建房间、加入房间
2. **故事接龙页**：故事时间线、段落编辑器、AI续写辅助、故事画板、房间统计、故事导出

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 房间大厅 | 房间列表 | 展示所有活跃房间，显示房间号、主题、参与人数、段落数 |
| 房间大厅 | 创建房间 | 输入房间主题，生成6位房间号，跳转至故事接龙页 |
| 房间大厅 | 加入房间 | 输入6位房间号加入已有房间 |
| 故事接龙 | 故事时间线 | 按顺序展示所有段落，新段落打字机动画+绿色光晕闪烁 |
| 故事接龙 | 段落编辑器 | 100-500字输入框，提交后加入时间线 |
| 故事接龙 | AI续写辅助 | 点击按钮获取3个续写建议，浮动标签展示，点击插入编辑器 |
| 故事接龙 | 故事画板 | Canvas 2D绘制意象图，基于最近3段关键词生成，段落更新时动画过渡 |
| 故事接龙 | 房间统计 | 总段落数、总字数、参与人数、环形图展示用户贡献比例 |
| 故事接龙 | 故事导出 | 导出为txt文件，包含作者和序号 |

## 3. 核心流程

用户进入应用 → 浏览房间列表或创建/加入房间 → 进入故事接龙页 → 阅读已有段落 → 撰写新段落或点击AI续写获取建议 → 提交段落 → 段落以动画效果出现在时间线 → 画板自动更新意象图 → 继续接龙或导出故事

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深色主题（主背景 #1a1a2e，卡片背景 #16213e，强调色 #e94560）
- 按钮风格：圆角8-12px，强调色按钮带hover动画
- 字体：标题使用 Noto Serif SC（衬线体，文学气质），正文使用 Noto Sans SC
- 布局风格：三栏布局（左侧导航+中央内容+右侧辅助），卡片式设计
- 图标风格：线条图标，lucide-react

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 房间大厅 | 房间列表 | 卡片网格布局，每张卡片含房间号、主题预览、人数标签，深色卡片+微弱白边框 |
| 房间大厅 | 创建房间按钮 | 强调色背景(#e94560)，圆角8px，hover时发光 |
| 故事接龙 | 故事时间线 | 垂直时间线，段落卡片左右交错排列，hover上移2px+加深阴影 |
| 故事接龙 | 段落编辑器 | 底部固定，textarea输入框，聚焦时#e94560光晕，字数计数器 |
| 故事接龙 | AI续写建议 | 编辑器上方浮动标签，浅蓝到浅紫渐变背景，圆角8px，hover放大1.1倍+下划线 |
| 故事接龙 | 故事画板 | Canvas 2D区域，段落更新时旧图缩小消失+新图放大出现动画0.6s |
| 故事接龙 | 环形统计图 | 按色相环分配颜色，hover放大1.05倍+显示标签 |
| 故事接龙 | 导出按钮 | 次要按钮样式，点击触发txt文件下载 |

### 4.3 响应式设计
- 桌面优先设计，三栏布局
- 视口宽度 < 768px：右侧辅助区折叠到底部，左侧导航栏变为顶部汉堡菜单
- 移动端（宽度 < 480px）：卡片内边距缩减至12px，字体缩小至14px，AI建议标签横向滚动

### 4.4 动效规范
- 新段落出现：打字机动画（字符间隔30ms）+ 段落完成时绿色光晕闪烁0.3秒
- 段落卡片hover：向上平移2px，阴影加深，过渡0.2s
- 输入框聚焦：#e94560外边框光晕（扩散半径4px）
- AI建议标签hover：放大1.1倍+下划线
- 画板过渡：旧图缩小至左上角消失，新图从右下角放大出现，持续0.6秒
- 环形图扇区hover：放大1.05倍+显示标签
