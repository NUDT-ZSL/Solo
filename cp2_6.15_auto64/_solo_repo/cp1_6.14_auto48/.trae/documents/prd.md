## 1. 产品概述

TribeTales 是一个面向小型社区（班级、读书会、兴趣小组）的在线互动式故事接龙平台，成员可以在故事树的不同分支上接力续写，每个节点包含一段文字和一张配图，读者可以沿不同路径探索故事的不同结局，提供比群聊文字投票更有沉浸感和结构性的协作体验。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 加入方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需加入 | 浏览故事森林和阅读故事内容 |
| 成员 | 邀请链接加入，设置昵称和颜色标识 | 浏览、续写故事、实时协作 |

### 2.2 功能模块

1. **首页 - 故事森林**：顶部导航栏（Logo+创建按钮）、瀑布流卡片布局展示故事列表、CSS几何抽象封面插画
2. **故事详情页 - 故事树**：SVG可交互故事树、节点内容面板、续写编辑器、右侧边栏（贡献者+时间线）
3. **加入流程**：邀请链接→设置昵称→随机分配颜色标识→进入故事

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 导航栏 | 半透明毛玻璃导航栏，左侧Logo（手写风格），右侧创建故事按钮（渐变紫，hover阴影扩散） |
| 首页 | 故事卡片瀑布流 | 每张卡片300px宽，顶部CSS几何抽象封面，中间标题+摘要，底部分支数+参与人数，hover漂浮效果 |
| 故事详情页 | 故事树SVG | 树根底部向上生长，贝塞尔曲线主干，分支从节点斜向分出，节点圆形可点击，hover扩大发光 |
| 故事详情页 | 节点内容面板 | 从节点弹出360px宽面板，显示文字段落和配图，右上角续写按钮 |
| 故事详情页 | 续写编辑器 | 全屏半透明遮罩，中央600x400编辑框，行高1.8，提交按钮绿色圆角，点击缩放动画 |
| 故事详情页 | 侧边栏 | 固定右侧260px深色面板，贡献者头像列表（带代际色条），时间线列表 |
| 加入流程 | 昵称设置 | 输入2-10字符昵称，随机分配20种预设色之一，显示在头像环形色圈 |

## 3. 核心流程

用户通过邀请链接加入故事 → 设置昵称获得临时身份 → 浏览故事森林首页 → 点击卡片进入故事详情 → 在故事树上点击节点查看内容 → 点击续写按钮在编辑器中撰写 → 提交后新节点淡入生成 → 其他用户通过WebSocket实时看到新节点和编辑状态

```mermaid
flowchart TD
    "A[用户访问首页]" --> "B[浏览故事森林]"
    "B" --> "C[点击故事卡片]"
    "C" --> "D[进入故事详情页]"
    "D" --> "E[点击树节点查看内容]"
    "E" --> "F[点击续写按钮]"
    "F" --> "G{是否已加入?}"
    "G" --> "|否| H[邀请链接加入]"
    "H" --> "I[设置昵称和颜色]"
    "I" --> "J[进入编辑器]"
    "G" --> "|是| J"
    "J" --> "K[撰写内容+上传配图]"
    "K" --> "L[提交续写]"
    "L" --> "M[新节点淡入生成]"
    "M" --> "N[WebSocket广播给其他用户]"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖象牙白 `#faf3e0` 背景，紫色系 `#6c5ce7` → `#a29bfe` 作为强调色
- **辅助色**：绿色 `#00b894`（提交按钮）、黄色系 `#fffde7`/`#ffeaa7`（内容面板）、深色 `#2d3436`（侧边栏/文字）
- **按钮风格**：大圆角（20-22px），渐变背景，hover时阴影扩散
- **字体**：Logo使用手写风格字体，正文16px行高1.8，标题加粗
- **布局**：卡片式布局，瀑布流排列，故事树SVG居中展示
- **动画**：卡片hover漂浮、节点hover发光、续写提交缩放、新节点淡入

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 导航栏 | 背景#ffffff99，backdrop-filter: blur(12px)，高56px，Logo手写字#2d3436，创建按钮140x40px圆角20px渐变紫 |
| 首页 | 故事卡片 | 宽300px，最小高280px，圆角16px，白底，1px solid #dfe6e9边框，hover上浮6px+阴影变化，过渡0.35s缓出 |
| 首页 | 封面插画 | CSS几何抽象画，200x120px，圆角8px，色块随机组合 |
| 故事详情页 | 故事树 | SVG绘制，贝塞尔曲线主干，线宽2px #636e72，节点圆径14px #6c5ce7，hover扩至18px+发光 |
| 故事详情页 | 内容面板 | 360px宽，背景#fffde7，圆角12px，2px solid #ffeaa7边框，配图340px宽圆角8px |
| 故事详情页 | 编辑器 | 全屏遮罩#00000088，编辑框600x400px，圆角16px，内边距24px，提交按钮120x44px圆角22px #00b894 |
| 故事详情页 | 侧边栏 | 260px宽，背景#2d3436，圆角12px，内边距16px，距顶部100px固定 |

### 4.3 响应式设计

- 桌面优先设计，适配1024px以上屏幕
- 瀑布流卡片在小屏幕下从多列变为单列
- 故事树SVG自适应容器宽度
- 侧边栏在小屏幕下折叠为底部抽屉

### 4.4 3D场景指南

本项目不涉及3D场景。
