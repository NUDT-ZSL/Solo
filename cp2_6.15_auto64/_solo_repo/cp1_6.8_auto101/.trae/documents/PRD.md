## 1. 产品概述

「烛龙之息」是一款2D Roguelite探索游戏，玩家操控幼年烛龙在随机生成的熔岩洞穴中飞行冒险。通过喷吐龙息点燃火把照亮黑暗，躲避暗影怪物与岩浆陷阱，收集远古龙鳞解锁天赋，逐层深入探索。

- 目标用户：喜欢Roguelite探索、暗黑风格和东方神话题材的玩家
- 核心价值：将中国烛龙神话与Roguelite机制融合，以"光与暗"为核心玩法张力，打造沉浸式的洞穴探索体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 说明 |
|------|------|
| 幼年烛龙 | 玩家唯一操控角色，具备飞行、龙息喷吐、天赋成长能力 |

### 2.2 功能模块

1. **游戏主画面**：洞穴探索、烛龙操控、龙息喷吐、火把点燃、怪物战斗
2. **HUD界面**：生命值、龙鳞计数、小地图、天赋菜单

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主画面 | 洞穴场景渲染 | Canvas绘制熔岩洞穴、岩浆流动动画、火把粒子光照 |
| 游戏主画面 | 烛龙操控 | WASD/触摸方向键控制飞行，空格/按钮喷吐扇形龙息 |
| 游戏主画面 | 龙息与火把 | 龙息呈扇形火焰，可点燃火把；火把照亮周围区域 |
| 游戏主画面 | 暗影怪物 | 蝙蝠型暗影怪从死角飞出，触碰扣血 |
| 游戏主画面 | 岩浆陷阱 | 岩浆池动态流动，接触灼伤龙身 |
| 游戏主画面 | 龙鳞收集 | 散落的远古龙鳞，用于解锁天赋 |
| 游戏主画面 | 层级生成 | 每层随机生成洞穴地图，火把、怪物、龙鳞位置随机 |
| HUD界面 | 生命值显示 | 左上角心形图标显示当前生命值 |
| HUD界面 | 龙鳞计数 | 左上角显示收集的龙鳞总数 |
| HUD界面 | 小地图 | 右上角显示已探索区域和火把位置 |
| HUD界面 | 天赋菜单 | 底部天赋按钮，点击弹出毛玻璃天赋树面板 |

## 3. 核心流程

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：暗红（#8B0000）→ 橙黄（#FF6B00）渐变，模拟熔岩洞穴氛围
- 强调色：金色（#FFD700）用于龙鳞和天赋高亮
- 按钮风格：圆角毛玻璃按钮，带暖光辉光效果
- 字体：标题使用衬线体（Noto Serif SC），正文使用无衬线体（Noto Sans SC）
- 布局：全屏Canvas游戏区域，HUD覆盖层半透明悬浮
- 图标风格：东方神话风格的手绘线稿图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主画面 | 洞穴场景 | 暗红渐变背景、岩浆流动贴图、火把暖光粒子、黑暗遮罩 |
| 游戏主画面 | 烛龙 | 半透明发光鳞片、龙息扇形火焰粒子、微光光晕 |
| 游戏主画面 | 暗影怪物 | 黑紫色蝙蝠轮廓、红色眼睛、暗影拖尾 |
| HUD界面 | 生命值 | 左上角心形图标×当前生命值，受伤时闪烁动画 |
| HUD界面 | 龙鳞计数 | 心形旁金色龙鳞图标+数字 |
| HUD界面 | 小地图 | 右上角半透明圆角矩形，已探索区域亮色，火把亮点 |
| HUD界面 | 天赋按钮 | 底部居中毛玻璃按钮"天赋"，点击弹出天赋树面板 |
| 天赋面板 | 天赋树 | 毛玻璃背景、树状天赋节点、龙鳞价格、已解锁高亮 |

### 4.3 响应式适配

- 桌面端：WASD键盘控制+空格喷吐，全屏Canvas
- 移动端：左下角虚拟方向键+右下角喷吐按钮，Canvas自适应屏幕尺寸，HUD元素缩放适配

### 4.4 视觉特效

- 岩浆：动态流动贴图 + 粒子飞溅效果
- 火把：暖光粒子系统，径向渐变光照范围
- 烛龙：半透明鳞片发光效果，身体微光光晕
- 龙息：扇形火焰粒子，橙红渐变，带烟雾拖尾
- 暗影怪物：黑紫轮廓+暗影拖尾+红色眼点
- 黑暗遮罩：未点燃区域全黑，烛龙周围微光渐变
- 所有元素使用缓动动画（easing），帧率保持60fps
