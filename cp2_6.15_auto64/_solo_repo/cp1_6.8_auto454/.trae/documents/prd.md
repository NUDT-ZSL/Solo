## 1. 产品概述

「流光幻境」是一款基于 WebGL 的 3D 交互式粒子可视化体验，用户在动态发光粒子海洋中通过鼠标操控视角与水面交互，点击水面激发彩色光波涟漪，营造持续变幻的赛博幻彩光之幻境。

- 核心目标：打造沉浸式的 3D 粒子交互体验，以极致视觉冲击力展现赛博幻彩美学
- 目标用户：视觉艺术爱好者、交互设计从业者、数字艺术展览观众

## 2. 核心功能

### 2.1 功能模块

1. **3D 粒子场景页**：粒子海洋、波浪动画、光网连接、光波涟漪、视角控制
2. **控制面板**：粒子数量调节、波浪幅度调节、颜色主题切换、重置视角

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 3D 粒子场景页 | 粒子海洋 | 1000-8000 个六边形发光粒子分布在水面高度附近，做波浪起伏运动，颜色从紫红渐变到青蓝再到亮金 |
| 3D 粒子场景页 | 光网连接 | 近距离粒子间以半透明细丝相连，形成动态光网结构 |
| 3D 粒子场景页 | 波浪动画 | 粒子随时间做正弦波浪起伏，幅度可调 |
| 3D 粒子场景页 | 光波涟漪 | 点击水面从点击点向外扩散圆形光晕，光晕经过的粒子暂时变亮并跳变为暖色，然后逐渐恢复 |
| 3D 粒子场景页 | 视角控制 | 鼠标拖拽旋转视角、滚轮缩放，粒子随视角改变流动方向 |
| 控制面板 | 粒子数量滑块 | 范围 1000-8000，实时调整粒子数量 |
| 控制面板 | 波浪幅度滑块 | 范围 0.1-3.0，实时调整波浪起伏幅度 |
| 控制面板 | 颜色主题选择器 | 4 种预设：幻彩、极光、熔岩、深海 |
| 控制面板 | 重置视角按钮 | 一键恢复默认相机位置和角度 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深黑 (#0a0a1a) → 暗蓝 (#0d1b3e) 渐变背景
- **粒子颜色**：紫红 (#ff2d7b) → 青蓝 (#00e5ff) → 亮金 (#ffd700) 渐变
- **按钮样式**：圆角胶囊型，半透明毛玻璃效果，悬停时发光边框
- **字体**：标题使用 Orbitron（科技感），UI 文字使用 Noto Sans SC
- **布局风格**：全屏沉浸式 3D 场景，左下角浮动毛玻璃控制面板
- **图标风格**：细线条线性图标，发光效果

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 3D 粒子场景页 | 粒子海洋 | 六边形发光粒子、深黑到暗蓝渐变背景、半透明光网丝线 |
| 3D 粒子场景页 | 光波涟漪 | 点击点向外扩散的圆形光晕环、粒子短暂变亮暖色闪烁 |
| 控制面板 | 毛玻璃面板 | 半透明背景 + backdrop-blur、圆角、微妙边框发光 |
| 控制面板 | 滑块控件 | 自定义样式滑块、渐变轨道、发光拇指 |
| 控制面板 | 主题选择器 | 色块按钮组、选中态发光高亮 |
| 控制面板 | 重置按钮 | 胶囊型按钮、悬停变色 + 微缩放动效 |

### 4.3 响应式适配

- 桌面端：全屏 3D 场景，控制面板左下角固定定位
- 移动端：触控拖拽旋转、双指缩放、点击涟漪，控制面板底部居中展示，可折叠收起
- 帧率目标：60fps，通过 InstancedMesh 和 GPU 着色器优化性能

### 4.4 3D 场景指引

- **环境氛围**：深黑到暗蓝渐变天空，无 HDRI，纯程序化背景
- **灯光设置**：环境光微弱照亮粒子，不使用方向光，粒子自发光通过 ShaderMaterial 实现
- **相机设置**：透视相机，初始位置俯视水面约 45° 角，FOV 60°，近裁剪面 0.1，远裁剪面 1000
- **构图与焦点**：粒子海洋铺满视野，水面为视觉焦点平面
- **交互与动画**：OrbitControls 旋转/缩放，Raycaster 检测点击水面位置，ShaderMaterial 实现粒子波浪和颜色变化
- **后处理效果**：UnrealBloomPass 泛光效果增强发光感
- **性能预算**：8000 粒子使用 InstancedMesh + 自定义着色器，光网使用 LineSegments 几何体批量渲染
