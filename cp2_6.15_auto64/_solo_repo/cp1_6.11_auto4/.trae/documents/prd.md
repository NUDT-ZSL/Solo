## 1. 产品概述

「音律星轨」是一款基于Canvas的交互式音乐可视化工具，将键盘/触摸输入的MIDI音符实时映射为在深邃星空背景上飞舞的发光轨迹。用户通过数字键1-7或点击屏幕触发音符，生成带有渐隐拖尾效果的星轨光点，星轨之间可产生发光连线与融合爆裂交互，营造出流星划过夜空的沉浸式体验。

- 目标用户：创意编程爱好者、音乐可视化爱好者、数字艺术创作者
- 核心价值：将抽象音符转化为具象的视觉运动，提供直觉式的音乐-视觉映射体验

## 2. 核心功能

### 2.1 功能模块

1. **星空背景**：全屏径向渐变背景 + 60-100颗缓慢闪烁的静态星光点
2. **星轨生成**：键盘1-7或点击触发，从触发位置/随机区域生成星轨，初始速度120-180px/s，拖出8-12个渐隐轨迹块
3. **颜色映射**：力度从青色(#00D4FF)到橙红(#FF6B35)渐变，轨迹块随时间变暗至灰色
4. **星轨交互**：距离<30px产生发光连线，距离<10px融合成更大星轨+金色粒子爆裂
5. **生命周期**：默认8-12秒存活，衰减后亮度渐降直至消失
6. **控制面板**：密度滑块(30-80)、时长滑块(5-20s)、重置按钮

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主画布 | 星空背景 | 径向渐变(#0A0E1A→#0A1628) + 闪烁星光(亮度0.3-1.0正弦波动，周期2-4s) |
| 主画布 | 星轨渲染 | 发光圆点+渐隐轨迹块，颜色映射力度，生命周期管理 |
| 主画布 | 星轨交互 | 距离检测→发光连线/融合爆裂 |
| 控制面板 | 密度控制 | 滑块30-80，数值显示 |
| 控制面板 | 时长控制 | 滑块5-20秒，数值显示 |
| 控制面板 | 重置按钮 | 清空所有星轨，暗红#8B2252，悬停#C44569，点击缩放动画 |

## 3. 核心流程

用户打开应用→看到深邃星空背景→通过键盘数字键1-7或点击屏幕触发音符→系统从触发位置生成星轨→星轨以随机方向飞出并拖出渐隐轨迹→多颗星轨相遇时产生连线或融合→星轨生命周期结束后衰减消失→用户可通过控制面板调整参数或重置

```mermaid
    "距离检测" --> "|距离<30px| 发光连线"
    "距离检测" --> "|距离<10px| 融合+粒子爆裂"
    "距离检测" --> "|无交互| 继续飞行"
    "星轨飞出+拖尾" --> "生命周期衰减"

```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深空黑(#0A0E1A)到极深蓝(#0A1628)径向渐变
- 星轨色调：青色(#00D4FF)至橙红(#FF6B35)力度渐变
- 控制面板：毛玻璃质感(rgba(10,14,26,0.7))，圆角12px
- 字体：无衬线字体，加权500，柔白(#E0E5EC)
- 按钮：暗红#8B2252，悬停#C44569
- 布局：全屏Canvas + 左下角控制面板

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主画布 | 星空背景 | 径向渐变底色，60-100颗闪烁星光(1-3px白色圆点) |
| 主画布 | 星轨 | 发光圆点(半径4px)+8-12个渐隐轨迹块(4→1px, 透明度1→0) |
| 主画布 | 连线 | 2px宽发光线，混合色，透明度0.6 |
| 主画布 | 融合爆裂 | 8-12个金色粒子(3-5px)，2秒衰减 |
| 控制面板 | 密度滑块 | range input + 数值显示 |
| 控制面板 | 时长滑块 | range input + 数值显示 |
| 控制面板 | 重置按钮 | 暗红按钮，悬停变亮，点击缩放 |

### 4.3 响应式设计

- 桌面端：全屏Canvas，左下角控制面板(宽度240px)
- 手机横屏(宽>高)：控制面板移至右下角
- 平板竖屏：控制面板保持左下角，滑块间距增大
- 移动端：控制面板缩小至宽度180px，滑块变圆形旋钮

### 4.4 性能目标

- 50颗星轨+连线：稳定60FPS
- 80颗星轨满载：不低于45FPS
- Canvas每帧渲染：不超过14ms
