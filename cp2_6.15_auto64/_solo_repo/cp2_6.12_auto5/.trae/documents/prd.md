## 1. 产品概述

CodeSnap 是一个在线代码片段分享与运行平台，用户可以创建、编辑、运行并分享 JavaScript/Python 代码片段。通过 Docker 沙箱提供安全的代码执行环境，并生成唯一短链接实现代码分享。
- 目标用户：开发者、编程学习者、技术面试者
- 核心价值：零配置的代码编写-运行-分享一体化体验，安全隔离的执行环境

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 创建、运行、分享代码片段 |
| 查看者 | 通过短链接访问 | 只读查看代码和输出 |

### 2.2 功能模块

1. **编辑器页面**：代码编辑、语言切换、运行按钮、标题/描述编辑、主题切换
2. **分享查看页面**：只读代码展示、运行结果展示、Markdown 描述渲染

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 编辑器页面 | 顶部导航栏 | 平台名称、新建片段按钮、暗色/亮色主题切换开关 |
| 编辑器页面 | 代码编辑区（65%宽度） | Monaco Editor 代码编辑器，支持 JS/Python 语法高亮、行号、自动补全、暗色/亮色主题 |
| 编辑器页面 | 可拖动分隔条 | 左右区域宽度调节，拖动有平滑动画 |
| 编辑器页面 | 输出面板（35%宽度） | 运行结果/错误信息展示，正常输出绿色、错误红色、超时橙色 |
| 编辑器页面 | 代码元信息区 | 标题输入、Markdown 描述编辑 |
| 编辑器页面 | 底部页脚 | 版权信息 |
| 分享查看页面 | 只读代码展示 | 语法高亮的只读代码视图 |
| 分享查看页面 | 运行结果展示 | 历史运行结果或重新运行 |
| 分享查看页面 | Markdown 描述 | 渲染代码片段的 Markdown 描述 |

## 3. 核心流程

用户打开平台 → 在编辑器中编写代码 → 选择语言（JS/Python）→ 点击运行 → 代码发送到后端 → Docker 沙箱执行 → 返回结果 → 输出面板显示 → 填写标题和描述 → 保存 → 生成短链接 → 分享给他人 → 他人通过链接只读查看

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：暗色主题以 #0D1117 为基底，亮色主题以 #F6F8FA 为基底
- 强调色：运行按钮使用翠绿 #3FB950，分享按钮使用蓝色 #58A6FF
- 按钮风格：圆角 8px，hover 时缩放 1.05，过渡 0.2s ease
- 字体：代码区 JetBrains Mono，UI 区 system-ui
- 布局：卡片式设计，细微投影 box-shadow，圆角 8px
- 图标风格：使用 lucide-react 线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 编辑器页面 | 顶部导航栏 | 深灰#1E1E1E/白色#FFFFFF背景，平台Logo文字，新建按钮，主题切换Toggle，平滑过渡动画 |
| 编辑器页面 | 代码编辑区 | #1E1E1E/#FFFFFF背景，Monaco Editor，JetBrains Mono字体，语言选择下拉 |
| 编辑器页面 | 分隔条 | 4px宽，hover高亮，拖动时平滑宽度变化动画 |
| 编辑器页面 | 输出面板 | #161B22/#F6F8FA背景，终端风格输出，绿色/红色/橙色文字 |
| 编辑器页面 | 元信息区 | 标题输入框，Markdown描述文本域，保存/分享按钮 |
| 编辑器页面 | 底部页脚 | 简洁版权文字，居中排列 |

### 4.3 响应式设计

- 桌面优先设计，768px 为断点
- 移动端（<768px）：编辑区和输出区改为上下布局，编辑器自动调整大小
- 触摸优化：按钮最小触控区域 44px，分隔条在移动端隐藏

### 4.4 动画效果

- 主题切换：背景色、文字色平滑过渡（transition: 0.3s ease）
- 按钮交互：hover 缩放 1.05，0.2s ease
- 分隔条拖动：宽度变化平滑动画
- 页面加载：组件渐入动画
