## 1. 产品概述

数据叙事工作台是一款在线交互式数据可视化与故事讲述工具，让用户能够从结构化JSON数据中自动生成多种图表，并通过文字标注和动画过渡来讲述数据驱动的故事。

- 核心目标：帮助用户将枯燥的数据转化为生动的叙事故事，降低数据可视化门槛
- 目标用户：数据分析师、内容创作者、教育工作者、产品经理
- 产品价值：提供直观的拖拽式操作，支持多种图表类型切换，配合动画播放功能，让数据故事更具感染力

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，直接使用 | 上传数据、创建图表、添加标注、播放故事、导出HTML |

### 2.2 功能模块
1. **数据上传模块**：拖拽/点击上传JSON文件，数据验证与解析
2. **图表渲染模块**：时间线、柱状图、折线图三种图表，Canvas 2D渲染
3. **标注编辑模块**：文字标注卡片，支持字号、颜色、对齐方式调整
4. **故事控制面板**：拖拽排序图表和标注，调整故事顺序
5. **故事播放模块**：按顺序播放图表和标注，支持暂停功能
6. **导出模块**：一键导出为HTML静态文件

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 工作台主页 | 数据上传区 | 拖拽或点击上传JSON，拖入高亮，成功提示 |
| 工作台主页 | 故事板区域 | 展示图表卡片和标注卡片，支持类型切换 |
| 工作台主页 | 标注工具栏 | 添加标注，调整字号、颜色、对齐方式 |
| 工作台主页 | 故事控制面板 | 右侧面板，拖拽排序，播放控制 |
| 工作台主页 | 导出功能 | 右上角导出按钮，进度条，自动下载 |

## 3. 核心流程

用户上传JSON数据文件 → 系统解析验证数据 → 自动生成默认图表（时间线+柱状图）→ 用户切换图表类型/添加标注 → 通过控制面板调整顺序 → 点击播放故事 → 导出HTML文件

```mermaid

    A["上传JSON数据"] --> B["解析验证数据"]
    B --> C["生成默认图表"]
    C --> D["编辑图表与标注"]
    D --> E["调整故事顺序"]
    E --> F["播放预览"]
    F --> G["导出HTML文件"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：蓝色 #3B82F6
- 辅助色：紫色 #8B5CF6、绿色 #10B981
- 背景色：浅灰 #F9FAFB（故事板区域）、白色（卡片、面板）
- 按钮风格：圆角设计（4px/8px/12px层级），悬停放大1.05倍，阴影加深
- 字体：系统默认无衬线字体，标题粗体
- 布局风格：卡片式布局，左侧3/4故事板，右侧1/4控制面板
- 动效：统一淡入淡出动画（0.3s ease-in-out）

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 工作台主页 | 数据上传区 | 虚线边框，拖入高亮（浅蓝半透明），0.3s过渡 |
| 工作台主页 | 图表卡片 | 白底、圆角12px、阴影0 2px 8px rgba(0,0,0,0.08) |
| 工作台主页 | 柱状图 | HSL循环配色，柱体从下到上生长动画，依次延迟0.1s |
| 工作台主页 | 折线图 | 深蓝线条，白色圆点标记，从左到右逐步描绘0.5s |
| 工作台主页 | 时间线 | 水平布局，圆形节点（半径6px），节点依次铺开0.3s |
| 工作台主页 | 标注卡片 | 白底气泡，圆角12px，顶部小三角指向图表 |
| 工作台主页 | 控制面板 | 固定宽280px，白底，圆角8px，左侧2px蓝色竖条 |
| 工作台主页 | 播放按钮 | 渐变背景#8B5CF6到#6366F1，压暗效果 |
| 工作台主页 | 导出按钮 | 绿色#10B981，下载箭头图标 |

### 4.3 响应式
- 桌面端（≥768px）：左右布局，左侧3/4故事板，右侧280px控制面板
- 移动端（<768px）：控制面板折叠到底部固定底栏，横向滚动；故事板撑满宽度；导出按钮缩为图标按钮
- 触控优化：按钮最小点击区域44px，拖拽手势流畅

### 4.4 性能要求
- 同时显示10张图表和20个标注时，帧率≥50fps
- 单图表渲染时间≤200ms
- 拖拽排序响应延迟≤16ms
