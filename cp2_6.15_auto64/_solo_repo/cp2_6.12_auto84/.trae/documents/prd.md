## 1. 产品概述

邻里工具箱是一个面向城市社区的工具共享与预约管理应用，解决社区工具共享中工具被重复预约和忘记归还的问题。目标用户为社区中自发组织的工具共享小组成员，核心价值在于让工具借用流程可视化、可追溯、有信用保障。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 社区成员 | 用户名注册 | 发布工具、借用预约、确认归还、信用评价 |
| 访客 | 无需注册 | 浏览工具列表、查看工具详情 |

### 2.2 功能模块

1. **工具市场页**：工具搜索筛选、工具卡片网格展示、工具发布入口
2. **预约管理页**：预约日历视图、借用预约弹窗、归还确认与评价
3. **个人信用面板**：信用分圆环图、近期评价列表、借出/借用历史时间线
4. **导航栏**：全局搜索栏、类别筛选、用户头像与信用分徽标

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 工具市场页 | 搜索筛选栏 | 顶部搜索栏支持按工具名称模糊搜索，下方过滤按钮组按类别筛选 |
| 工具市场页 | 工具卡片网格 | 响应式网格布局展示工具卡片，卡片含图片、名称、状态标签，点击展开详情 |
| 工具市场页 | 工具发布表单 | 发布新工具，填写名称、类别、图片URL、描述、可用时段 |
| 预约管理页 | 预约日历 | 显示工具空闲时段，选择起止日期（不超过7天），填写借用用途 |
| 预约管理页 | 预约弹窗 | 毛玻璃背景，日历选择日期范围和备注输入，提交后按钮变为"已预约" |
| 预约管理页 | 归还确认与评价 | 确认归还按钮，弹出评价卡片（1-5星评分+简短评价） |
| 个人信用面板 | 信用分圆环图 | 0-100分，颜色从红色渐变到绿色，动画填充1秒 |
| 个人信用面板 | 近期评价列表 | 卡片展示，左侧纵向条纹按评分着色 |
| 个人信用面板 | 借出/借用历史时间线 | 横向时间轴，事件节点圆点颜色区分状态 |

## 3. 核心流程

### 3.1 工具借用流程
用户浏览工具市场 → 点击可用工具卡片 → 弹出预约弹窗 → 选择日期范围和填写用途 → 提交预约 → 工具状态变为"已借出" → 到期后借出方点击"确认归还" → 弹出评价卡片 → 打分并提交评价 → 信用分更新

### 3.2 工具发布流程
用户点击发布工具 → 填写工具信息（名称、类别、图片、描述、可用时段）→ 提交 → 工具卡片出现在市场页

## 4. 用户界面设计

### 4.1 设计风格

- **主色**：浅橙 #F5A623，辅色：深蓝 #2C5F8A
- **背景色**：米白 #FDF6EC
- **按钮风格**：圆角8-16px，渐变背景，悬停亮度提高10%，点击缩小至0.95倍
- **字体**：系统默认无衬线字体
- **布局风格**：卡片式网格布局，顶部固定导航栏
- **图标风格**：简洁线条图标（lucide-react）
- **动画风格**：柔和的暖色调过渡动画，微光效果状态标签

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 工具市场页 | 搜索筛选栏 | 圆角20px过滤按钮，选中时浅灰→主题蓝+白色文字，0.2秒切换动画 |
| 工具市场页 | 工具卡片网格 | 圆角12px卡片，桌面4列/平板2列/手机1列，间距24px，悬停上移4px+阴影，0.2秒过渡 |
| 工具市场页 | 状态标签 | 可用=绿色渐变微光，已借出=橘色渐变闪烁，维修中=灰色静态 |
| 预约管理页 | 预约弹窗 | 圆角16px，毛玻璃背景，中央放大展开0.3秒，日历高亮空闲时段 |
| 预约管理页 | 评价卡片 | 从右侧滑入0.3秒，1-5星点击放大抖动0.15秒 |
| 个人信用面板 | 信用分圆环图 | 0-100分，红色→绿色渐变，动画填充1秒 |
| 个人信用面板 | 评价列表 | 卡片左侧纵向条纹按评分着色 |
| 个人信用面板 | 时间线 | 横向时间轴，绿色=已归、橘色=进行中、灰色=已逾期 |

### 4.3 响应式设计

- 桌面端优先，4列工具卡片网格
- 平板端：2列网格，导航栏自适应
- 手机端：1列网格，导航栏折叠
- 触摸优化：所有可点击元素点击时缩小至0.95倍再弹回0.15秒

### 4.4 全局UI规范

- 导航栏：高度56px，半透明毛玻璃效果，滚动时下层内容模糊可见，右侧用户头像+信用分徽标
- 骨架屏：渐变色占位动画，从左到右闪烁1.5秒无限循环
- 空状态：暖色调圆角矩形背景插画+鼓励文字
- 链接交互：下划线从中间向两端展开
