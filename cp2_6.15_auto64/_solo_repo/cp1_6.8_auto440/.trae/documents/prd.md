## 1. 产品概述

「声景漫游」是一个环境声音标记与城市音景可视化平台。用户可在交互式地图上标记自己录制的环境声音片段（雨声、鸟鸣、车流等），配以文字描述和心情标签，系统将不同地点的声音混合生成动态"城市音景图"，以音频波形可视化展示各区域声音特征。
- 面向城市声音爱好者、声音设计师、冥想用户群体
- 价值：构建一个可探索、可混合、可分享的城市声景地图社区

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览声音卡片、收听音景、查看详情 |
| 用户 | 邮箱注册 | 标记声音、上传音频、管理自己的声音标记 |

### 2.2 功能模块
1. **首页（音景探索）**：搜索栏、筛选器、声音卡片瀑布流、悬浮"城市音景图"入口
2. **声音详情页**：完整波形、地点信息、文字描述、播放控制、波形缩放平移
3. **添加声音页**：地图选点、上传音频文件、填写描述和心情标签
4. **城市音景图页**：交互式地图上展示所有声音标记点，动态音频波形可视化

## 3. 核心流程

**浏览声音流程**：用户进入首页 → 浏览瀑布流声音卡片 → 可使用筛选器按心情/类型筛选 → 点击卡片进入详情页 → 查看完整波形并播放 → 返回继续浏览

**添加声音流程**：用户点击"添加声音" → 在地图上选择标记点 → 上传音频文件 → 填写描述、选择心情标签和声音类型 → 提交保存 → 返回首页看到新卡片

**音景探索流程**：用户点击悬浮"城市音景图"按钮 → 进入地图视图 → 查看各标记点 → 点击标记点预览声音 → 混合聆听区域音景

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：低饱和度蓝 (#6B8CAE) 与灰 (#8A9BAE)
- **背景**：浅灰到白渐变 (#F0F2F5 → #FFFFFF)
- **卡片风格**：圆角毛玻璃面板 (backdrop-filter: blur)，微发光边框 (box-shadow glow)
- **按钮风格**：圆角胶囊形，悬停时轻微上浮 + 光晕扩散动画
- **字体**：标题使用 Noto Serif SC（衬线体，优雅感），正文使用 Noto Sans SC
- **图标**：线性图标风格，配合 emoji 表达心情标签
- **布局**：卡片式布局，瀑布流网格

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 搜索与筛选栏 | 固定顶部，毛玻璃背景，搜索图标+输入框，筛选下拉标签 |
| 首页 | 声音卡片网格 | 瀑布流3列(桌面)/2列(平板)/1列(手机)，卡片圆角16px，毛玻璃面板 |
| 首页 | 悬浮入口按钮 | 底部居中浮动，蓝色渐变胶囊按钮，脉冲动画 |
| 首页 | 添加声音按钮 | 右上角，+图标+文字，悬停上浮+光晕 |
| 声音详情页 | 完整波形 | 大尺寸Canvas波形，播放时流动动画+光晕脉冲，支持缩放平移 |
| 声音详情页 | 信息面板 | 左侧/上方，毛玻璃面板，地点+描述+标签 |
| 添加声音页 | 地图选点 | 全宽地图，点击放置标记，标记带脉冲动画 |
| 添加声音页 | 上传区域 | 虚线边框拖拽区域，上传图标，支持mp3/wav |
| 城市音景图页 | 音景地图 | 全屏地图，标记点带声音类型图标，点击弹出迷你播放器 |

### 4.3 响应式适配
- 桌面端(≥1024px)：3列瀑布流，详情页左右分栏
- 平板端(768-1023px)：2列瀑布流，详情页上下布局
- 手机端(<768px)：1列瀑布流，详情页全宽堆叠，触控优化

### 4.4 动画与交互
- 卡片筛选时：淡入淡出 + 错落过渡 (staggered transition)
- 播放时波形：流动动画 + 节奏性光晕脉冲
- 按钮悬停：轻微上浮 (translateY -2px) + 光晕扩散
- 页面切换：平滑过渡动画
- 所有动画目标60fps，使用 transform/opacity 驱动
