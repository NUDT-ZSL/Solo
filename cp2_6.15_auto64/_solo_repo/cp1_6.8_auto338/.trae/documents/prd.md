# 言灵诗歌 - 产品需求文档（PRD）

## 1. 产品概述
「言灵诗歌」是一个匿名诗歌共创平台，用户提交一行短诗后，系统基于语义相似度自动匹配其他用户的诗行，拼接成一首由3-5位匿名作者共同贡献的完整诗歌，以沉浸式深色界面和优雅动画呈现。
- 解决孤立的创作无法形成完整诗歌的痛点，让陌生人通过语义共鸣实现共创
- 面向文学爱好者、诗歌创作者和寻求精神共鸣的互联网用户

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 匿名用户 | 无需注册，系统自动分配匿名ID | 提交诗行、浏览诗歌、查看匿名主页、搜索 |

### 2.2 功能模块
1. **首页**：沉浸式诗歌生成器、瀑布流诗歌展示、星光粒子背景
2. **匿名用户主页**：用户诗行历史、被拼接统计
3. **今日热诗榜**：过去24小时被拼接次数最多的前10首诗

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 诗歌输入区 | 输入框提交一行短诗，系统自动匹配2-4行近7天内其他用户提交的诗行，拼接成完整诗歌 |
| 首页 | 瀑布流展示区 | 展示生成的诗歌，每行有入场动画（模糊→清晰、下→上），显示匿名ID+时间戳来源 |
| 首页 | 星光粒子背景 | Canvas绘制的深色渐变背景+微弱星光粒子效果 |
| 首页 | 全局搜索 | 按关键词搜索诗行或匿名ID，卡片渐入动画展示结果 |
| 匿名用户主页 | 诗行历史 | 展示该用户过去30天内贡献的所有诗行 |
| 匿名用户主页 | 被拼接统计 | 展示该用户诗行被拼接次数的统计信息 |
| 今日热诗榜 | 热诗列表 | 过去24小时内被拼接次数最多的前10首诗，点击查看完整诗和每条来源 |

## 3. 核心流程

用户打开首页 → 输入一行短诗并提交 → 系统基于TF-IDF余弦相似度匹配近7天内2-4行诗 → 拼接成完整诗歌（3-5行）→ 瀑布流展示拼接结果（含匿名ID+时间戳）→ 用户可点击任意诗行跳转至该行来源匿名用户主页 → 用户可查看今日热诗榜 → 用户可使用全局搜索功能

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深灰到墨黑渐变背景（#0a0a0f → #1a1a2e）
- **辅助色**：浅金到白色渐变诗行文字（#d4a853 → #ffffff）
- **按钮风格**：圆角毛玻璃效果（backdrop-blur + 半透明白色边框），悬停时光晕扩散动画
- **字体**：衬线体（Noto Serif SC / Source Han Serif），正文16-18px，标题24-32px
- **布局风格**：瀑布流居中布局，最大宽度960px
- **图标风格**：极简线性图标（lucide-react）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 星光粒子背景 | Canvas全屏，深色渐变+星光粒子，持续微动 |
| 首页 | 诗歌输入区 | 居中输入框，毛玻璃卡片容器，提交按钮带光晕动画 |
| 首页 | 瀑布流展示区 | 诗歌卡片纵向排列，每行诗独立动画入场，显示匿名ID标签 |
| 首页 | 搜索区 | 顶部搜索栏，搜索结果以卡片渐入展示 |
| 匿名用户主页 | 诗行列表 | 卡片列表，每张卡片显示诗行内容+被拼接次数 |
| 匿名用户主页 | 统计面板 | 数字+图表展示总贡献行数、总被拼接次数 |
| 今日热诗榜 | 热诗排行 | 编号列表，1-3名有金色标识，点击展开完整诗 |

### 4.3 响应式适配
- 桌面优先设计，移动端自适应
- 桌面端：960px最大宽度居中，搜索栏固定顶部
- 移动端：全宽布局，输入框和卡片适配窄屏，搜索栏折叠为图标
- 触控优化：按钮最小44px触摸区域

### 4.4 动画性能
- 所有动画使用CSS transform + opacity，确保GPU加速
- Canvas粒子效果使用requestAnimationFrame，控制粒子数量≤150
- 页面切换和动画帧率保持60fps
- 入场动画使用CSS animation + animation-delay实现错落效果
