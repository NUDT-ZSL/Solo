## 1. 产品概述

在线旅行日志与足迹地图展示应用，让用户记录旅行经历、添加景点照片和文字描述，并在交互式地图上标记足迹位置。目标用户为旅行爱好者和背包客，帮助他们以可视化的方式回顾和分享旅行记忆。

- 解决问题：旅行记录分散、缺乏地理可视化、回忆难以系统整理
- 核心价值：将文字、图片与地图位置结合，提供沉浸式旅行回忆体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 浏览、创建、搜索、删除旅行日志，查看足迹地图 |

### 2.2 功能模块

1. **首页**：日志卡片网格列表、搜索筛选栏、添加日志按钮及模态表单
2. **地图页**：全屏足迹地图视图，Canvas绘制世界地图轮廓，红色标记点显示足迹
3. **详情页**：单条日志完整内容，图片轮播、文字描述、小地图标注、删除功能

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 搜索筛选栏 | 关键字搜索（匹配标题、地点、描述）+ 日期范围筛选，实时过滤，下拉显示匹配数量 |
| 首页 | 日志卡片网格 | 每行3/2/1列响应式布局，卡片含缩略图、标题、日期、地点、截断描述、查看详情按钮 |
| 首页 | 添加日志模态框 | 底部滑入动画表单，含标题、日期、地点、图片URL（最多3张）、描述（最多500字），前端验证 |
| 地图页 | 足迹地图 | Canvas绘制世界地图轮廓，红色标记点，悬停放大+tooltip，支持拖拽缩放 |
| 详情页 | 图片轮播 | 最多3张图片幻灯片，左右箭头切换，悬停透明度变化，无图显示灰色占位 |
| 详情页 | 完整内容 | 文字描述、日期、地点信息、小地图标注 |
| 详情页 | 删除功能 | 确认对话框（中心缩放出现），确认后0.3秒缩小消失动画并移除 |

## 3. 核心流程

用户打开应用 → 首页展示所有日志卡片（按日期倒序）→ 点击"添加日志"弹出模态表单 → 填写信息并提交 → 新日志出现在列表中 → 点击"查看详情"进入详情页 → 浏览完整内容或删除 → 切换到"地图"选项卡查看足迹分布 → 搜索筛选定位特定日志

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：#2C3E50 深蓝灰色（导航栏、边框聚焦色）
- **辅助色**：#E8D5B7 米色（卡片背景、文字区域）、#F5F5F0 米白色（卡片底色）
- **强调色**：#E74C3C 红色（按钮、标记点）、#D4A574 深米色（按钮悬停）
- **按钮风格**：圆角按钮，点击时0.1秒缩放动画(scale 0.95→1)
- **字体**：中文使用系统字体栈，数字和英文使用等宽字体；标题16-20px，正文14px，小字12px
- **布局风格**：顶部固定导航栏 + CSS Grid卡片网格布局
- **边框**：0.5px #D5C8B0 微弱边框，输入框聚焦从#D5C8B0变为#2C3E50

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 导航栏 | 高度60px，背景#2C3E50，白色文字，左侧应用名称，右侧首页/地图选项卡，下划线滑动0.25s |
| 首页 | 搜索筛选栏 | 搜索输入框+下拉匹配数量+日期范围筛选器 |
| 首页 | 日志卡片 | 320px宽，#F5F5F0背景，左上角圆形缩略图，标题/日期/地点/截断描述，查看详情按钮悬停#E8D5B7→#D4A574 |
| 首页 | 添加日志模态框 | 半透明黑色遮罩，底部滑入0.3s ease-out，红色边框验证失败+0.3s晃动 |
| 地图页 | 全屏地图 | Canvas世界地图轮廓，#E74C3C红色圆点标记，悬停1.5倍放大+tooltip(#2C3E50半透明背景，白色圆角文字) |
| 详情页 | 图片轮播 | 幻灯片切换，左右箭头悬停0.5→1透明度0.2s过渡，灰色#CCC占位 |
| 详情页 | 删除按钮 | 点击弹出确认对话框(中心缩放0.2s ease-out)，确认后0.3s缩小消失 |

### 4.3 响应式设计

- **桌面优先**设计，适配三个断点
- **1280px以上**：卡片三列布局
- **768px-1280px**：卡片两列布局
- **768px以下**：卡片单列布局，地图自动切换全屏并隐藏工具栏
- 触摸优化：移动端按钮足够大(最小44px)，滑动手势支持

### 4.4 动画与交互

- 导航选项卡下划线滑动：0.25s ease-in-out
- 模态表单底部滑入：0.3s ease-out
- 卡片悬停阴影变化：0-2px-4px → 0-6px-12px，0.3s过渡
- 按钮点击缩放：0.1s scale 0.95→1
- 输入框聚焦边框：#D5C8B0→#2C3E50，0.2s过渡
- 验证失败晃动：0.3s
- 删除缩小消失：0.3s
- 确认对话框缩放出现：0.2s ease-out
