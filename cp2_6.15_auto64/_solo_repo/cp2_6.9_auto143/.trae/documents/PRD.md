## 1. 产品概述

在线协同歌词MV字幕生成器，一款支持多人实时协作的音乐歌词编辑与可视化工具。用户可以导入MP3音乐文件，通过波形图和时间轴精确同步歌词，最终生成带滚动歌词的MV播放器页面。

- 主要目的：提供高效的多人协作歌词编辑体验，降低歌词制作门槛
- 解决问题：传统歌词编辑工具缺乏实时协作能力，同步精度低，操作繁琐
- 目标用户：音乐爱好者、独立音乐人、视频创作者、翻译社区
- 产品价值：通过WebSocket实时同步技术，实现多人并行编辑，大幅提升歌词制作效率

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 协作用户 | 自动分配匿名ID | 导入音频、编辑歌词、实时同步、预览播放 |

### 2.2 功能模块
1. **音频处理模块**：MP3文件导入、波形图可视化、音频播放控制
2. **时间轴编辑模块**：时间标记添加、歌词时间段调整、精确同步
3. **歌词编辑模块**：歌词文本输入、中英文支持、编辑状态锁定
4. **实时协作模块**：WebSocket同步、冲突处理、在线用户列表
5. **歌词预览模块**：滚动显示、高亮动画、MV播放器效果

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主编辑页 | 波形图区域 | Canvas绘制渐变波形（#1DB954→#191414），高度自适应，支持点击定位 |
| 主编辑页 | 时间轴区域 | 秒级刻度（主刻度5s/次刻度1s），红色时间标记，拖拽调整范围 |
| 主编辑页 | 歌词编辑面板（左） | 多段落歌词输入框，Enter提交，锁定状态显示，未保存黄色边框 |
| 主编辑页 | 歌词预览区域（中） | 居中高亮（白色→#1DB954渐变，0.3s淡入），前后行半透明，平滑滚动 |
| 主编辑页 | 在线用户列表（右） | Gravatar头像，编辑状态显示，滑入滑出动画 |
| 主编辑页 | 播放控制栏（底） | 播放/暂停、进度滑块、音量控制、时间显示、循环模式 |
| 主编辑页 | 通知提示（右下） | 冲突覆盖通知（黄色背景，2s自动消失，非阻塞） |

## 3. 核心流程

用户导入MP3音频文件后，系统自动解析并绘制波形图。用户在时间轴上点击添加歌词标记，每个标记间生成歌词编辑框。用户输入歌词文本并按Enter提交，歌词与对应时间段绑定。多人同时编辑时，通过WebSocket实时同步所有操作（新增/修改/删除），同步延迟控制在100ms以内。播放时歌词按时间平滑滚动并高亮显示。

    D --> E["Enter提交歌词"]
    E --> F["WebSocket广播同步"]

    H["多人实时协作"] --> F

## 4. 用户界面设计

### 4.1 设计风格
- 主背景色：#121212（深色）
- 卡片背景色：#282828
- 主色调：#1DB954（Spotify绿）
- 字体颜色：#FFFFFF（主要）/ #B3B3B3（次要）
- 分割线：1px #333333
- 按钮风格：圆角，hover时主色高亮，active时微缩放
- 字体：系统无衬线字体，字号层次分明
- 布局风格：CSS Grid多栏布局，卡片式设计
- 图标风格：简洁线性图标

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|----------|
| 主编辑页 | 整体布局 | CSS Grid三栏布局，底栏固定，深色主题 |
| 主编辑页 | 波形图区域 | Canvas绘制，渐变填充，高度自适应容器 |
| 主编辑页 | 时间轴区域 | 刻度线标尺，红色标记竖线，hover高亮 |
| 主编辑页 | 歌词编辑面板 | 可折叠（300px宽），多个输入框，黄色未保存边框 |
| 主编辑页 | 歌词预览区域 | 60%宽度（最大800px），居中显示，淡入动画 |
| 主编辑页 | 在线用户列表 | 200px宽，头像+状态，滑入动画 |
| 主编辑页 | 播放控制栏 | 固定底部，进度条，控制按钮组 |
| 主编辑页 | 通知提示 | 右下角浮动，黄色背景，自动消失 |

### 4.3 响应式设计
- 桌面优先布局
- 屏幕宽度 < 768px：左侧编辑面板和右侧用户列表折叠为浮动按钮
- 波形图和预览区域自适应宽度
- 触摸设备优化拖拽交互
