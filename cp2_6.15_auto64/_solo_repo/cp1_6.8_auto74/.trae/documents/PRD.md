## 1. 产品概述

「絮语浮光」是一款交互式心情日记可视化工具，用户每天记录一段文字并选择代表心情的颜色，系统将日记封装为发光气泡漂浮在动态「心情墙」上。通过温暖的视觉语言和轻柔的动画交互，让情绪记录变成一场沉浸式的光影体验。

- 目标用户：注重情绪觉察与表达的年轻人群，追求美学体验的日记爱好者
- 核心价值：将文字日记从静态列表升级为动态可视化艺术墙，让情绪记录兼具仪式感与美感

## 2. 核心功能

### 2.1 用户角色

无需角色区分，单用户本地使用。

### 2.2 功能模块

1. **心情墙页面**：Canvas 绘制的动态气泡墙，气泡浮动、呼吸光晕、鼠标悬停/点击交互
2. **写日记页面**：毛玻璃输入框，文字输入 + 8种心情颜色选择 + 提交
3. **统计页面**：情绪占比圆环图 + 过去7天心情趋势折线图

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 心情墙 | 气泡画布 | Canvas 渲染大量半透明发光气泡，随机漂浮、缓慢上下浮动，气泡颜色根据心情渐变 |
| 心情墙 | 悬停交互 | 鼠标悬停气泡时微微放大，显示日记摘要前几个字 |
| 心情墙 | 点击交互 | 点击气泡触发彩色粒子爆散动画，弹出毛玻璃卡片展示完整日记、日期和情绪标签 |
| 写日记 | 输入框 | 毛玻璃弹窗，包含文字输入区域和8种预设心情颜色选择器 |
| 统计 | 圆环图 | 按颜色分类的情绪占比 Canvas 圆环图 |
| 统计 | 折线图 | 过去7天心情趋势 Canvas 折线图 |

## 3. 核心流程

用户打开应用 → 看到心情墙上的气泡浮动 → 点击底部「写日记」→ 弹出毛玻璃输入框 → 输入文字并选择心情颜色 → 提交 → 新气泡淡入漂浮到墙上 → 鼠标悬停查看摘要 → 点击气泡查看完整日记 → 点击「统计」切换到统计页面查看图表

## 4. 用户界面设计

### 4.1 设计风格

- 主色：浅米色 (#FFF8F0) 到淡粉 (#FFE4E8) 渐变背景
- 辅色：8种马卡龙心情色（淡红、橘、黄、绿、薄荷、天蓝、淡紫、玫瑰粉）
- 按钮风格：圆角胶囊型，半透明毛玻璃效果，柔和阴影
- 字体：正文使用柔和圆润的中文字体，标题使用更有装饰感的字体
- 布局：全屏 Canvas 气泡墙 + 底部固定导航栏 + 浮层弹窗
- 图标风格：简洁线条图标，来自 lucide-react

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 心情墙 | 气泡画布 | 全屏Canvas，浅米色-淡粉渐变背景，半透明渐变气泡带发光阴影和呼吸光晕动画 |
| 心情墙 | 悬停提示 | 气泡微微放大（1.15x），顶部浮现小标签显示日记前几个字 |
| 心情墙 | 点击爆散 | 彩色粒子从气泡中心向四周飞散，2秒内淡出消失 |
| 心情墙 | 日记卡片 | 居中弹出毛玻璃卡片（backdrop-filter: blur），显示完整日记文本、日期、情绪色标签 |
| 写日记 | 输入弹窗 | 居中毛玻璃弹窗，textarea文字输入，8个圆形颜色选择器，提交按钮 |
| 统计 | 圆环图 | Canvas 绘制的环形图，各颜色段对应情绪占比，中心显示总数 |
| 统计 | 折线图 | Canvas 绘制的7天趋势图，X轴日期，Y轴情绪值，带渐变填充 |
| 底部导航 | 导航栏 | 固定底部半透明栏，「写日记」和「统计」两个按钮，毛玻璃效果 |

### 4.3 响应式适配

- 桌面端（≥1024px）：气泡墙全屏展示，气泡数量多（30-50个），Canvas 分辨率高
- 平板端（768px-1023px）：气泡数量适中（15-30个），触控优化（touch 事件支持）
- 移动端（<768px）：暂不作为主要适配目标，但保证基本可用性

### 4.4 动画与性能

- 气泡浮动：正弦波上下浮动，周期3-6秒随机
- 呼吸光晕：径向渐变透明度脉冲，周期2-4秒
- 粒子爆散：60-80个粒子，初始速度随机方向，带重力和阻力衰减
- 缓动函数：cubic-bezier 用于所有 UI 过渡
- 帧率目标：60fps，使用 requestAnimationFrame 驱动 Canvas
