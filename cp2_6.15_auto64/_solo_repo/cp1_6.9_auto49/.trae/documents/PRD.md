## 1. 产品概述

「符文编织·元素共鸣」是一款基于浏览器的2D策略解谜游戏，玩家通过在六边形网格棋盘上拖拽组合不同属性的符文石，触发元素共振获得高分。游戏旨在解决传统消消乐类游戏策略深度不足的问题，引入元素联动、形状匹配和连击机制，提供更具挑战性和沉浸感的游戏体验。

- **核心目标**：在60秒内通过策略性放置符文石触发共振，累积尽可能高的分数
- **目标用户**：休闲游戏爱好者、策略解谜玩家
- **核心价值**：深度策略玩法 + 华丽视觉特效 + 沉浸式音效体验

## 2. 核心功能

### 2.1 功能模块

1. **主游戏界面**：六边形棋盘、符文石、信息栏、控制按钮
2. **符文石系统**：生成、拖拽放置、交换、五种元素属性（火/水/风/土/光）
3. **共振检测系统**：直线、三角形、菱形三种形状匹配检测
4. **粒子特效系统**：消除爆炸粒子、元素潮汐旋涡、金色光环
5. **音效系统**：Web Audio API 八音盒音效，五种元素不同频率
6. **元素潮汐机制**：连续三次不同属性共振触发的特殊事件
7. **计分与评级系统**：分数计算、连击加成、S-D级评级

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主游戏页面 | 信息栏 | 倒计时显示、分数显示（跳动动画）、连击计数、元素提示 |
| 主游戏页面 | 六边形棋盘 | 37格可旋转棋盘、符文石放置、拖拽交互 |
| 主游戏页面 | 控制按钮组 | 左旋15°、右旋15°、重置视角、两倍速切换 |
| 主游戏页面 | 评级弹窗 | 游戏结束时显示最终分数、评级、共振种类数量 |

## 3. 核心流程

```
用户启动游戏
    ↓
初始化棋盘（随机生成8-12颗符文石）
    ↓
[游戏循环]
    ↓
玩家拖拽符文石 → 放置/交换（0.3s动画）
    ↓
检测共振形状（直线/三角形/菱形）
    ↓
├─ 触发共振 → 高亮闪烁（0.5s）→ 爆炸粒子（0.6s）→ 播放音效
│       ↓
│   相邻符文石获得能量加成（5秒得分倍率）
│       ↓
│   记录共振属性 → 检查是否连续3次不同属性
│       ↓
│   └─ 是 → 触发元素潮汐：金色光环 + 洗牌 + 金色符文 + 旋涡特效
    ↓
更新分数、连击计数
    ↓
倒计时是否归零？
    ├─ 否 → 继续游戏循环
    └─ 是 → 计算评级 → 显示结果弹窗
```

## 4. 用户界面设计

### 4.1 设计风格

- **整体主题**：深邃神秘的魔法星空风格
- **主色调**：深紫黑渐变背景（#2a1a3a → #1a0a2a），配亮色元素符文
- **元素色板**：
  - 火：#ff4444（炽红）
  - 水：#44aaff（湛蓝）
  - 风：#44ff88（翠绿）
  - 土：#aa8844（棕褐）
  - 光：#ffdd44（金辉）
- **字体**：展示型衬线字体用于标题，现代无衬线字体用于信息显示
- **视觉层次**：棋盘居中发光，信息栏顶部悬浮，控制按钮底部排列
- **动效风格**：流体过渡、粒子爆发、脉动发光、螺旋收缩

### 4.2 页面设计概览

| 页面/模块 | UI元素 | 设计要点 |
|---------|--------|---------|
| 信息栏-倒计时 | 白色大号发光字体 | 顶部居中，text-shadow发光效果，数字变化时微缩放 |
| 信息栏-分数 | 白色字体 | 居中偏右，分数变化时数字跳动弹跳动画（0.2s） |
| 信息栏-连击 | 彩色指示器 | 右侧显示连击数，高亮下一个所需元素 |
| 六边形棋盘 | 37格网格 | 半透明深灰格（#2a2a3a 60%），0.5px亮线边框，外圈径向渐变光晕 |
| 符文石 | 圆形元素图标 | 80%透明度底色，亮色元素符号，悬浮放大1.1倍+外发光 |
| 控制按钮 | 四个圆角按钮 | 底部排列，半透明背景，hover时亮度提升，点击按压效果 |
| 评级弹窗 | 中央模态框 | 毛玻璃背景，大号评级字母（S金/A银/B铜/C蓝/D灰），星星装饰 |

### 4.3 响应式设计

- **桌面端优先**：以 1280×800 为基准设计
- **画布自适应**：Canvas 按窗口尺寸等比缩放，保持棋盘居中
- **触控支持**：移动端适配触摸事件（touchstart/touchmove/touchend）
- **最小尺寸**：支持 800×600 分辨率，超出时显示滚动条

### 4.4 性能指标

- **帧率目标**：稳定 60 FPS
- **动画时长**：符文消除到粒子消散 ≤ 1.2s
- **计时精度**：60秒倒计时误差 ≤ ±0.1s
- **粒子管理**：对象池复用，单帧粒子数 ≤ 500
