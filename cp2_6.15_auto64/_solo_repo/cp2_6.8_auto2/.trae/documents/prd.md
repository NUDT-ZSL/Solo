## 1. 产品概述

在线协作白板是一款支持多人实时协作的绘图工具，适用于远程头脑风暴、教学演示和团队讨论场景。用户可以在同一个画板上画画、写字和贴便利贴，所有操作实时同步。

* 核心价值：打破地理限制，提供沉浸式协作绘图体验

* 目标用户：远程团队、在线教育师生、创意工作者

## 2. 核心功能

### 2.1 用户角色

无需注册，直接加入协作房间即可使用。

| 角色  | 使用方式     | 核心权限         |
| --- | -------- | ------------ |
| 协作者 | 打开页面自动加入 | 绘图、编辑图层、实时同步 |

### 2.2 功能模块

1. **工具栏**：画笔、橡皮擦、文本框工具切换，颜色和粗细选择器
2. **画布区**：HTML5 Canvas 实现的绘图区域，支持鼠标和触摸操作
3. **图层面板**：图层列表、缩略图预览、显示/隐藏切换、拖拽排序

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述                   |
| ---- | ---- | ---------------------- |
| 主页面  | 工具栏  | 画笔/橡皮擦/文本框切换、颜色选择、粗细调节 |
| 主页面  | 画布区  | 自由绘图、网格背景、触摸支持         |
| 主页面  | 图层面板 | 图层列表、缩略图、显示隐藏、拖拽排序     |
| 主页面  | 实时同步 | WebSocket 消息收发、多人操作同步  |

## 3. 核心流程

用户打开页面 → 自动加入协作房间 → 选择绘图工具 → 在画布上绘制 → 操作通过 WebSocket 广播 → 其他用户实时接收并渲染 → 可通过图层面板管理图层
