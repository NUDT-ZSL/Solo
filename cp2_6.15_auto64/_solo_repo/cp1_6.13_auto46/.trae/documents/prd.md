## 1. 产品概述

FitFlow 是一个连锁社区健身房课程管理与动态通知应用，旨在解决会员频繁询问课程名额、临时取消课程后会员白跑一趟的痛点。系统提供实时课程状态展示、一键预约/取消、自动通知和门店满员率热力图分析功能。

- 目标用户：连锁健身房分店管理员和会员
- 核心价值：减少信息不对称，提升课程管理效率和会员体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 分店管理员 | 系统分配账号 | 发布/修改/删除课程、查看满员率热力图 |
| 会员 | 自助注册 | 查看课程日历、预约/取消课程、接收通知 |

### 2.2 功能模块

1. **仪表板页面**：满员率热力图、门店课程列表弹窗、课程管理操作
2. **日历页面**：7天课程时间轴、课程卡片、搜索过滤、预约/取消操作
3. **导航系统**：左侧导航栏（桌面端）、底部Tab栏（移动端）

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 仪表板 /dashboard | 热力图矩阵 | 7列(门店)×10行(时间段)色块矩阵，绿→红渐变表示满员率，点击红色格子弹出该门店该时间段课程列表与预约详情 |
| 仪表板 /dashboard | 课程管理 | 管理员可修改课程信息或删除课程，操作后自动通知已预约会员 |
| 日历 /calendar | 课程时间轴 | 默认显示未来7天，每天每行展示课程卡片序列，含课程名、教练头像占位圆、剩余名额 |
| 日历 /calendar | 课程卡片 | 按类型区分背景色：瑜伽#e9d5ff、力量训练#fed7aa、动感单车#fecaca、普拉提#bfdbfe，圆角12px，悬停上移4px加深阴影 |
| 日历 /calendar | 详情弹窗 | 显示课程完整描述、教练简介、已预约人数列表，提供预约/取消按钮 |
| 日历 /calendar | 搜索框 | 输入教练名或课程名即时过滤，圆角24px，聚焦边框#3b82f6，0.3s淡入淡出 |
| 全局 | Toast通知 | 预约成功绿色/失败红色toast(3秒消失)；课程变更铃铛🔔通知(5秒消失) |
| 全局 | 导航栏 | 桌面端左侧240px，移动端(<768px)底部Tab栏64px |

## 3. 核心流程

### 3.1 会员预约流程
会员打开日历页面 → 浏览7天课程 → 搜索/筛选课程 → 点击课程卡片 → 查看详情弹窗 → 点击预约 → 卡片剩余名额实时更新 → 底部toast通知预约成功

### 3.2 管理员管理流程
管理员登录 → 进入仪表板 → 查看热力图识别高满员率时段 → 点击色块查看课程详情 → 修改或删除课程 → 所有已预约会员收到🔔通知

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，背景#0f172a，卡片面板#1e293b
- **强调色**：亮蓝#3b82f6（主操作）、亮橙#f97316（警示/次要操作）
- **文字色**：主色#e2e8f0
- **按钮样式**：圆角8px，悬停亮度+10%，点击缩放0.95
- **卡片样式**：圆角12px，悬停上移4px加深阴影，过渡0.2s
- **字体**：使用 Outfit 作为显示字体，Source Sans 3 作为正文字体
- **布局风格**：左侧导航 + 主内容区，卡片式内容展示

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 仪表板 | 热力图矩阵 | Canvas色块矩阵，7列×10行，绿(#22c55e)→黄(#eab308)→红(#ef4444)渐变，色块悬停显示数值tooltip |
| 仪表板 | 课程列表弹窗 | 深色面板，列表项显示课程名+预约详情，操作按钮 |
| 日历 | 时间轴 | 垂直7天布局，每天标题行+课程卡片流，卡片按类型着色 |
| 日历 | 搜索框 | 圆角24px，深灰边框#374151，聚焦亮蓝#3b82f6，背景#1e293b |
| 日历 | 课程卡片 | 按类型背景色，内含课程名+教练圆+剩余名额，悬停上移4px |
| 日历 | 详情弹窗 | 居中模态，显示描述/教练/预约列表，底部预约/取消按钮 |
| 全局 | 导航栏 | 深色#1e293b，菜单项悬停#334155圆角8px，活动项亮蓝指示 |

### 4.3 响应式设计

- 桌面优先（≥768px）：左侧导航栏240px + 主内容区
- 移动端（<768px）：导航栏折叠为底部Tab栏64px，日历卡片宽度100%
- 触摸优化：按钮最小点击区域44px，卡片触控友好
