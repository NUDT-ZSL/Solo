## 1. 产品概述

「星尘交响曲」是一款基于 Web 的 3D 交互可视化项目，模拟宇宙星尘在引力作用下汇聚成星云、并随音乐律动闪烁的奇幻场景。用户可以自由旋转、缩放视角，点击粒子触发音乐脉冲，体验沉浸式的太空视听交互。

- 目标用户：视觉艺术爱好者、交互设计从业者、创意编程学习者
- 核心价值：将物理模拟（引力坍缩）与音频生成（Web Audio API）融合于实时 3D 场景中，创造独一无二的视听体验

## 2. 核心功能

### 2.1 功能模块

1. **3D 星尘场景**：深邃太空背景 + 多色粒子分组旋转 + 远景星点
2. **交互控制**：鼠标拖拽旋转、滚轮缩放、点击粒子触发脉冲
3. **引力坍缩**：随机触发粒子向中心聚集后爆发扩散的动画
4. **音乐脉冲**：点击粒子时用 Web Audio API 生成随机音符并产生彩色光晕
5. **控制面板**：旋转速度、粒子密度、音乐音量滑块 + 重置视角按钮
6. **信息卡片**：点击粒子后显示颜色分组、大小、最近引力坍缩时间

## 3. 核心流程

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：纯黑（#000000）背景 + 深空蓝紫渐变光晕
- 粒子颜色分组：蓝（#4488ff）、紫（#aa44ff）、粉（#ff44aa）、青（#44ffcc）、橙（#ff8844）
- 按钮风格：圆角半透明毛玻璃，悬停发光
- 字体：Orbitron（标题/数字），Rajdhani（正文），太空科幻风格
- 布局：全屏 3D 场景 + 浮动 UI 面板
- 图标风格：线性发光图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主场景 | 控制面板 | 左上角固定定位，毛玻璃背景（backdrop-filter: blur），圆角12px，包含3个自定义滑块（旋转速度0.1-3、粒子密度100-2000、音量0-1）和重置视角按钮 |
| 主场景 | 信息卡片 | 右下角固定定位，毛玻璃背景，显示粒子颜色（色块标识）、大小数值、最近坍缩时间戳 |
| 主场景 | 3D 场景 | 全屏 canvas，纯黑背景，远景散布白色小星点，5组彩色发光粒子球体 |
| 主场景 | 光晕效果 | 点击粒子后产生圆形彩色光晕，从粒子位置向外扩散并淡出 |

### 4.3 响应式

- 桌面端优先：全屏 3D 场景
- 移动端适配：控制面板和信息卡片缩小，触摸旋转/缩放支持
- 性能目标：60fps，粒子密度可调节

### 4.4 3D 场景指导

- 环境：纯黑深空，无环境光依赖，粒子自发光为主
- 灯光：微弱环境光（AmbientLight 0x111122）+ 每组粒子中心点光源
- 相机：PerspectiveCamera，FOV 60°，初始距离 80，OrbitControls 交互
- 构图：5组粒子分布在球形空间内，每组绕各自的引力中心旋转
- 交互：OrbitControls 拖拽旋转/缩放，Raycaster 点击检测
- 动画：粒子持续旋转、随机引力坍缩（聚集→爆发）、点击光晕扩散
- 后处理：Bloom 发光效果（UnrealBloomPass）增强粒子辉光
- 性能预算：默认 800 粒子，可调至 2000，目标 60fps
