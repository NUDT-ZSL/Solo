## 1. 产品概述

灵兽驯养师是一款回合制灵兽驯养与对战游戏，玩家在8x8网格野外地图中探索、遭遇灵兽、通过QTE驯服机制捕获灵兽，并组建最多3只灵兽的队伍与AI进行回合制对战。游戏融合探索冒险、驯服QTE、策略对战三大核心玩法，胜利后获得经验并解锁新地图区域。

- 目标用户：喜欢休闲策略游戏、回合制对战、收集养成类游戏的玩家
- 核心价值：将探索冒险、驯服QTE、属性克制对战三种玩法融合，提供有节奏感的游戏体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 自动创建 | 探索地图、驯服灵兽、组建队伍、对战战斗 |

### 2.2 功能模块

1. **探索地图页**：8x8网格世界、地形生态渲染、玩家移动与足迹效果、遭遇事件触发
2. **驯服场景页**：QTE指令驯服、灵兽抵抗值削减、捕获成功/失败动画
3. **对战场景页**：回合制对战、灵兽技能释放、属性克制计算、战斗结果
4. **状态面板**：玩家信息、灵兽队伍、当前区域进度

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 探索地图 | 网格世界渲染 | 8x8网格，按生态区域随机生成森林(绿)、水域(蓝)、火山(红)、草原(黄)地形 |
| 探索地图 | 玩家移动与足迹 | 方向键控制角色移动，脚下留下半透明足迹光点(1.5秒淡出) |
| 探索地图 | 遭遇事件触发 | 移动到特定格子有概率触发遭遇，白光闪烁(0.1秒)+灵兽出场动画(0.6秒粒子扩散) |
| 驯服场景 | QTE指令机制 | 灵兽攻击前摇时出现红色预警圈(0.8秒填充)，玩家按对应方向键(上下左右)对应灵兽弱点方向 |
| 驯服场景 | 抵抗值系统 | 红色血条下方黄色进度条，每次成功削减抵抗值，归零则捕获成功 |
| 驯服场景 | 捕获动画 | 成功：灵兽化作彩色虹光飞入背包(1秒带尾迹)；失败：灵兽消失，损失10体力 |
| 对战场景 | 回合制对战 | 玩家和AI各最多3只灵兽，每回合选择一只释放技能 |
| 对战场景 | 五属性克制 | 火、水、木、光、暗五种属性：火克木、木克水、水克火、光暗互克 |
| 对战场景 | 技能动画 | 全屏动画：镜头拉近→技能图标飞出击中目标→属性特效(0.5秒)→恢复视角 |
| 状态面板 | 玩家信息 | 体力值、经验值、已解锁区域 |
| 状态面板 | 灵兽队伍 | 当前队伍中的灵兽列表，支持切换查看详情 |

## 3. 核心流程

1. 玩家进入探索地图，使用方向键在8x8网格上移动
2. 移动到特定格子触发灵兽遭遇事件（白光闪烁+灵兽出场动画）
3. 进入驯服场景：灵兽做出攻击前摇，出现红色预警圈，玩家在预警圈填满前按下对应方向键
4. 成功按键削减灵兽抵抗值，抵抗值归零则捕获成功（虹光动画），灵兽加入队伍
5. 驯服失败则灵兽消失，玩家损失10体力
6. 玩家组建灵兽队伍（最多3只），发起对战
7. 回合制对战：选择灵兽释放技能，根据属性克制计算伤害
8. 战斗胜利获得经验，经验积累解锁新地图区域

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝色渐变背景 (#0a0e27 → #1a1f4e)，营造暗色奇幻氛围
- **强调色**：金色描边 (#d4a844)，用于面板边框、按钮高亮
- **按钮风格**：圆角矩形，按下时下凹3px物理反馈(0.1秒)，金色流光边框(1.5秒周期)
- **字体**：标题使用带奇幻风格的像素风字体，正文使用易读的无衬线字体
- **布局**：桌面端左右两栏(70%/30%)，移动端上下排列
- **图标风格**：像素风格灵兽图标，切换时0.3秒翻转动画(扑克牌翻面)

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 探索地图 | 网格世界 | Canvas 2D渲染，8x8网格，地形着色(森林绿/水域蓝/火山红/草原黄)，像素风角色32x32，足迹光点半透明淡出 |
| 探索地图 | 遭遇动画 | 白光全屏闪烁0.1秒，灵兽从球形光芒展开+粒子四散0.6秒 |
| 驯服场景 | QTE界面 | 灵兽居中展示，红色预警圈从透明到实心0.8秒，方向键提示(上下左右)，抵抗值黄色进度条，血条红色 |
| 驯服场景 | 结果动画 | 成功：彩色虹光飞向屏幕顶部1秒+尾迹；失败：灵兽消散+体力-10提示 |
| 对战场景 | 战斗界面 | 双方灵兽左右对峙，技能选择面板3个技能按钮，属性克制提示，血条+技能CD显示 |
| 对战场景 | 技能动画 | 镜头拉近→技能图标飞出→击中目标→属性特效0.5秒→恢复视角 |
| 状态面板 | 玩家信息 | 体力条、经验条、区域进度、灵兽队伍列表(翻转切换动画) |

### 4.3 响应式设计

- 桌面端(≥768px)：左右两栏布局，左70%主内容区(地图/驯服/对战)，右30%状态信息区
- 移动端(<768px)：上下排列，主内容区在上，状态信息区在下可折叠
- 最小适配宽度320px，最大1920px
- 所有页面切换0.2秒淡入过渡

### 4.4 性能要求

- 驯服QTE响应延迟 ≤ 50ms
- 对战动画帧率稳定60fps
- Canvas 2D渲染优化，使用requestAnimationFrame
