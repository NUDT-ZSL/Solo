## 1. 产品概述
星轨拼图是一款将经典俄罗斯方块玩法与星座图案拼接相结合的浏览器休闲游戏。玩家通过消除方块，在背景中逐渐点亮星点，最终完成一幅完整的星空画卷。
- 目标用户：像素艺术爱好者、休闲游戏玩家
- 产品价值：融合经典消除玩法与艺术创作体验，提供兼具挑战性与美学享受的游戏过程

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 无需注册，直接访问 | 进行游戏、查看得分、完成星座 |

### 2.2 功能模块
1. **游戏主界面**：俄罗斯方块棋盘、背景星空、信息面板、控制按钮
2. **方块控制系统**：左右移动、旋转、软降、硬降、下一块预览
3. **行消除系统**：消除检测、渐隐动画、星点点亮、连线绘制
4. **特效系统**：Tetris流星特效、星座轮廓动画、胜利粒子爆炸
5. **星座进度系统**：星点计数、12种星座模板、胜利判定

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主页 | 棋盘区域 | 10×20网格方块下落与消除 |
| 游戏主页 | 背景星空 | 32颗星点逐渐点亮，星座连线绘制 |
| 游戏主页 | 信息面板 | 下一块预览、星点计数器、当前得分 |
| 游戏主页 | 控制按钮 | 左移、右移、旋转、软降、硬降5个像素风按钮 |
| 游戏主页 | 胜利界面 | 全屏星座图、"星河绘成"文字、粒子爆炸特效 |

## 3. 核心流程
玩家通过键盘或按钮控制下落的方块，将其排列成完整的行。消除一行时，对应位置的星点被点亮并与相邻已点亮星星连线。每消除7颗星显示星座轮廓提示，一次消除4行触发流星特效。全部32颗星点亮后游戏胜利。

```mermaid
flowchart TD
    A["开始游戏"] --> B["方块生成并下落"]
    B --> C["玩家操作方块"]
    C --> D{"方块是否触底?"}
    D -- 否 --> C
    D -- 是 --> E{"是否有完整行?"}
    E -- 否 --> B
    E -- 是 --> F["播放消除渐隐动画"]
    F --> G["根据消除行点亮对应星点"]
    G --> H["绘制星点间连线"]
    H --> I{"是否Tetris(4行)?"}
    I -- 是 --> J["触发流星特效"]
    J --> K
    I -- 否 --> K
    K {"已点亮星数 % 7 == 0?"}
    K -- 是 --> L["显示星座轮廓动画"]
    L --> M
    K -- 否 --> M
    M {"全部32颗点亮?"}
    M -- 否 --> B
    M -- 是 --> N["切换全屏星座图"]
    N --> O["显示胜利信息与粒子爆炸"]
    O --> P["游戏结束"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深蓝渐变（#0B0D17 → #1A1F33）
- 强调色：亮黄 #FFD700（星点）、蓝紫 #6A5ACD（粒子）
- 辅助色：#4A6B8A（边框）、#5A7A9A（虚线边框）、#2A3A4A（按钮背景）
- 按钮风格：圆角4px，像素风格，按下时有背景色变化过渡
- 字体：像素风格等宽字体，数字大字醒目
- 布局：中央棋盘 + 右侧信息面板，桌面端水平排列，移动端垂直排列
- 过渡动画：所有元素0.2秒平滑过渡

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主页 | 棋盘区域 | 320×640px，圆角8px，1px实线#4A6B8A边框，半透明深蓝背景rgba(15,20,40,0.8) |
| 游戏主页 | 信息面板 | 宽150px，包含下一块预览区(120×120px圆角4px虚线边框)、星点计数器(亮黄大字+渐变星星图标)、得分显示 |
| 游戏主页 | 控制按钮 | 5个60×40px像素风格按钮，圆角4px，背景#2A3A4A，边框#4A6A8A，按下变#4A6A8A |
| 游戏主页 | 背景星空 | 全屏深蓝渐变，32颗灰白色星点(#555555)逐渐点亮为金色(#FFD700) |
| 胜利界面 | 胜利区域 | "星河绘成"大字，中心向四周扩散的彩色粒子爆炸(200颗，2秒) |

### 4.3 响应式设计
- 桌面优先设计
- 窗口宽度 < 800px 时：
  - 信息面板移至底部并改为水平排列
  - 控制按钮放大为 80px 宽
- 触摸优化：按钮尺寸适合手指点击

### 4.4 性能指标
- 游戏循环稳定 60FPS
- 消除动画和流星特效期间帧率 ≥ 50FPS
- 单帧 Canvas 绘制耗时 ≤ 16ms
