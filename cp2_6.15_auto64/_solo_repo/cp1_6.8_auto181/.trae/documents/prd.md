## 1. 产品概述

**音律流光**是一款交互式音乐可视化工具，将音频的频率、节奏与音量数据转化为沿圆形轨道流动的动态粒子流，配合爆散光晕与频谱信息卡片，让用户"看见"音乐。
- 目标用户：音乐爱好者、创意工作者、视觉艺术爱好者
- 核心价值：将听觉体验转化为沉浸式视觉体验，提供直觉化的频谱理解与情感标注

## 2. 核心功能

### 2.1 用户角色
无需注册，所有访客均为普通用户，直接使用全部功能。

### 2.2 功能模块
1. **主视觉页面**：3D粒子流可视化画布、音频控制栏、毛玻璃信息卡片
2. **音频选择面板**：预设古典乐片段选择 + 本地音频上传（≤30秒）

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主视觉页面 | 粒子画布 | Canvas全屏渲染粒子流，粒子沿圆形轨道运动，低音红紫/中音青绿/高音金黄配色 |
| 主视觉页面 | 音频控制栏 | 播放/暂停、进度条、音量调节、当前曲目信息 |
| 主视觉页面 | 爆散光晕特效 | 节奏强拍时粒子向外爆散，并扩散出环形光晕 |
| 主视觉页面 | 毛玻璃信息卡片 | 点击粒子群暂停，显示频谱分析图与情感标签 |
| 主视觉页面 | 视角交互 | 鼠标拖拽旋转视角、滚轮缩放 |
| 音频选择面板 | 预设选择 | 提供多个古典乐片段供选择播放 |
| 音频选择面板 | 文件上传 | 支持上传本地短音频（≤30秒），实时分析并可视化 |

## 3. 核心流程

用户打开页面 → 选择预设片段或上传音频 → 系统加载并解码音频 → 实时FFT分析 → 按频段生成对应颜色粒子 → 粒子沿圆形轨道流动 → 节奏强拍触发爆散与光晕 → 用户可拖拽旋转/缩放视角 → 点击粒子群暂停查看频谱卡片 → 关闭卡片恢复播放

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：黑色(#050510)到深蓝紫(#0a0a2e)渐变背景
- **粒子配色**：低音红紫(#e040fb/#9c27b0)、中音青绿(#00e5ff/#00c853)、高音金黄(#ffd740/#ffab00)
- **控件风格**：毛玻璃(glassmorphism)面板，backdrop-blur + 半透明白色边框
- **字体**：中文使用"思源黑体"或系统默认，英文标题使用 Orbitron（科幻感）
- **圆角**：12-16px 柔和圆角
- **发光边缘**：粒子与UI元素带有柔和的box-shadow光晕
- **布局**：全屏画布 + 浮动控制面板叠加
- **动画**：页面切换缓动淡入(ease-in-out)，背景星点缓慢飘浮

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主视觉页面 | 粒子画布 | 全屏Canvas，黑色到深蓝紫渐变，背景细小星点飘浮 |
| 主视觉页面 | 音频控制栏 | 底部毛玻璃条栏，播放按钮、进度条、音量滑块、曲目名 |
| 主视觉页面 | 信息卡片 | 毛玻璃卡片居中弹出，频谱柱状图、情感标签、关闭按钮 |
| 主视觉页面 | 选择面板 | 左侧毛玻璃侧边栏，预设列表+上传按钮 |

### 4.3 响应式设计
- 桌面优先设计，Canvas自适应窗口尺寸
- 控制面板在小屏幕下折叠为底部抽屉
- 粒子数量根据屏幕尺寸动态调整以维持60fps

### 4.4 3D场景指引
- **环境**：深空极简风，远处细小星点缓慢飘浮
- **光照**：粒子自发光，无外部光源依赖
- **相机**：正交/透视投影，默认俯视角度，支持拖拽旋转与缩放
- **构图**：圆形轨道居中，粒子沿轨道分布
- **交互**：拖拽旋转视角，滚轮缩放，点击选中粒子群
- **后处理**：光晕(bloom)效果增强粒子发光感
