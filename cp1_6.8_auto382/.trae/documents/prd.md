## 1. 产品概述

「匿名画廊」是一个无需登录的匿名图片分享平台。用户可以上传图片并添加文字描述，系统自动生成唯一短链接供分享，查看者可匿名评论。所有公开图片以瀑布流形式展示在「画廊广场」，整体视觉走极简毛玻璃风格。

- 目标用户：希望匿名分享图片、获取匿名反馈的互联网用户
- 核心价值：零门槛匿名创作分享，极简美学体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 匿名访客 | 无需注册 | 上传图片、浏览画廊、查看详情、发表评论 |

### 2.2 功能模块

1. **画廊广场页**：瀑布流图片展示、图片上传入口、悬停交互效果
2. **图片详情页**：图片查看、文字描述展示、匿名评论区
3. **上传功能**：图片选择、文字描述输入（可选）、短链接生成

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 画廊广场 | 瀑布流卡片列表 | 展示所有公开图片，卡片含缩略图、描述片段、评论数；悬停放大+阴影加深 |
| 画廊广场 | 上传弹窗 | 点击上传按钮弹出模态窗，选择图片、输入描述、提交后返回短链接 |
| 图片详情 | 图片展示区 | 全尺寸图片（淡入动画）+ 完整描述文字 |
| 图片详情 | 评论区 | 匿名评论列表（气泡样式、打字机弹出效果）+ 评论输入框（50字限制） |

## 3. 核心流程

**上传流程**：用户点击上传 → 选择图片文件 → 填写描述（可选）→ 提交 → 后端保存图片并生成短链接 → 前端展示短链接

**浏览流程**：用户进入画廊广场 → 瀑布流浏览图片 → 点击卡片进入详情页 → 查看图片和描述 → 可匿名留言评论

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：柔和蓝紫色渐变（从 `#7C83FD` 到 `#A855F7`）
- **背景**：浅灰到白色渐变（`#F0F0F5` → `#FFFFFF`）
- **卡片/面板**：半透明磨砂玻璃质感（`backdrop-filter: blur(16px)`，白色半透明背景）
- **按钮**：圆角胶囊形状（`border-radius: 999px`），蓝紫渐变填充，悬停微光效果
- **字体**：标题用 `Playfair Display`，正文用 `Noto Sans SC`
- **布局**：卡片式布局，瀑布流排列，顶部导航栏

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 画廊广场 | 导航栏 | 毛玻璃背景，Logo「匿名画廊」居左，上传按钮居右 |
| 画廊广场 | 瀑布流区域 | CSS columns 瀑布流，卡片毛玻璃质感，缩略图+描述片段+评论数徽标 |
| 画廊广场 | 上传弹窗 | 毛玻璃模态窗，拖拽上传区域，文本输入框，胶囊提交按钮 |
| 图片详情 | 图片区域 | 全宽图片展示，淡入动画（opacity 0→1 + translateY 20→0） |
| 图片详情 | 描述区 | 图片下方文字描述区域 |
| 图片详情 | 评论区 | 气泡样式评论卡片，打字机逐条弹出动画，底部输入框+发送按钮 |

### 4.3 响应式适配

- **桌面端优先**：3-4列瀑布流，详情页双栏（图片+评论）
- **平板**：2列瀑布流，详情页单栏
- **手机**：1-2列瀑布流，详情页单栏，评论区全宽
- **触控优化**：按钮最小 44px 点击区域，评论输入框适配软键盘

### 4.4 动效规范

- **卡片悬停**：`transform: scale(1.03)`，`box-shadow` 加深，过渡 0.3s ease
- **图片淡入**：初始 `opacity: 0; transform: translateY(20px)` → `opacity: 1; transform: translateY(0)`，持续 0.6s ease-out
- **评论弹出**：从底部依次出现，每条延迟 0.15s，打字机文字效果
- **弹窗出现**：`scale(0.95) → scale(1)` + 毛玻璃背景渐现
