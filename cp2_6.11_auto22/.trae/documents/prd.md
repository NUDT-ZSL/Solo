## 1. 产品概述

「声纹相簿」是一个全栈Web应用，让用户上传音频文件，系统自动提取声音频谱特征并生成独特的抽象色彩图像（声纹图），用户可以浏览、收藏、分享声纹图，并为每张声纹图添加文字故事或标签。

- 目标用户：音乐爱好者、播客创作者、声音艺术家，以及任何希望将声音视觉化的用户
- 核心价值：将听觉体验转化为视觉艺术，为声音赋予独特的视觉身份

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 邮箱注册 | 上传音频、浏览声纹图、收藏、标签、分享、搜索 |

### 2.2 功能模块

1. **登录/注册页**：邮箱注册和登录表单，登录后跳转个人声纹画廊
2. **声纹画廊页**：3列网格展示所有声纹图卡片，顶部搜索框，右上角上传按钮
3. **声纹详情页**：大尺寸声纹图展示、频段能量圆环进度条、文字故事编辑、收藏和标签管理

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

## 3. 核心流程

用户注册/登录 → 进入个人声纹画廊（骨架屏加载）→ 点击上传按钮选择音频文件 → 显示加载动画 → 后端提取MFCC频谱特征返回能量分布 → 前端Canvas绘制抽象声纹图 → 声纹图卡片出现在画廊中 → 点击卡片进入详情页 → 查看频段能量、编辑故事、收藏、添加标签 → 通过分享按钮复制链接

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深灰蓝#0F1923，卡片背景半透明深灰#1E2A3A
- **强调色**：青蓝#00BCD4到靛蓝#3F51B5渐变（按钮），橙/绿/紫（频段）
- **按钮风格**：圆角渐变，悬停亮度提升
- **字体**：标题使用Outfit，正文使用Noto Sans SC
- **布局风格**：深色模式，磨砂玻璃卡片，3列网格画廊
- **图标风格**：Lucide线性图标，心形收藏图标带动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录/注册页 | 表单区域 | 深色背景，居中卡片，圆角输入框(border-radius:12px)，边框#2A3A4A，聚焦变#00BCD4 |
| 声纹画廊页 | 导航栏 | 顶部固定，左侧logo，中间搜索框，右侧上传按钮(青蓝到靛蓝渐变) |
| 声纹画廊页 | 卡片网格 | 3列布局，卡片backdrop-filter:blur(10px)，悬停上浮5px+阴影(transition 0.3s) |
| 声纹画廊页 | 卡片内容 | 左上角日期(YYYY-MM-DD)，中央声纹图缩略图，右下角删除按钮 |
| 声纹详情页 | 声纹图 | 70%页面宽度，中央展示 |
| 声纹详情页 | 频段环 | 三个SVG圆环进度条(橙#FF9800/绿#4CAF50/紫#9C27B0) |
| 声纹详情页 | 标签区 | 胶囊形状标签，背景从#E0F7FA到#FCE4EC渐变 |
| 声纹详情页 | 分享提示 | 半透明黑色背景，居中显示1.5秒渐隐 |

### 4.3 响应式设计

- 桌面优先（≥768px）：3列网格布局
- 平板（<768px）：2列网格布局
- 手机（<480px）：1列网格布局
- 搜索框和输入框自适应宽度

### 4.4 声纹图视觉映射规则

- **高频能量** → 明亮黄色和橙色放射状线条
- **中频能量** → 绿色和蓝色波浪形色块
- **低频能量** → 紫色和深蓝色背景渐变
- 不同频段图案叠加产生层次感，每张声纹图根据频谱数据独特生成
