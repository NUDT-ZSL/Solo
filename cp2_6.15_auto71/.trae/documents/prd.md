## 1. 产品概述

海洋洋流漂浮物迁移路径交互式可视化应用，面向海洋生态研究者和环保人士，解决不同洋流模式和季节变化下污染物扩散范围难以直观预测的问题。通过动态向量场可视化、粒子追踪模拟和交互式控制面板，让用户直观观察和理解海洋漂浮物的迁移规律。

## 2. 核心功能

### 2.1 功能模块

1. **海洋场景可视化页**: 俯视矩形海域、流速场着色网格、洋流方向箭头、视角旋转缩放
2. **控制面板**: 洋流模式切换、季节偏移调节、粒子释放速率控制
3. **粒子交互系统**: 点击释放粒子、洋流跟随运动、尾迹显示、淡出消失
4. **地形障碍系统**: 随机岛屿生成、粒子碰撞堆积、堆积闪烁提示

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 海洋场景 | 流速场可视化 | 底部动态着色网格（#0d1b2a到#4fc3f7渐变），白色箭头线指示洋流方向（每40px一个，长度与流速成正比，最大20px） |
| 海洋场景 | 视角控制 | 鼠标拖拽旋转视角（Y轴，-60°到60°），滚轮缩放（0.5x到3x） |
| 控制面板 | 洋流模式选择 | 下拉选择：北太平洋环流、南大西洋环流、印度洋季风环流，切换后0.6s ease-in-out过渡动画 |
| 控制面板 | 季节偏移 | 滑块0-3（春夏秋冬），改变洋流强度和路径 |
| 控制面板 | 粒子释放速率 | 滑块1-10个/秒，从海域中心释放漂浮物粒子 |
| 海洋场景 | 粒子释放与运动 | 点击释放5个红色粒子（#ff5252，半径3px），跟随洋流运动（速度上限80px/s），尾迹半透明（alpha 0.3，长度15px），5秒淡出消失，上限500个 |
| 海洋场景 | 岛屿与碰撞 | 5-8个随机椭圆岛屿（40-80px，#4e342e），粒子碰撞停止并堆积，超10个变橙色#ff9800闪烁（0.5s周期） |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**: 深蓝色海洋主题，主背景#0a0e27
- **流速场**: 深蓝#0d1b2a到浅蓝#4fc3f7渐变
- **控制面板**: 毛玻璃效果（rgba(26,26,46,0.85)，背光模糊8px），宽280px，圆角12px，内边距16px
- **按钮**: 圆角8px，悬停亮度提高15%+上移translateY(-2px)，过渡0.2s
- **粒子**: 红色#ff5252（正常），橙色#ff9800（堆积闪烁）
- **岛屿**: 椭圆形#4e342e
- **字体**: 科技感无衬线字体
- **布局**: 全屏无滚动，左侧控制面板浮动于海域Canvas之上

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 海洋场景 | 海域Canvas | 1200x800px矩形区域，动态着色网格+白色箭头，支持拖拽旋转和滚轮缩放 |
| 控制面板 | 洋流模式下拉 | 下拉选择框，深色背景，白色文字，选中项高亮 |
| 控制面板 | 季节偏移滑块 | 自定义样式滑块，0-3四档，显示当前季节名称 |
| 控制面板 | 释放速率滑块 | 自定义样式滑块，1-10范围，显示当前速率值 |
| 海洋场景 | 粒子层 | 红色圆形粒子+半透明尾迹，淡出动画 |
| 海洋场景 | 岛屿层 | 椭圆棕色静态障碍物，堆积粒子橙色闪烁 |

### 4.3 响应式

- 桌面优先设计，Canvas区域1200x800px
- 控制面板固定左侧浮动，不随缩放变化
- 全屏无滚动布局

### 4.4 3D场景指引

- 俯视矩形海域，支持Y轴旋转视角（-60°到60°）
- 通过2D Canvas变换矩阵模拟3D透视效果
- 缩放范围0.5x到3x
