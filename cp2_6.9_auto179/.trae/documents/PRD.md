## 1. 产品概述
「梦境迷宫」是一个基于Three.js的3D交互式可视化探索应用，通过随机生成的彩色半透明迷宫、漂浮光点粒子和梦幻视觉效果，为用户提供沉浸式的探索体验。
- 核心目的：创造一个具有艺术美感和交互趣味性的3D迷宫探索体验
- 目标用户：对创意交互、3D可视化、艺术体验感兴趣的用户

## 2. 核心特性

### 2.1 功能模块
1. **3D迷宫系统**：随机生成9x9网格迷宫，墙体具有呼吸脉动和颜色渐变效果
2. **粒子系统**：60个漂浮光点粒子，公转与上下浮动运动，接触触发粒子爆发
3. **用户控制**：WASD键控制半透明球形光标移动，自动沿路径前进
4. **视角系统**：接触光点触发俯视视角切换动画，自动恢复
5. **迷宫重构**：每60秒自动重构，空格键手动触发（15秒冷却），带淡入淡出动画和光晕效果
6. **UI界面**：左上角半透明控制面板，显示操作提示和冷却倒计时

## 3. 核心流程

用户进入应用后，自动生成初始迷宫和粒子。用户通过WASD键控制金色光标在迷宫中穿行，移动过程中可以观察墙体的呼吸脉动效果和漂浮光点。当光标接触到光点粒子时，粒子爆发扩散，同时相机视角短暂切换为俯视视角后恢复。每60秒迷宫自动重构一次，用户也可按空格键手动触发重构（需注意15秒冷却时间）。

```mermaid
flowchart TD
    "A[启动应用]" --> "B[生成初始迷宫和粒子]"
    "B" --> "C[等待用户输入]"
    "C" --> "D{WASD移动?}"
    "D" -->|是| "E[移动光标并检测碰撞]"
    "E" --> "F{接触光点?}"
    "F" -->|是| "G[粒子爆发+视角切换]"
    "F" -->|否| "C"
    "G" --> "C"
    "D" -->|否| "H{空格键?}"
    "H" -->|是且冷却完成| "I[触发迷宫重构]"
    "H" -->|否或冷却中| "J{60秒到达?}"
    "J" -->|是| "I"
    "J" -->|否| "C"
    "I" --> "K[淡出动画2秒]"
    "K" --> "L[重新生成迷宫和粒子]"
    "L" --> "M[淡入动画2秒+光晕闪烁]"
    "M" --> "C"
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深紫色背景(#0F0C29)，HSL渐变墙体，冷白色光点
- **整体调性**：梦幻、半透明、呼吸感、艺术化
- **动画过渡**：所有交互0.2秒动画反馈，流畅自然
- **视觉层次**：墙体半透明叠加，光点辉光提亮，UI面板悬浮

### 4.2 页面设计
| 区域 | 元素 | 设计细节 |
|------|------|---------|
| 全屏背景 | 3D画布 | 深紫色(#0F0C29)背景，雾化效果增强空间感 |
| 迷宫墙体 | 几何体 | 9x9网格，HSL色环配色（#FF6B9D, #6A5ACD, #20B2AA等），透明度0.7，呼吸脉动 |
| 粒子光点 | Sprite/Points | 冷白到淡蓝渐变(#E0F7FA→#B3E5FC)，柔和辉光，漂浮运动 |
| 用户光标 | 球体 | 金色半透明(#FFD700)，半径0.3 |
| 左上角 | UI面板 | #1A1A2E底色，透明度0.6，圆角8px，白色14px文字 |
| 重构效果 | 屏幕边缘 | 白色光晕闪烁0.3秒 |

### 4.3 响应式
- 桌面端优先，全屏自适应
- Canvas自适应窗口大小变化
- UI面板固定左上角，不随缩放变化

### 4.4 3D场景指引
- **环境**：深紫色雾化背景，增强梦幻空间感
- **光照**：环境光+方向光，保证墙体半透明效果和粒子辉光可见
- **相机**：默认跟随光标第三人称视角，俯视切换时距离10单位、仰角60度
- **动画**：墙体呼吸脉动、粒子公浮动、光标移动、视角切换、重构淡入淡出
- **性能**：帧率30fps以上，粒子数80以内，重构动画流畅不卡顿
