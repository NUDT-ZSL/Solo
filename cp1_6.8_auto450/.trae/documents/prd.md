## 1. 产品概述
「咒印流光」是一款2D动作解谜游戏，玩家操控带有发光咒印的小球，在布满古老符文和光桥的悬浮平台上，通过点击和长按激活不同颜色咒印来操控机关，按特定顺序解谜通关。目标用户为休闲解谜游戏爱好者，核心卖点为古老符文与赛博光效融合的独特视觉风格和咒印激活的解谜玩法。

## 2. 核心功能

### 2.1 功能模块
1. **游戏主界面**: 全屏Canvas游戏画面，包含平台、小球、机关、符文等全部游戏元素渲染
2. **关卡系统**: 包含至少一个完整关卡，展示核心解谜流程

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主界面 | Canvas游戏画面 | 渲染悬浮平台、符文纹理、光桥/光门/光柱机关、玩家小球、粒子特效 |
| 游戏主界面 | 咒印序列指示器 | 左上角显示3个彩色小圆点，按顺序闪烁提示当前需要激活的咒印 |
| 游戏主界面 | 暂停/重置按钮 | 右上角毛玻璃风格暂停和重置按钮 |
| 游戏主界面 | 当前咒印徽章 | 底部中间显示当前激活的咒印图标，圆形发光徽章，颜色随激活变化 |
| 游戏主界面 | 关卡通关特效 | 所有符文闪烁并汇聚成大型传送光柱 |

## 3. 核心流程
玩家进入游戏后，小球出现在起始平台上。左上角显示当前关卡的咒印序列提示（3个按顺序闪烁的彩色圆点）。玩家点击或长按屏幕/按键激活对应颜色的咒印，小球的光晕颜色随之变化。按正确顺序激活咒印后，对应机关（红光桥升起、蓝光桥下降、绿光门开启、金光柱传送）触发，小球可以继续前进。若顺序错误则机关不触发，需要重新尝试。所有机关正确触发后，关卡通关，符文闪烁汇聚成传送光柱。

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深墨蓝(#0a0e27)到藏青(#1a1a4e)渐变背景
- 强调色：红(#ff4444)、蓝(#4488ff)、绿(#44ff88)、金(#ffcc44)四种咒印色
- 小球：暖白色(#fff8e7)发光球体
- 按钮：毛玻璃效果，半透明暗色底
- 字体：无字体需求（纯Canvas渲染）
- 布局：全屏Canvas，UI元素叠加渲染

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主界面 | 背景层 | 深墨蓝到藏青渐变，微弱星尘粒子飘浮 |
| 游戏主界面 | 平台层 | 半透明暗色石板，发光符文纹理，边缘柔和光晕 |
| 游戏主界面 | 小球 | 暖白发光球体，环形彩色光晕，移动星尘拖尾 |
| 游戏主界面 | 机关层 | 红蓝光桥（升降）、绿光门（开关）、金光柱（传送），粒子升腾+光晕脉冲 |
| 游戏主界面 | 咒印序列 | 左上角3个彩色小圆点，按序闪烁 |
| 游戏主界面 | 控制按钮 | 右上角毛玻璃暂停/重置按钮 |
| 游戏主界面 | 咒印徽章 | 底部中间圆形发光徽章 |

### 4.3 响应式
- 桌面端优先，Canvas自适应窗口大小
- 键盘操控为主（方向键移动，1/2/3/4切换咒印，空格激活）
- 支持鼠标点击UI按钮
