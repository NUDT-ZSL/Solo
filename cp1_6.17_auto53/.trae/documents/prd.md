## 1. 产品概述

声波迷宫是一款基于声波共鸣机制的迷宫探索游戏。玩家在由声音驱动的迷宫环境中，通过发射不同频率和节奏的声波来解锁路径、激活机关并收集共鸣碎片。目标用户为喜欢解谜和音游的休闲玩家，核心卖点在于声波与物理墙壁交互的独特玩法。

## 2. 核心功能

### 2.1 功能模块

1. **迷宫探索页面**：Canvas绘制的8x8/10x10/12x12随机迷宫，玩家角色移动，声波可视化传播
2. **声波控制面板**：波形选择（正弦/方波/三角）、频率滑块（200-2000Hz）、发射/停止按钮、实时波形与频谱显示
3. **HUD信息栏**：当前关卡、碎片收集数、剩余时间、当前频率与波形图标

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 迷宫探索 | 迷宫视图 | Canvas绘制迷宫、玩家移动、声波传播动画、墙壁交互效果、碎片发光效果 |
| 迷宫探索 | 声波控制面板 | 频率调节、波形选择、发射/停止、实时波形图、频谱显示 |
| 迷宫探索 | HUD信息栏 | 关卡进度、碎片计数、计时器、频率/波形状态显示 |
| 迷宫探索 | 暂停覆盖层 | Esc键暂停、半透明蒙层、暂停文字 |

## 3. 核心流程

玩家进入迷宫 → 移动角色探索 → 发现机关门（频率图案提示）→ 调节频率发射声波 → 声波与墙壁交互（反射/折射/吸收）→ 解锁机关门 → 收集共鸣碎片 → 到达终点 → 进入下一关 → 三关全部通关显示总成绩

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深空蓝 (#0A0E27) 背景，营造神秘太空氛围
- **辅助色**：迷宫瓦片深色 (#1A1C3B)、石墙灰 (#4A4C6E)、水晶墙半透明蓝 (#3F51B5 80%)、金属墙渐变 (#B0BEC5→#78909C)
- **强调色**：碎片发光 (#FFD54F)、声波拖尾渐变光效
- **按钮风格**：圆角、半透明背景、悬停放大110% + 0.2s阴影过渡
- **字体**：无衬线字体，HUD白色12px
- **布局**：迷宫居中，HUD顶部，控制面板底部居中

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 迷宫探索 | 迷宫视图 | Canvas居中绘制，深空蓝背景，瓦片颜色根据声波频率动态变化 |
| 迷宫探索 | 声波控制面板 | 底部水平居中，半透明(#1E1E2E 85%)背景，圆角12px，内填充16px |
| 迷宫探索 | HUD | 顶部栏，左侧关卡/碎片/时间，右侧频率/波形图标 |
| 迷宫探索 | 暂停覆盖 | 半透明蒙层(#000000 60%)，白色"暂停"文字48px居中 |

### 4.3 响应式设计

- 桌面优先设计，移动端（<768px）自适应：
  - 迷宫瓦片尺寸缩小
  - HUD改为两行布局
  - 控制面板宽度自适应

### 4.4 游戏机制细节

- **声波传播**：从玩家位置沿直线传播，遇石墙反射、水晶墙折射、金属墙吸收
- **机关门**：频率图案提示，点击音叉图标+正确频率声波解锁，0.5s滑动打开+粒子特效
- **碎片**：死路末端隐藏，接近时周围墙体#FFD54F脉动(2s周期)，发射声波击中后1s回响+吸入动画+环绕光球
- **关卡**：3关(8x8→10x10→12x12)，碎片3→4→5个
- **计分**：通关时间+碎片60s奖励-错误频率扣5s(3次后)
- **暂停**：Esc键暂停/继续，半透明蒙层+暂停文字
