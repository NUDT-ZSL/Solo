## 1. 产品概述

「记忆琥珀」是一款数字时间胶囊全栈 Web 应用，用户可以创建封存文字与图片的数字胶囊，设定未来解锁时间，搭配系统自动生成的环境音与倒计时，营造沉浸式回忆体验。目标用户为希望留住珍贵时刻、为未来留言的个人用户。

- 核心价值：将转瞬即逝的记忆封存在「琥珀」中，在时光流转后以仪式感开启
- 差异化亮点：环境音生成 + 琥珀复古视觉 + 信封创建动画 + 光晕粒子开启特效

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 创建、浏览、开启胶囊 |

### 2.2 功能模块

1. **胶囊大厅页**: 胶囊列表展示、分类与状态筛选、创建入口
2. **胶囊详情/开启页**: 倒计时展示、环境音播放、开启动画

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 胶囊大厅 | 筛选栏 | 按类别（回忆、许愿、计划等）和状态（未开启、已开启）筛选胶囊 |
| 胶囊大厅 | 胶囊卡片列表 | 展示所有胶囊，每张卡片含标题、类别标签、倒计时、柔和光芒动画 |
| 胶囊大厅 | 创建胶囊按钮 | 点击弹出创建表单，含信封折叠展开动画 |
| 创建胶囊弹窗 | 信封动画 | 信封展开显示表单，填写文字/上传图片/选择类别/设定解锁时间 |
| 胶囊详情 | 倒计时 | 显示距离解锁的精确倒计时（天/时/分/秒） |
| 胶囊详情 | 环境音播放 | 自动播放系统生成的随机波形环境音（雨声/风声等） |
| 胶囊详情 | 开启动画 | 到达解锁时间后可开启，触发光晕粒子上升特效，展示内容 |

## 3. 核心流程

用户打开应用 → 浏览胶囊大厅 → 按类别/状态筛选 → 点击胶囊查看倒计时与环境音 → 或点击创建按钮 → 信封展开动画 → 填写内容与解锁时间 → 提交创建 → 胶囊出现在列表中 → 到达解锁时间后点击开启 → 光晕粒子上升特效 → 查看封存内容

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**: 琥珀色 (#D4A04A) 与乳白色 (#FFF8E7)
- **背景**: 暖黄 (#F5E6C8) 到深棕 (#3E2723) 渐变
- **按钮**: 琥珀色圆形按钮，带微光悬停效果（box-shadow glow + scale transform）
- **卡片**: 半透明磨砂圆角面板（backdrop-filter: blur），金色细边框（1px solid rgba(212,160,74,0.4)）
- **字体**: 标题用 "Playfair Display"（衬线体，复古优雅），正文用 "Noto Serif SC"（中文衬线体，温润质感）
- **图标**: lucide-react 图标库
- **动画**: CSS keyframes + requestAnimationFrame 确保 60fps

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 胶囊大厅 | 筛选栏 | 类别标签组（回忆/许愿/计划），状态切换（未开启/已开启），琥珀色激活态 |
| 胶囊大厅 | 胶囊卡片 | 磨砂圆角卡片，金色边框，悬停光芒扩散，倒计时数字，类别徽章 |
| 胶囊大厅 | 创建按钮 | 琥珀色圆形浮动按钮，右下角，微光脉冲动画 |
| 创建弹窗 | 信封动画 | 信封折叠→展开过渡，内含表单：标题输入、文字区域、图片上传、类别选择、时间选择器 |
| 胶囊详情 | 倒计时 | 大号数字显示，天/时/分/秒分段，琥珀色高亮 |
| 胶囊详情 | 环境音 | 底部音频控制条，波形可视化，播放/暂停按钮 |
| 胶囊详情 | 开启特效 | 背景光晕扩散，金色粒子从胶囊中心上升，内容淡入 |

### 4.3 响应式设计

- **桌面端优先**: 最大宽度 1200px 居中布局，卡片网格 3 列
- **平板端**: 768px 以下 2 列布局
- **手机端**: 480px 以下单列布局，卡片全宽，底部浮动创建按钮
- **触控优化**: 按钮最小 44px 触控区域，卡片间距增大至 16px

### 4.4 动效规范

- **信封折叠展开**: CSS transform (rotateX + translateY) + opacity，持续 600ms，ease-out
- **光芒扩散**: box-shadow 从 0 到 20px blur 琥珀色光晕，持续 300ms
- **光晕粒子上升**: Canvas 绘制金色圆点粒子，向上漂浮 + 淡出，持续 2000ms
- **倒计时**: 每秒更新，数字变化时轻微 scale 弹跳动画
- **环境音波形**: Canvas 绘制流动波形，与音频数据同步
- **所有动画**: 使用 will-change + transform 确保合成层优化，维持 60fps
