## 1. 产品概述

「图腾攀岩者」是一款基于浏览器Canvas的2D攀岩休闲游戏，玩家控制一个发光小人沿随机生成的图腾柱向上攀爬，通过点击匹配正确颜色的手抓点来攀升得分，错误点击则触发坠落。游戏以逐渐递增的旋转和缩放难度为核心机制，追求60FPS流畅体验。

- 目标用户：休闲游戏玩家，追求快节奏反应类体验
- 核心价值：简单上手、深度挑战、视觉冲击力强的攀岩体验

## 2. 核心功能

### 2.1 功能模块

1. **游戏主界面**：全屏Canvas游戏画面、星空背景、旋转图腾柱、攀爬小人、顶部状态栏
2. **游戏结束界面**：半透明遮罩、Game Over文字、最终得分、重新开始按钮

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主界面 | 图腾柱渲染 | 随机生成5-8环层图腾柱，每层2-4个彩色手抓点，灰褐色柱体、金色分隔线 |
| 游戏主界面 | 玩家控制 | 鼠标点击/触摸彩色圆点，匹配目标颜色则上升+10分，不匹配则坠落；左右滑动微调位置 |
| 游戏主界面 | 难度递增 | 图腾柱绕Y轴旋转（0.5度/帧起，每100分+0.1度）、柱体缩放（1.0→0.7）、每10层新增环层 |
| 游戏主界面 | 得分与连击 | 顶部显示得分/层数/目标颜色块；连续3次正确触发连击（得分翻倍+脉冲光环0.5秒） |
| 游戏主界面 | 星空背景 | 深绿到深蓝渐变背景，随机闪烁白色星点 |
| 游戏结束界面 | 结束遮罩 | 半透明黑色遮罩，Game Over文字(48px白色)，最终得分(32px金色)，重新开始按钮 |

## 3. 核心流程

玩家开始游戏 → 图腾柱随机生成 → 屏幕底部显示目标颜色 → 玩家点击匹配颜色的手抓点 → 正确则上升加分 / 错误则坠落 → 连续3次正确触发连击 → 攀爬越高难度越大（旋转加速+缩放+新层） → 坠落到底部 → Game Over → 点击重新开始

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深绿(#0b2d1e)到深蓝(#0a1a30)渐变背景，营造神秘丛林夜空氛围
- 强调色：金色(#c8a859)用于分隔线和连击效果，亮白(#ffffff)用于玩家小人
- 手抓点颜色：红、橙、黄、绿、蓝、紫六色纯色填充+半透明发光光晕
- 字体：白色无衬线字体，顶部状态栏信息清晰可读
- 布局：全屏Canvas，顶部半透明黑色条带显示游戏信息
- 动画风格：发光效果、粒子拖尾、脉冲光环、缩放动画，强调反馈感

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主界面 | 背景层 | 深绿到深蓝垂直渐变、随机闪烁白色星点 |
| 游戏主界面 | 图腾柱 | 灰褐色(#6b5b4f)柱体、金色(#c8a859)分隔线、彩色圆点+光晕 |
| 游戏主界面 | 玩家小人 | 亮白(#ffffff)发光圆形(12px)、蓝色光晕(#4488ff,0.4透明度) |
| 游戏主界面 | 坠落效果 | 红色光点+粒子尾迹(1秒消失) |
| 游戏主界面 | 状态栏 | 半透明黑色条带(0.5透明度,50px高)、得分/层数/目标颜色块(30x30圆角矩形) |
| 游戏主界面 | 连击效果 | 屏幕底部金色渐变光晕(0.8秒)、小人脉冲光环(0.5秒) |
| 游戏结束界面 | 遮罩 | 半透明黑色(#000000,0.7透明度) |
| 游戏结束界面 | 文字 | Game Over(48px白色)、最终得分(32px金色#ffcc00) |
| 游戏结束界面 | 按钮 | 圆角矩形(#3a6b35,悬停#4a8b45)、白色文字 |

### 4.3 响应式

- 全屏Canvas自适应100%宽高
- 窗口resize时重新适配Canvas尺寸
- 支持鼠标点击和触摸操作

### 4.4 交互反馈

- 正确点击：圆点0.1秒内放大1.2倍再恢复
- 错误点击：小人0.2秒内上下偏移5px两次抖动
- 连击：脉冲光环+底部金色光晕
- 坠落：红色光点+粒子尾迹
