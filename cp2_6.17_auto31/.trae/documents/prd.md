## 1. 产品概述

核磁共振成像（MRI）原理教学演示工具，通过3D交互式动画直观展示自旋回波、梯度编码和k空间填充等抽象概念，帮助医学影像专业学生理解MRI成像的物理原理。

- 核心目标：将复杂的MRI物理原理转化为可视化、可交互的教学工具
- 目标用户：医学影像专业学生、MRI技术初学者
- 产品价值：降低学习门槛，通过参数调节实时观察成像变化，加深理解

## 2. 核心功能

### 2.1 功能模块

1. **3D质子场景**：半透明椭球体代表扫描物体，内部散布100个彩色自旋质子，支持360°旋转观察
2. **脉冲序列控制**：TR、TE、翻转角三个参数滑块，实时调节并驱动质子相位动画
3. **k空间可视化**：实时显示k空间采样轨迹，展示梯度编码与k空间填充的关系
4. **图像重建**：采集完成后通过逆FFT生成重建图像，参数变化影响图像质量
5. **视角控制**：支持鼠标拖拽旋转、一键重置视角

### 2.2 功能详情

| 模块名称 | 功能描述 |
|---------|----------|
| 3D场景 | 椭球体（1.2×0.8×0.8，淡粉色半透明），100个彩色质子点，OrbitControls交互，阻尼0.1 |
| 控制面板 | TR滑块（200-1000ms）、TE滑块（10-100ms）、翻转角滑块（30-180°），数值实时显示 |
| 质子动画 | 质子围绕z轴旋转，频率与位置梯度线性相关，相位随参数变化 |
| k空间轨迹 | 320×240 Canvas，笛卡尔网格采样，颜色从#3498db到#e74c3c渐变 |
| 图像重建 | 128×128灰度图，320×320显示区域，逆FFT重建，TR/TE影响对比度和SNR |
| 采集按钮 | 60×30px圆角矩形，#2ecc71，悬浮#27ae60，白色文字 |

## 3. 核心流程

用户打开应用 → 观察默认参数下的3D质子分布 → 调节TR/TE/翻转角滑块 → 实时观察质子相位变化和k空间轨迹 → 点击采集按钮 → 查看重建图像 → 继续调整参数对比不同成像效果

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：暗色科技风，模拟医学影像设备界面
- **主背景**：#1a1a2e（深蓝黑）
- **面板背景**：#16213e（深蓝）
- **边线**：1px solid #0f3460
- **文字**：#eaeaea（浅灰白）
- **高亮色**：蓝绿色系（#2ecc71、#3498db、#4ecdc4）
- **过渡效果**：0.25s ease
- **字体**：现代无衬线字体，清晰可读

### 4.2 页面布局

| 区域 | 位置 | 内容 |
|------|------|------|
| 左侧控制面板 | 左侧固定宽度 | 参数滑块、采集按钮、数值显示 |
| 中央3D场景 | 右侧主区域 | Three.js渲染的椭球体和质子系统 |
| 右下角k空间 | 3D场景右下角浮动 | k空间轨迹Canvas |
| 重建图像 | 控制面板下方或右侧 | 重建图像Canvas |

### 4.3 响应式

- 桌面端优先，Canvas自适应窗口大小变化，保持宽高比
- 所有交互元素支持悬浮反馈
- 过渡动画流畅，无卡顿

### 4.4 3D场景设计

- **环境**：深色背景，模拟MRI扫描室氛围
- **光照**：环境光 + 方向光，保证质子点和椭球体清晰可见
- **相机**：PerspectiveCamera，初始视角略微俯视
- **交互**：OrbitControls，支持旋转、缩放，开启阻尼效果
- **材质**：椭球体使用半透明MeshPhysicalMaterial，质子使用PointsMaterial
