## 1. 产品概述

「元素共鸣」是一款2D像素风解谜游戏，玩家控制小魔法师在网格地图上移动，通过按特定顺序踩踏火、水、土、风四种元素方块来触发共鸣，激活所有方块后打开传送门过关。

- 目标用户：喜欢休闲解谜和像素风游戏的玩家
- 核心价值：融合元素排列解谜与复古像素美学，提供沉浸式的策略闯关体验

## 2. 核心功能

### 2.2 功能模块

1. **游戏主界面**：Canvas画布渲染游戏画面，UI叠加层显示关卡信息、星星评价、暂停按钮和操作提示
2. **关卡系统**：多关卡地图布局与元素方块顺序配置，关卡递进难度设计
3. **玩家控制**：WASD键控制魔法师在网格上移动，踩踏元素方块触发共鸣
4. **效果系统**：像素粒子、光晕扩散、音效生成（Web Audio API）
5. **评价系统**：每关三星评价（时间、步数、完美顺序）

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主界面 | Canvas渲染区 | 渲染网格地图、元素方块、魔法师角色、传送门、背景渐变 |
| 游戏主界面 | HUD信息层 | 左上角关卡编号与星星，右上角毛玻璃暂停按钮，底部操作提示 |
| 游戏主界面 | 暂停弹窗 | 暂停游戏，显示重开和继续选项 |
| 游戏主界面 | 过关动画 | 传送门闪烁光效，星星评价展示 |

## 3. 核心流程

玩家进入关卡 → 观察地图与元素方块分布 → 使用WASD移动魔法师 → 按正确顺序踩踏元素方块 → 方块高亮+音效+粒子效果 → 全部方块正确激活后传送门开启 → 进入传送门过关 → 显示星级评价 → 下一关

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝(#0a0a2e)到紫黑(#1a0a2e)渐变背景
- **砖块纹理**：暗色(#2a2a4a)砖块格线，略带纹理感
- **元素配色**：火红(#ff4444)、水蓝(#4488ff)、土棕(#aa7744)、风绿(#44ff88)
- **像素风格**：8x8像素图标，无抗锯齿渲染，复古CRT感
- **字体**：像素风格字体(Press Start 2P)
- **按钮风格**：像素边框按钮，毛玻璃暂停按钮
- **布局**：居中Canvas画布，HUD叠加在画布上方

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主界面 | Canvas渲染区 | 深蓝紫黑渐变背景，暗色砖块网格，8x8像素元素图标，像素魔法师角色 |
| 游戏主界面 | HUD信息层 | 左上角关卡文字+星星图标，右上角半透明暂停按钮，底部半透明操作提示 |
| 游戏主界面 | 效果层 | 元素踩踏光晕扩散，像素粒子上升，传送门闪烁光效 |

### 4.3 响应式

- 桌面优先，Canvas居中显示
- 固定逻辑分辨率，按窗口等比缩放
- 键盘操作为主(WASD + R)

## 4.4 3D场景指引

不适用，本项目为2D Canvas游戏。
