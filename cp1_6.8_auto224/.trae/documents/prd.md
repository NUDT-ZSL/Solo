## 1. 产品概述

「灵感便签」是一个在线创意收集与可视化平台，帮助用户快速记录零碎灵感并通过动态「灵感星图」直观呈现创意之间的关联。目标用户为创意工作者、设计师、开发者及任何需要捕捉碎片化灵感的人群。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 创建、浏览、搜索、编辑、删除便签，查看星图 |

### 2.2 功能模块
1. **首页（瀑布流）**：搜索栏、便签卡片瀑布流、新增便签入口
2. **灵感星图页**：Canvas 动态星图可视化、鼠标悬停预览、点击进入详情
3. **便签详情页**：完整内容展示、编辑、删除

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 搜索栏 | 支持按标签和关键词过滤便签 |
| 首页 | 便签卡片瀑布流 | 毛玻璃质感卡片，带标签色条和优先级星标，微上浮阴影动效 |
| 首页 | 新增便签按钮 | 打开新增便签弹窗，填写文字、标签、优先级 |
| 灵感星图页 | Canvas 星图 | 便签变为发光星点，大小代表优先级，颜色按标签分组，缓慢旋转并相互连接形成网状结构 |
| 灵感星图页 | 悬停预览 | 鼠标悬停星点时显示便签摘要 |
| 灵感星图页 | 点击详情 | 点击星点跳转至便签详情页 |
| 便签详情页 | 内容展示 | 显示完整便签内容、标签、优先级、创建时间 |
| 便签详情页 | 编辑/删除 | 支持编辑便签内容和删除便签 |

## 3. 核心流程

用户打开应用 → 首页展示瀑布流便签卡片 → 可通过搜索栏按标签/关键词过滤 → 点击「灵感星图」切换到星图可视化视图 → 鼠标悬停星点预览内容 → 点击星点进入详情页 → 可编辑或删除便签 → 新增便签通过首页弹窗完成

## 4. 用户界面设计

### 4.1 设计风格
- **主题**：深空极简风
- **主色调**：深蓝灰渐变背景（#0a0e1a → #1a1f3a）
- **强调色**：星点颜色按标签分组 — 技术-蓝(#3b82f6)、设计-紫(#a855f7)、生活-绿(#22c55e)、其他-橙(#f97316)
- **卡片风格**：毛玻璃质感（backdrop-filter: blur），柔和圆角(12px)，细微发光边缘
- **字体**：标题用 Outfit（display），正文用 Noto Sans SC（中文友好）
- **布局**：顶部导航，内容区居中，卡片瀑布流
- **图标**：Lucide React 图标库

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 搜索栏 | 毛玻璃输入框，聚焦时发光边框，标签筛选按钮组 |
| 首页 | 瀑布流卡片 | 毛玻璃卡片，顶部标签色条，左上角优先级星标，hover微上浮+阴影加深，缓动淡入 |
| 首页 | 新增按钮 | 右下角浮动按钮，发光效果，点击弹出毛玻璃弹窗 |
| 灵感星图页 | Canvas 星图 | 深蓝渐变背景，星点大小1-5级映射，颜色按标签，缓慢旋转动画，同标签星点间有半透明连线 |
| 灵感星图页 | 悬停提示 | 毛玻璃浮层，显示便签标题和标签 |
| 详情页 | 内容区 | 毛玻璃面板，完整便签信息，编辑/删除按钮 |

### 4.3 响应式适配
- 桌面端（≥1024px）：双列瀑布流，星图全屏Canvas
- 平板端（768px-1023px）：单列瀑布流，星图全屏Canvas
- 帧率目标：60fps，Canvas 动画使用 requestAnimationFrame

### 4.4 动效设计
- 页面切换：缓动淡入动画（fade-in, 300ms ease-out）
- 卡片入场：交错延迟淡入上浮（staggered, 50ms interval）
- 星图旋转：缓慢持续旋转，同标签星点连线随旋转波动
- 背景：缓慢飘浮的细小星点粒子
- 交互反馈：hover 上浮+阴影，按钮点击缩放
