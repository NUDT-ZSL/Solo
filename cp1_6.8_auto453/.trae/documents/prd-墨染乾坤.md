## 1. 产品概述

「墨染乾坤」是一款2D水墨风格动作跑酷游戏。玩家操控水墨侠客在动态展开的山水画卷中奔跑，通过跳跃和滑铲躲避墨迹障碍物，收集金色诗意碎片解锁完整诗句。
- 目标用户：喜爱中国传统文化、水墨美学和休闲动作游戏的玩家
- 核心价值：将中国传统水墨画的视觉美学与流畅的跑酷玩法结合，创造沉浸式的文化体验

## 2. 核心功能

### 2.2 功能模块

1. **游戏画面**：卷轴山水背景、水墨侠客、墨迹障碍物、金色碎片、粒子特效
2. **操作交互**：点击/按键跳跃和滑铲，流畅响应
3. **HUD界面**：卷轴进度条、碎片计数、生命值显示、诗句浮现
4. **游戏流程**：开始→奔跑→收集/躲避→关卡结束→诗句解锁

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏画面 | 卷轴山水背景 | 米白浅灰卷轴底色，山峦、瀑布、松树用淡墨勾勒，带缓慢晕染动画，随奔跑进度不断展开 |
| 游戏画面 | 水墨侠客 | 简笔水墨人形角色，冲刺拖出半透明墨痕，跳跃和滑铲动画 |
| 游戏画面 | 墨迹障碍物 | 墨团、毛笔字、印章三种类型，泼墨扩散动画，被躲避时碎成细墨点消散 |
| 游戏画面 | 金色诗意碎片 | 散落金色碎片，收集时金色光晕和淡金光粒子飞散 |
| 游戏画面 | 诗句解锁 | 每收集一定数量碎片，画面短暂暂停，诗句从右向左浮现，带墨迹干涸渐变效果 |
| HUD | 卷轴进度条 | 左上角，水墨风格填充，显示当前关卡进度 |
| HUD | 碎片计数 | 右上角，显示收集的碎片数量 |
| HUD | 生命值 | 右上角，毛笔笔触墨滴表示，三滴代表三条命 |
| HUD | 底部区域 | 简洁无多余UI |

## 3. 核心流程

玩家进入游戏后，水墨侠客自动向右奔跑。山水画卷持续展开，障碍物从画中飞出。玩家通过点击屏幕或按键控制侠客跳跃（避开地面和空中障碍）或滑铲（穿过空中障碍）。沿途收集金色碎片，每收集5个碎片在关卡结束时解锁一句诗句。三条生命耗尽则游戏结束，可重新开始。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：米白(#F5F0E8)到浅灰(#D8D0C4)的卷轴色
- 辅助色：淡墨黑(#2C2C2C)、浓墨(#0A0A0A)、金色(#C9A84C)、朱红(#B22222)
- 按钮风格：水墨晕染风格，无明确边框
- 字体：楷体/宋体风格中文书法字体
- 布局风格：全屏Canvas游戏画面，HUD覆盖在画面上方
- 图标风格：水墨笔触风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏画面 | 卷轴背景 | 米白渐变底色，淡墨山水轮廓，缓慢晕染动画，视差滚动 |
| 游戏画面 | 侠客 | 简笔水墨人形，跑步帧动画，跳跃弧线，滑铲低位，墨痕拖尾 |
| 游戏画面 | 障碍物 | 墨团（圆形泼墨）、毛笔字（书法字体）、印章（朱红方印），泼墨扩散动画 |
| 游戏画面 | 碎片 | 金色菱形，光晕环绕，收集时金色粒子爆发 |
| 游戏画面 | 诗句 | 书法字体从右向左浮现，墨迹干涸渐变，画面暂停效果 |
| HUD | 进度条 | 左上角，水墨风格填充条，浓墨渐变 |
| HUD | 碎片计数 | 右上角，金色数字 + 金色菱形图标 |
| HUD | 生命值 | 右上角，墨滴图标（1-3滴），毛笔笔触风格 |

### 4.3 响应式适配

- 桌面端：键盘操作（空格跳跃，下箭头滑铲），鼠标点击
- 移动端：触摸操作（点击上半区域跳跃，点击下半区域滑铲）
- Canvas自适应屏幕尺寸，保持16:9或类似宽高比

### 4.4 性能要求

- 帧率稳定60fps
- Canvas 2D渲染，避免频繁GC
- 粒子系统使用对象池
- 背景元素按需生成和回收
