## 1. 产品概述

智能家庭植物养护记录与社交分享平台，帮助植物爱好者像记录宠物日常一样追踪每盆植物的养护历史，并与同城花友交换扦插枝条、交流养护经验。

- **目标用户**：家庭植物爱好者、园艺新手与资深花友
- **核心价值**：植物养护记录数字化 + 同城植物交换社区
- **产品定位**：治愈系、轻量化的植物养护社交应用

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 应用内使用（单用户演示） | 添加植物档案、记录养护历史、发布交换信息、点赞互动 |

### 2.2 功能模块

1. **首页**：植物概览、动态消息流、推荐交换
2. **植物详情页**：养护时间线、操作记录、点赞互动
3. **交换广场**：交换信息发布、图片展示、私信弹窗、位置连线

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 植物概览（左栏） | 展示用户所有植物卡片，支持懒加载，点击进入详情 |
| 首页 | 动态消息流（中栏） | 展示养护动态、用户互动信息 |
| 首页 | 推荐交换（右栏） | 展示热门交换信息，点击进入交换广场 |
| 植物详情页 | 植物信息头部 | 展示植物大图、名称、健康状态、基本信息 |
| 植物详情页 | 养护时间线 | 按时间倒序展示浇水、施肥、换盆等操作记录，带动画效果 |
| 植物详情页 | 记录点赞 | 每条养护记录支持"觉得有用"点赞 |
| 交换广场 | 交换列表 | 展示所有交换信息，带图片网格 |
| 交换广场 | 发布交换 | 上传1-3张图片，填写交换信息 |
| 交换广场 | 私信弹窗 | 点击"想要"弹出私信窗口，显示双方位置连线 |

## 3. 核心流程

### 3.1 植物养护记录流程

用户添加植物档案 → 日常记录养护操作（浇水/施肥/换盆） → 查看时间线回顾历史 → 为有用的记录点赞

```mermaid
flowchart LR
    A["添加植物档案"] --> B["记录养护操作"]
    B --> C["生成时间线记录"]
    C --> D["查看养护历史"]
    D --> E["为有用记录点赞"]
```

### 3.2 植物交换流程

用户发布交换信息 → 其他用户浏览 → 点击"想要" → 发送私信 + 查看位置 → 达成交换

```mermaid
flowchart LR
    A["发布交换信息"] --> B["上传图片/填写描述"]
    B --> C["展示在交换广场"]
    C --> D["其他用户浏览"]
    D --> E["点击想要"]
    E --> F["发送私信+位置"]
    F --> G["达成交换"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：莫兰迪色系 — 主色 `#a5d6a7`、辅色 `#81c784`、强调色 `#ef9a9a`、背景 `#f1f8f5`
- **卡片风格**：轻微圆角（16px）、柔和阴影（0 4px 12px rgba(0,0,0,0.08)）、渐变背景
- **按钮风格**：圆角按钮、点击缩放反馈（scale 0.95，0.2s）
- **字体**：使用系统无衬线字体，搭配柔和的字重层次
- **整体氛围**：治愈、清新、自然、舒缓

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 植物卡片 | 竖长卡片（200×300px）、渐变背景、底部状态边框、圆形头像、迷你状态条 |
| 首页 | 三栏布局 | 左栏植物概览、中栏消息流、右栏推荐交换 |
| 植物详情页 | 时间线 | 左侧图标分类、中间描述、右侧时间备注、滑入动画 |
| 交换广场 | 图片网格 | 120×120px方格、悬停放大1.05倍、阴影加深 |
| 交换广场 | 位置地图 | 两个位置标签连线、简约地图风格 |

### 4.3 响应式设计

- **桌面端**（>980px）：三栏布局
- **平板端**（768px-980px）：两栏布局（植物概览 + 消息流合并，交换独立或底部）
- **移动端**（<768px）：单栏布局，底部Tab切换
- 所有交互按钮均适配触控操作

### 4.4 动效设计

- 植物卡片：悬停时轻微上浮、底部边框过渡动画（0.4s ease）
- 时间线项目：从左侧滑入（0.35s ease-out）、错落延迟
- 图片网格：悬停放大1.05倍 + 阴影加深
- 按钮点击：scale(0.95) 缩放反馈（0.2s）
- 页面切换：淡入过渡
- 懒加载：滚动时渐入效果

## 5. 性能要求

- 首屏加载时间 ≤ 2秒
- 卡片懒加载（滚动进入视口时渲染）
- 图片优化与懒加载
- 流畅的滚动体验（60fps）
