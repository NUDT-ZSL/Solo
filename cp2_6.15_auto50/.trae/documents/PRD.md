## 1. 产品概述
在线协作歌词排版与动态字幕生成器，为音乐爱好者和视频创作者提供歌词时间轴编辑、动态字幕预览和多格式导出功能。
- 核心价值：简化字幕制作流程，提供可视化拖拽编辑和实时预览体验
- 目标用户：音乐爱好者、视频创作者、字幕制作人员

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，直接使用 | 上传歌词、编辑时间轴、预览效果、导出字幕 |

### 2.2 功能模块
1. **上传区域**：歌词文件上传、拖拽上传、文件解析
2. **时间轴编辑模块**：时间条展示、拖拽调整、样式管理、选中状态
3. **字幕预览模块**：实时预览、逐字动画、播放控制、字体大小调节
4. **导出功能**：SRT格式导出、ASS格式导出

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 上传区域 | 支持TXT文件拖拽/点击上传，解析歌词文本为逐行条目 |
| 主页面 | 时间轴编辑区 | 以可拖拽时间条展示每句歌词，支持位置调整和选中高亮 |
| 主页面 | 字幕预览区 | 实时显示动态字幕效果，支持打字机动画和滚动消失效果 |
| 主页面 | 控制面板 | 播放/暂停、字体大小滑块调节、导出按钮 |

## 3. 核心流程
用户上传TXT歌词文件 → 系统解析为逐句歌词 → 在时间轴上显示可拖拽时间条 → 用户拖拽调整时间轴位置 → 右侧实时预览字幕动画 → 满意后导出SRT/ASS格式文件

```mermaid
flowchart LR
    A["上传TXT歌词"] --> B["解析歌词文本"]
    B --> C["生成时间条列表"]
    C --> D["拖拽调整时间轴"]
    D --> E["事件总线通知更新"]
    E --> F["预览区刷新动画"]
    F --> G["导出SRT/ASS文件"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深色主题，主背景#121212，卡片背景#1e1e1e
- **配色方案**：柔和紫绿配色（紫色#b388ff/#ede7f6，绿色#4db6ac/#81c784）
- **强调色**：橙色#ffb74d（选中高亮），#ffcc80（字幕文字）
- **按钮风格**：圆角12px，过渡动画0.2秒ease
- **字体**：系统无衬线体
- **圆角规范**：上传区16px、按钮12px、时间条6px

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 上传区域 | 700px×160px，圆角16px，虚线边框2px #b388ff，背景#f3e5f5，拖拽时实线边框+背景#ede7f6 |
| 主页面 | 时间轴编辑区 | 左侧宽55%，背景#263238，时间条高32px圆角6px，奇偶色#4db6ac/#81c784，选中#ffb74d边框，悬停上移2px 0.15s |
| 主页面 | 字幕预览区 | 右侧宽45%，背景#1e1e1e，打字机效果0.06s/字，颜色#ffcc80带阴影，已播半透明0.3上滚30px 0.4s ease-out |
| 主页面 | 控制面板 | 字体滑块12-48px步长2px，导出按钮组 |

### 4.3 响应式
桌面端优先布局，时间轴与预览区左右分栏（55%/45%），上传区顶部居中700px固定宽度。

### 4.4 性能要求
- 拖拽响应延迟 ≤ 16ms
- 预览刷新帧率稳定 60fps
- 使用requestAnimationFrame优化动画渲染
