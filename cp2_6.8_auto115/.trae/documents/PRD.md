## 1. 产品概述
基于间隔重复算法的在线知识卡片闪卡复习系统，面向语言学习者和考试复习者，帮助用户创建个性化卡片库并高效复习。
- 核心目标：通过科学的间隔重复算法提升记忆效率，提供流畅的3D翻转交互体验
- 目标用户：语言学习者、考试备考人群、需要记忆大量知识点的用户
- 产品价值：降低记忆遗忘曲线带来的损失，让复习更有针对性和效率

## 2. 核心功能

### 2.1 功能模块
1. **首页**：卡片组网格展示、导航栏、创建入口
2. **复习页**：卡片3D翻转展示、评分系统、自动调度下一张
3. **创建页**：卡片组创建、卡片添加表单（支持多行文本）

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页 | 导航栏 | 深蓝底色品牌展示，创建卡片组按钮 |
| 首页 | 卡片组网格 | 响应式网格展示卡片组，显示标题、卡片数量、下次复习时间，悬停上浮动画 |
| 复习页 | 卡片展示区 | 3D水平翻转动画展示卡片正反面，0.4秒缓入缓出 |
| 复习页 | 评分操作区 | 困难(红)/良好(黄)/简单(绿)三色评分按钮，点击后动态调整间隔 |
| 创建页 | 卡片组信息 | 卡片组标题输入 |
| 创建页 | 卡片表单 | 正面问题和背面答案多行输入，支持实时添加 |
| 创建页 | 卡片列表 | 已添加卡片的预览列表 |

## 3. 核心流程
用户访问首页浏览所有卡片组 → 选择卡片组进入复习模式 → 思考问题后翻转查看答案 → 根据掌握程度选择评分 → 系统根据间隔重复算法调整下次复习时间 → 自动显示下一张卡片，直至复习完成。

```mermaid

    A["用户访问首页"] --> B["浏览卡片组网格"]
    B --> C["点击卡片组"]
    C --> D["进入复习模式"]
    D --> E["显示卡片正面问题"]
    E --> F["用户思考后点击翻转"]
    F --> G["3D动画展示背面答案"]
    G --> H["用户选择评分(困难/良好/简单)"]
    H --> I["系统计算下次复习时间"]
    I --> J["自动切换下一张卡片"]
    J --> E
    J --> K["复习完成返回首页"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深蓝 #1A365D（导航栏）
- 卡片背景：纯白 #FFFFFF，圆角 12px
- 阴影色：淡蓝 #63B3ED（悬停阴影）
- 评分按钮：红色（困难）、黄色（良好）、绿色（简单）
- 翻转动画：0.4秒，缓入缓出（ease-in-out）
- 按钮交互：点击时0.1秒缩放反馈
- 布局：左对齐，响应式网格（手机单列 → 平板双列 → 桌面四列）
- 性能目标：翻转动画和评分切换响应时间 ≤ 100ms

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 首页 | 导航栏 | 深蓝背景、品牌名称、创建按钮、白色文字 |
| 首页 | 卡片组网格 | 白色圆角卡片、悬停上浮4px+淡蓝阴影、标题/数量/复习时间信息展示 |
| 复习页 | 卡片展示 | 透视3D翻转效果、正反面清晰区分、居中展示 |
| 复习页 | 评分按钮 | 三色横向排列、圆角设计、点击缩放反馈 |
| 创建页 | 表单区域 | 多行文本框、提交按钮、已添加卡片列表 |

### 4.3 响应式
桌面端优先设计，移动端自适应。卡片组网格在不同屏幕尺寸下自动调整列数：手机(≤640px)单列、平板(641-1024px)双列、桌面(1025-1440px)三列、大屏(>1440px)四列。
