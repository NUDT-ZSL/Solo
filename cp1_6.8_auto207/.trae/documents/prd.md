## 1. 产品概述

「词云诗镜」是一款基于中国古典诗词的交互式词云生成器，将文本分析与可视化艺术融为一体。用户输入文本或选择预设诗词后，系统自动提取关键词，以螺旋路径动态生成词云，配合毛玻璃信息卡片和粒子爆散动画，打造沉浸式诗画体验。

- 目标用户：诗词爱好者、文学教育者、数据可视化爱好者
- 核心价值：将古典文学之美与交互技术结合，提供兼具审美与信息量的诗词探索工具

## 2. 核心功能

### 2.1 功能模块

1. **主页面**：文本输入区、预设诗词选择器、词云展示区、诗句展示区
2. **词云交互**：螺旋布局词云、悬停信息卡片、点击粒子动画与诗句跳转

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 文本输入区 | 支持自由输入文本或从预设诗词库选择 |
| 主页面 | 预设诗词库 | 提供6-8首经典诗词，点击即加载 |
| 主页面 | 词云展示区 | 按螺旋路径分布的动态词云，鼠标悬停放大词条并显示毛玻璃卡片（词频、情感标签、诗句出处） |
| 主页面 | 诗句展示区 | 点击词条后跳转展示关联诗句，高亮对应词 |
| 主页面 | 背景粒子 | 缓慢飘浮的细小墨点粒子营造诗画氛围 |

## 3. 核心流程

1. 用户打开页面，看到诗画淡彩风格的界面和飘浮墨点背景
2. 用户在文本输入区输入文字，或从预设诗词库选择一首诗词
3. 系统自动分析文本，提取关键词、计算词频和情感标签
4. 词条沿螺旋路径动态布局，形成词云
5. 用户鼠标悬停词条，词条放大并弹出毛玻璃信息卡片，展示词频、情感标签、诗句出处
6. 用户点击词条，触发粒子爆散动画，页面滚动到诗句展示区，高亮显示包含该词的诗句
7. 用户可切换不同诗词，词云以缓动动画过渡

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米白底色（#FAF7F2），暗金点缀（#B8860B），靛蓝点缀（#2C3E7B）
- **辅助色**：淡金色阴影（rgba(184,134,11,0.15)），墨色文字（#2C2C2C）
- **卡片风格**：毛玻璃效果（backdrop-filter: blur(12px)），淡金色阴影，圆角12px
- **字体**：标题使用「思源宋体」风格衬线体，正文使用清晰无衬线体
- **布局**：上方文本输入 + 诗词选择，中部大面积词云，底部诗句展示
- **动画**：页面切换缓动淡入，词条悬停弹性放大，点击粒子爆散，背景墨点缓慢飘浮

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 主页面 | 文本输入区 | 米白圆角卡片，暗金边框，内含 textarea 和诗词选择按钮组 |
| 主页面 | 词云展示区 | SVG 画布，词条按螺旋分布，大小映射词频，颜色区分情感 |
| 主页面 | 悬停信息卡片 | 毛玻璃浮层，显示词频数字、情感标签徽章、诗句出处文字 |
| 主页面 | 诗句展示区 | 卡片列表，含高亮关键词的诗句行，出处标注 |
| 主页面 | 背景粒子 | Canvas 层，半透明墨点粒子缓慢漂浮 |

### 4.3 响应式

- 桌面端（≥1024px）：词云居中大面积展示，侧边诗句展示区
- 平板端（768px-1023px）：词云上方，诗句下方垂直排列
- 触控优化：词条点击区域适当扩大，信息卡片点击触发而非悬停

### 4.4 性能目标

- 词云渲染帧率稳定 60fps
- 粒子系统不超过 80 个粒子
- 词云词条数量控制在 30-50 个
- SVG 动画使用 CSS transform 避免重排
