# BubblePoker 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 项目背景
BubblePoker 是一款在线多人扑克牌心理博弈应用，让2-4名玩家在同一房间内进行简化的心理博弈对战（类似德州扑克简化版）。

### 1.2 产品目标
- 提供流畅的实时多人扑克游戏体验
- 精美的视觉设计和流畅的动画效果
- 完整的游戏流程：登录 → 大厅 → 房间 → 游戏 → 结算
- 财务变化曲线可视化

### 1.3 目标用户
- 喜欢扑克牌游戏的休闲玩家
- 2-4人组队的朋友聚会场景

---

## 2. 功能需求

### 2.1 用户系统
- **昵称登录**：用户输入昵称即可进入大厅，无需注册
- **在线人数显示**：顶栏实时显示当前在线人数

### 2.2 大厅系统
- **房间列表**：展示所有可加入的房间，显示房间号、人数、状态
- **创建房间**：一键创建新房间，自动生成房间ID
- **加入房间**：点击房间卡片即可加入
- **房间卡片交互**：悬停时抬高6px并加深阴影

### 2.3 游戏系统
- **座位系统**：最多4个座位，弧形排列
- **发牌系统**：自动洗牌、发牌，带弧形飞行动画
- **手牌管理**：自己的牌可点击翻转查看，对手牌背面显示
- **公共牌系统**：最多5张公共牌，逐步展示
- **下注系统**：加注、跟注、弃牌、全下四种操作
- **回合管理**：自动轮流行动，当前玩家高亮提示
- **胜负判定**：自动比较牌型，判定获胜者
- **胜利动画**：获胜者金色光环旋转，胜利提示弹出

### 2.4 数据可视化
- **财务曲线**：游戏结束后展示每位玩家筹码量随时间变化的折线图
- **历史查看**：随时点击"历史"按钮查看筹码变化曲线

### 2.5 实时通信
- WebSocket 实时同步游戏状态
- 房间状态实时更新
- 玩家行动实时广播

## 3. 界面需求

### 3.1 整体风格
- **主题色调**：深绿色绒布纹理背景 (#1a3c2a)
- **牌桌设计**：渐变深绿色 (#1a3c2a 到 #0f2b1a)，中央略亮 (#2d5a3a)
- **圆角设计**：牌桌左上角48px，其他角24px

### 3.2 大厅页面
- **背景**：深绿色绒布纹理 (#1a3c2a)
- **顶栏**：应用名 + 在线人数
- **房间卡片**：圆角12px，白色背景 (#f0f0f0)，悬停动效
- **创建按钮**：深绿色 (#2d6a4f)，圆角20px，悬停变亮 (#40916c)

### 3.3 游戏页面
- **玩家座位卡片**：宽180px高260px，圆角16px，背景 (#2d3748)，毛玻璃效果
- **当前行动提示**：金色光芒 (#fbbf24) 闪烁动画
- **底池显示**：白色字体24px，带轻微阴影
- **公共牌区**：最多5张牌，牌间距8px

### 3.4 ChipBar 组件
- **位置**：牌桌底部中央
- **背景**：半透明黑色 (#00000080)，圆角20px，内边距20px
- **按钮**：
  - 加注：绿色 (#38a169)
  - 跟注：蓝色 (#3182ce)
  - 弃牌：红色 (#e53e3e)
  - 全下：橙色 (#dd6b20)
- **按钮样式**：圆角12px，高度48px，宽度120px，间距8px

### 3.5 ChartModal 组件
- **背景**：浅灰色 (#f7fafc)
- **折线图**：
  - 玩家1：红色 (#e53e3e)
  - 玩家2：蓝色 (#3182ce)
  - 玩家3：绿色 (#38a169)
  - 玩家4：橙色 (#dd6b20)
  - 线宽2px，圆点半径4px
- **关闭按钮**：左上角叉号，圆角50%

### 3.6 动画效果
- **发牌动画**：牌从中心弧形飞向座位，0.4秒，ease-out
- **卡牌翻转**：点击翻转动画
- **按钮交互**：hover时上移4px，阴影加深，0.2秒ease-out
- **弃牌闪烁**：弃牌按钮红色闪烁一次
- **胜利光环**：金色光环循环旋转，2秒一圈
- **胜利提示**：文字放大后恢复，0.6秒，白到金渐变

### 3.7 响应式设计
- **宽度 < 768px**：
  - 玩家座位卡片：宽120px高200px
  - 牌桌中央区域缩小
  - ChipBar按钮自适应宽度

## 4. 性能需求

| 指标 | 要求 |
|------|------|
| 首次加载时间 (FCP) | < 2秒 |
| 每帧渲染时间 | ≤ 16ms (60FPS) |
| WebSocket消息到UI更新延迟 | < 200ms |

---

## 5. 技术栈

### 5.1 前端
- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **路由**：React Router
- **图表**：Chart.js + react-chartjs-2
- **实时通信**：WebSocket (原生)

### 5.2 后端
- **运行时**：Node.js
- **Web框架**：Express
- **实时通信**：ws (WebSocket库)
- **ID生成**：uuid
