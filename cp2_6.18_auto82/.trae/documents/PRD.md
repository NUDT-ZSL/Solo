## 1. 产品概述

社区共享工坊管理工具是一款面向小型社区工坊的工具借用和维修记录管理系统，解决传统纸质登记方式数据混乱、无法追踪工具状态和会员行为的痛点。

- **核心价值**：通过数字化管理实现工具全生命周期追踪，自动统计损坏率和会员借用行为，提升工坊运营效率
- **目标用户**：社区工坊会员（工具使用者）和工坊管理员（工具维护者）

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 会员 | 默认访客身份 | 浏览工具、发起借用、查看个人借用记录、提交使用反馈 |
| 管理员 | 访问/admin路由 | 查看所有维修工单、标记维修完成、管理工具状态 |

### 2.2 功能模块

1. **首页**：工具搜索浏览、工具卡片列表、详情弹窗
2. **借用申请页**：工具信息展示、借用时长选择、申请提交
3. **个人中心**：借用记录列表、倒计时显示、状态管理、使用反馈
4. **维修管理页**：待修工单列表、修复标记、工具状态恢复

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页 | 搜索栏 | 支持按工具名称模糊搜索，实时过滤工具列表 |
| 首页 | 工具卡片 | 显示工具照片、名称、状态标签，点击展开详情 |
| 首页 | 详情弹窗 | 展示使用说明、最近三次借用记录，底部滑入动画 |
| 借用申请页 | 借用表单 | 选择借用时长（1/2/4/8/24小时），提交申请 |
| 个人中心 | 借用记录 | 显示借用卡片，倒计时精确到秒，超时红色边框闪烁 |
| 个人中心 | 反馈表单 | 提交工具状态（正常/轻微磨损/损坏需修理）和100字评价 |
| 维修管理页 | 工单列表 | 展示待修工具、报告人、描述、时间，支持标记修复 |

## 3. 核心流程

```mermaid
flowchart LR
    A["浏览工具"] --> B["查看工具详情"]
    B --> C["发起借用申请"]
    C --> D["选择借用时长"]
    D --> E["提交申请生成记录"]
    E --> F["个人中心倒计时"]
    F --> G["归还时提交反馈"]
    G --> H{"是否损坏"}
    H -->|是| I["自动生成维修工单"]
    H -->|否| J["工具恢复可用"]
    I --> K["管理员查看工单"]
    K --> L["标记已修复"]
    L --> J
```

**主要用户流程**：
1. 会员在首页浏览可用工具，通过搜索快速定位
2. 点击工具卡片查看详情和历史借用记录
3. 填写借用时长后提交申请，系统生成带倒计时的借用记录
4. 使用完毕后提交使用反馈，若标记损坏则自动生成维修工单
5. 管理员在维修管理页处理工单，标记修复后工具恢复可用

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：浅绿色系，背景渐变从#e8f5e9到#c8e6c9
- **辅助色**：白色为主，状态色（绿色#4caf50可用、橙色#ff9800借出、红色#f44336维修中）
- **强调色**：倒计时红色#e53935，逾期警告边框
- **卡片样式**：圆角12px，宽220px高280px，悬浮时上移3px带阴影，过渡0.3s ease-out
- **字体**：使用现代无衬线字体，层级清晰
- **布局**：卡片式网格布局，桌面端多列，移动端单列

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 首页 | 导航栏 | 居中高度56px，白底+底部1px#e0e0e0分割线，链接悬停变色 |
| 首页 | 工具卡片 | 顶部圆形照片（60px）、名称、状态标签、渐变背景 |
| 首页 | 详情弹窗 | 底部滑入，模糊背景遮罩0.5透明度 |
| 个人中心 | 借用卡片 | 倒计时红色字体，逾期时30秒红色边框闪烁 |
| 维修管理页 | 工单列表 | 表格布局，修复按钮绿色强调 |

### 4.3 响应式

- **桌面优先**设计，移动端自适应
- 桌面端：工具卡片网格布局，底部导航栏居中
- 移动端：工具卡片单列布局，导航栏变为顶部汉堡菜单
- 所有交互元素触摸优化，最小点击区域44px
- 列表页渲染性能要求200ms内完成

### 4.4 动画效果

- 卡片悬浮：上移3px，阴影加深，过渡0.3s ease-out
- 详情弹窗：从底部滑入，背景模糊遮罩淡入
- 逾期警告：边框红色闪烁，间隔30秒
- 倒计时：数字实时更新，红色醒目显示
- 页面切换：平滑过渡，保持用户上下文
