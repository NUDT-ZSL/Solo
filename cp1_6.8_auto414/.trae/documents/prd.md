## 1. 产品概述

「音轨炫舞」是一款赛博霓虹风格的2D节奏跑酷游戏，玩家控制发光小球在音符轨道上自动前进，根据音乐节拍切换轨道、跳跃或滑铲来收集音符碎片并躲避障碍物，完美跟随节拍可获得额外分数和连击特效。目标用户为喜爱音乐游戏和跑酷类休闲游戏的玩家。

## 2. 核心功能

### 2.1 功能模块

1. **游戏主画面**：全屏Canvas游戏画布，包含轨道、主角、障碍物、音符碎片、粒子特效
2. **状态栏**：左上角分数/连击/生命值，右上角曲目名和暂停按钮
3. **节拍指示器**：右下角毛玻璃风格预览面板，显示即将到来的音符和障碍位置
4. **暂停/恢复**：暂停按钮和快捷键支持

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 游戏主画面 | 轨道渲染 | 三条半透明发光彩色光带，颜色随节拍强度从蓝紫渐变至暖橙 |
| 游戏主画面 | 主角控制 | 暖黄色发光小球，左右键切换轨道，空格跳跃/滑铲，带粒子拖尾和脉冲光晕 |
| 游戏主画面 | 障碍物系统 | 红/橙/紫三色几何发光体，对应不同躲避动作（切换轨道/跳跃/滑铲） |
| 游戏主画面 | 音符碎片 | 蓝白色闪烁菱形，收集得分，完美节拍时金色光晕扩散 |
| 游戏主画面 | 特效系统 | 完美节拍：屏幕边缘光晕脉冲+粒子爆散；miss：屏幕震动+瞬时变暗 |
| 状态栏 | 分数/连击/生命值 | 左上角显示，生命值为三格能量环 |
| 状态栏 | 曲目名和暂停 | 右上角显示当前曲目名和暂停按钮 |
| 节拍指示器 | 预览面板 | 右下角毛玻璃风格，预览接下来几个节拍的音符和障碍位置 |

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深紫(#1a0033)到纯黑(#000000)渐变背景
- 轨道颜色：蓝紫(#6366f1) → 暖橙(#f97316) 随节拍强度变化
- 主角颜色：暖黄(#fbbf24)发光，带粒子拖尾和脉冲光晕
- 障碍物：红(#ef4444)/橙(#f97316)/紫(#8b5cf6)三色
- 音符碎片：蓝白闪烁(#60a5fa → #ffffff)
- 按钮风格：圆角半透明毛玻璃效果，霓虹边框
- 字体：Orbitron（标题/数字）+ Exo 2（正文）
- 布局风格：全屏Canvas画布，UI层半透明浮于Canvas之上

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 游戏主画面 | 轨道区域 | 三条水平发光光带，半透明渐变色，随节拍脉冲 |
| 游戏主画面 | 主角 | 暖黄发光圆球，径向渐变+外发光，拖尾粒子 |
| 游戏主画面 | 障碍物 | 旋转的几何发光体（三角/菱形/六边形），对应颜色 |
| 游戏主画面 | 音符碎片 | 闪烁菱形，蓝白渐变，旋转动画 |
| 游戏主画面 | 完美特效 | 金色光晕扩散环+屏幕边缘脉冲+粒子爆散 |
| 游戏主画面 | Miss特效 | 屏幕位移震动+颜色乘法变暗0.3s |
| 状态栏 | 分数区 | Orbitron字体，霓虹白数字，连击数带火焰色 |
| 状态栏 | 生命值 | 三格能量环，从绿到红渐变，受击闪烁 |
| 状态栏 | 暂停按钮 | 毛玻璃圆形按钮，暂停图标 |
| 节拍指示器 | 预览面板 | 毛玻璃半透明背景，小图标预览音符/障碍位置 |

### 4.3 响应式

- 桌面优先设计，全屏Canvas自适应
- 移动端：增加底部触控按钮（左移/右移/跳跃/滑铲）
- 触控按钮：半透明霓虹风格，不遮挡游戏画面核心区域
- Canvas使用window尺寸自适应，UI层使用绝对定位浮于Canvas之上

### 4.4 动画规格

- 主角移动：平滑过渡300ms，带弹性缓动
- 粒子拖尾：20-30个粒子，渐隐+缩小，生命周期0.5s
- 跳跃动画：抛物线轨迹+上升光晕
- 滑铲动画：压扁变形+下压粒子
- 完美收集：金色光晕从中心扩散至屏幕边缘，持续0.8s
- Miss反馈：屏幕水平位移±8px震动0.3s，全局亮度降至70%持续0.2s
- 轨道脉冲：节拍时轨道亮度增加30%，持续0.1s
