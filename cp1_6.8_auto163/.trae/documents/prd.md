## 1. 产品概述

「虚空牌阵」是一款暗黑幻想风格的2D策略卡牌对战游戏。玩家在3×3的棋盘格上放置不同元素属性（火、水、风、土）的卡牌，通过策略摆放触发元素连线和属性克制，击败AI对手。目标用户为喜欢策略卡牌和暗黑美学风格的玩家。

## 2. 核心功能

### 2.1 功能模块

1. **游戏主界面**：3×3棋盘、手牌区、生命值显示、回合控制、属性克制提示
2. **卡牌系统**：四种元素卡牌生成、攻击力计算、属性克制关系
3. **战斗系统**：回合制对战、自动攻击、元素爆发、胜负判定
4. **动画系统**：卡牌拖拽、放置光环、攻击冲撞、粒子碎裂特效

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主界面 | 棋盘区域 | 3×3网格，支持拖拽放置和交换卡牌，显示已放置卡牌及属性颜色 |
| 游戏主界面 | 手牌区 | 每回合随机3张手牌，支持拖拽到棋盘，悬停放大显示详细信息 |
| 游戏主界面 | 生命值面板 | 显示双方20点生命值，实时更新，受击时动画反馈 |
| 游戏主界面 | 回合控制 | 回合结束按钮、当前回合数显示、回合阶段提示 |
| 游戏主界面 | 属性克制提示 | 火→风→土→水→火的克制关系图示，当前对局克制事件提示 |
| 游戏主界面 | 元素爆发特效 | 同属性三连触发元素爆发，对敌方造成额外伤害并附加状态效果 |

## 3. 核心流程

1. 游戏开始，双方各20点生命值
2. 玩家回合：获得3张随机手牌 → 拖拽手牌到棋盘空格 → 卡牌自动攻击敌方对应位置 → 检测元素连线 → 触发元素爆发（如有）→ 点击"回合结束"
3. AI回合：AI自动放置手牌 → 自动攻击 → 检测元素连线 → 触发元素爆发
4. 检查胜负：任一方生命值≤0则游戏结束

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深紫色渐变背景（#1a0a2e → #2d1b4e），搭配暗黑幻想氛围
- **卡牌材质**：半透明发光材质，火-橙红(#ff6b35)、水-冰蓝(#4fc3f7)、风-青绿(#26a69a)、土-棕黄(#d4a017)
- **按钮风格**：圆角毛玻璃效果，hover时发光边框
- **字体**：标题使用Cinzel（暗黑幻想风格），正文使用Noto Sans SC
- **布局风格**：居中棋盘，上下对称显示双方信息，底部手牌区
- **图标风格**：元素符号图标（火焰、水滴、风旋、岩石）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主界面 | 棋盘区域 | 深色网格线、卡牌放置时属性色光环扩散、攻击冲撞动画+粒子碎裂 |
| 游戏主界面 | 手牌区 | 毛玻璃面板、卡牌悬停放大1.2倍并显示攻击力和属性、拖拽时缩放0.8→1.0缓动 |
| 游戏主界面 | 生命值面板 | 毛玻璃状态栏、血条渐变色(绿→黄→红)、受击时抖动+闪红 |
| 游戏主界面 | 回合控制 | 毛玻璃圆角按钮、发光边框、回合数显示 |
| 游戏主界面 | 属性克制提示 | 克制关系环形图示、触发克制时对应元素图标高亮闪烁 |

### 4.3 响应式设计

- 桌面优先（1920×1080基准），适配平板（768px以上）
- 棋盘区域保持正方形比例，手牌区在小屏时改为横向滚动
- 触控优化：拖拽区域适当扩大，长按触发卡牌详情

### 4.4 动画与性能

- 卡牌放置：缓动缩放(0.8→1.0, ease-out, 300ms) + 属性色光环扩散(600ms)
- 攻击动画：卡牌前冲(translateX, 400ms) + 粒子碎裂(CSS粒子, 500ms)
- 元素爆发：全屏属性色闪光 + 连线高亮脉冲
- 目标帧率：稳定60fps，动画使用requestAnimationFrame + CSS will-change优化
