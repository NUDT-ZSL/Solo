## 1. 产品概述

「时间胶囊」交互式个人博客首页——一个以年份为时间轴的回忆展示平台，用户通过点击年份标签，沉浸式浏览不同年份的短文与照片，背景随年份主题色平滑渐变，营造穿越时光的体验。

- 目标用户：追求视觉品质的个人博客作者及访客
- 核心价值：用极简毛玻璃美学 + 流畅动画，将零散记忆组织成可交互的时间长卷

## 2. 核心功能

### 2.1 功能模块

1. **首页**：年份时间线导航、瀑布流记忆卡片、卡片展开详情、背景主题渐变

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 年份导航栏 | 顶部横向排列所有年份按钮，当前年份高亮，hover 轻微放大，点击切换年份 |
| 首页 | 瀑布流卡片区域 | 展示当前年份的记忆卡片（短文 + 图片），PC 三列 / 平板两列 / 手机单列响应式布局 |
| 首页 | 卡片展开详情 | 点击卡片展开更多内容，展开动画自然流畅 |
| 首页 | 背景渐变 | 根据年份主题色平滑过渡背景（2024 暖橙、2020 冷蓝等） |

## 3. 核心流程

用户进入首页 → 默认展示当前年份内容 → 点击其他年份标签 → 背景渐变过渡 → 卡片淡入+底部滑入动画 → 点击卡片展开详情 → 再次点击收起

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：年份主题色系统（暖橙、冷蓝、薄荷绿、烟粉、琥珀金等），中性底色为深灰近黑
- **按钮样式**：圆角药丸型年份按钮，毛玻璃背景（backdrop-blur），当前年份实色填充
- **字体**：显示字体使用 Playfair Display（优雅衬线），正文使用 Noto Sans SC（中文友好）
- **布局风格**：顶部固定导航 + 居中瀑布流内容区
- **特效**：毛玻璃（glassmorphism）、渐变背景、卡片淡入滑入动画、hover 微放大

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 年份导航栏 | 药丸按钮、毛玻璃底栏、当前年份高亮填充、hover scale(1.08) |
| 首页 | 瀑布流卡片区 | 毛玻璃卡片、圆角 16px、图片占位+文字摘要、3/2/1 列响应式 |
| 首页 | 卡片展开态 | 背景遮罩、展开面板、全内容展示、平滑高度过渡 |
| 首页 | 全局背景 | 多层径向渐变、根据年份切换主色、transition 0.8s |

### 4.3 响应式设计

- **桌面端（≥1024px）**：三列瀑布流，导航栏居中
- **平板端（768-1023px）**：两列瀑布流，导航可横向滚动
- **移动端（<768px）**：单列卡片，导航紧凑排列
