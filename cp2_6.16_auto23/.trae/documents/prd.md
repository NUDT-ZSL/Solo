## 1. 产品概述

团队情绪晴雨表与微调查分析应用，用于远程办公团队每日匿名收集员工心情与压力数据，自动统计团队整体情绪指数并以时间趋势图展示，帮助管理者及时了解团队状态并安排调整活动。

- 目标用户：远程办公团队的普通员工（匿名提交）和管理者（查看报告）
- 核心价值：通过可视化情绪数据，实现团队心理健康的早期预警与主动管理

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 员工 | 无需注册（匿名） | 每日提交心情和文字描述 |
| 管理者 | 无需注册 | 查看团队情绪汇总报告、关键词云 |

### 2.2 功能模块

1. **情绪提交页面**：5个情绪表情按钮 + 文字描述输入 + 提交反馈
2. **管理报告页面**：今日情绪分布饼图 + 7天情绪趋势折线图 + 关键词云

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 情绪提交页 | 情绪选择区 | 5个圆形表情按钮（开心/平静/一般/低落/焦虑），点击放大1.2倍并保持选中，自动弹出文本输入框 |
| 情绪提交页 | 文字描述区 | 宽100%高120px文本框，placeholder"分享一点感受（可选）"，200字限制并实时显示剩余字数 |
| 情绪提交页 | 提交按钮 | 渐变色按钮，点击后loading旋转1秒，成功显示绿色对勾2秒，失败显示红色叉号并保留内容 |
| 管理报告页 | 情绪分布饼图 | 今日5种心情占比，颜色对应表情按钮 |
| 管理报告页 | 7天趋势折线图 | X轴日期，Y轴情绪指数(1-5)，渐变折线，圆点标记+悬浮提示 |
| 管理报告页 | 关键词云 | 前50个常见词，大小由频率决定(12-36px)，随机柔和色 |
| 全局 | 导航栏 | 标题"团队晴雨表"+ Tab切换（提交心情/查看报告），0.3s平滑过渡 |
| 全局 | Toast提示 | 顶部居中，320x56px，圆角16px，深色背景白字，1秒渐变消失 |

## 3. 核心流程

**员工提交流程**：打开应用 → 选择心情表情 → （可选）输入文字描述 → 点击提交 → 数据写入SQLite → 显示提交成功Toast

**管理者查看流程**：打开应用 → 切换到"查看报告" → 从SQLite读取数据 → 渲染饼图+折线图+关键词云

```mermaid
    "C" --> "E[直接提交]"

    "J[管理者切换报告页]" --> "K[GET /api/report]"
    "K" --> "L[SQLite查询数据]"
    "L" --> "M[计算统计数据]"

```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：毛玻璃风格，背景柔和渐变(#E3F2FD → #FCE4EC → #E8F5E9)
- 卡片样式：白色半透明卡片，backdrop-filter: blur(12px)，圆角32px，内边距40px，阴影0 8px 32px rgba(0,0,0,0.08)
- 按钮风格：圆角胶囊按钮，0.2s ease-in-out过渡动画
- 字体：标题24px/700/#333，Tab文字未选中#666选中白色
- 图标风格：表情用圆形彩色按钮表示，状态图标用对勾/叉号

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 情绪提交页 | 情绪选择区 | 5个60x60px圆形按钮，颜色：黄#FFD700/绿#81C784/灰#BDBDBD/蓝#64B5F6/红#E57373，选中放大1.2倍 |
| 情绪提交页 | 文字输入区 | 宽100%高120px，圆角12px，背景#F5F5F5，200字限制+剩余字数 |
| 情绪提交页 | 提交按钮 | 120x48px圆角24px，渐变#4FC3F7→#29B6F6，悬停亮度+10% |
| 管理报告页 | 饼图 | 今日情绪分布，颜色对应5种心情 |
| 管理报告页 | 折线图 | 渐变折线#FFD700→#E57373，圆点标记+悬浮提示 |
| 管理报告页 | 词云 | 50词，12-36px，10种柔和色随机 |
| 全局 | 导航栏 | 标题+Tab按钮(100x40px圆角20px)，选中背景#4FC3F7白字 |
| 全局 | Toast | 320x56px圆角16px，背景#323232白字，1s渐变消失 |

### 4.3 响应式设计

- 桌面优先设计，中央卡片宽1000px
- 手机端(<768px)：卡片宽度96%，内边距20px，表情按钮缩至48x48px
- 所有过渡动画0.2s ease-in-out
