## 1. 产品概述

『笔墨流光』是一款基于Web的交互式水墨画模拟应用，让用户通过鼠标或触屏在虚拟宣纸上挥毫泼墨。墨迹会随时间自然晕染、褪色并产生渐变效果，点击时爆散成墨点再缓慢聚拢，整体形成一幅不断变化的动态水墨画。

- 目标用户：喜欢中国传统文化、水墨艺术的用户，以及追求创意表达和视觉放松的人群
- 核心价值：以现代Web技术重现传统水墨画的韵味与意境，提供沉浸式的数字水墨创作体验

## 2. 核心功能

### 2.1 功能模块

1. **画布页面**：毛笔绘制、墨迹晕染、爆散聚拢动效
2. **控制面板**：墨色浓度调节、晕染速度调节、画布清空、保存为PNG

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 画布页面 | 毛笔绘制 | 鼠标/触屏拖拽绘制流畅线条，支持压力感应的粗细和浓度变化，边缘带随机不规则锯齿 |
| 画布页面 | 墨迹晕染 | 墨迹落纸后产生圆形波纹式扩散动画，速度随墨量递减，逐渐变淡褪色至浅灰色 |
| 画布页面 | 爆散聚拢 | 点击鼠标时爆散10-20个小墨点，缓慢向中心聚拢并融合为较大墨滴 |
| 控制面板 | 墨色浓度 | 滑块控制0.1-1.0，影响笔触墨量和晕染范围 |
| 控制面板 | 晕染速度 | 滑块控制0.5-2.0，影响墨迹扩散速率 |
| 控制面板 | 画布清空 | 清除所有墨迹，恢复空白宣纸 |
| 控制面板 | 保存为PNG | 将当前画布导出为PNG图片下载 |

## 3. 核心流程

用户打开页面后看到宣纸背景的画布和右侧控制面板。拖拽鼠标在画布上绘制墨迹，墨迹自然晕染扩散并逐渐褪色；点击画布时产生爆散-聚拢墨点效果。通过控制面板调整墨色浓度和晕染速度，可清空画布或保存作品。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米白(#F5F0E8)到浅灰(#E8E3DB)的宣纸渐变背景
- **墨色**：主体黑色(#1A1A1A)，晕染边缘淡蓝灰半透明(#7B8FA1, 0.3透明度)
- **控制面板**：半透明白底(rgba(255,255,255,0.72))加浅木色细边框(#C4A87C)
- **按钮样式**：仿古印章样式，深朱砂红底(#8B2500)，悬停时微微发光(box-shadow发光效果)
- **字体**：标题使用"ZCOOL XiaoWei"(站酷小薇体)，正文使用"Noto Serif SC"
- **布局**：桌面端竖屏适配，画布居左占大部分，右侧240px毛玻璃控制面板；移动端画布全宽，控制面板底部折叠

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 画布页面 | 宣纸画布 | 渐变背景、全屏Canvas、自定义光标(毛笔笔尖样式) |
| 画布页面 | 绘制效果 | 流畅线条、边缘锯齿、晕染波纹、褪色渐变 |
| 画布页面 | 爆散效果 | 10-20个小墨点向外散开再聚拢融合 |
| 控制面板 | 面板容器 | 毛玻璃效果、浅木色边框、圆角8px |
| 控制面板 | 墨色浓度滑块 | 自定义样式滑块、0.1-1.0范围、显示当前值 |
| 控制面板 | 晕染速度滑块 | 自定义样式滑块、0.5-2.0范围、显示当前值 |
| 控制面板 | 画布清空按钮 | 仿古印章样式、朱砂红底白字 |
| 控制面板 | 保存PNG按钮 | 仿古印章样式、朱砂红底白字 |

### 4.3 响应式设计

- **桌面端（≥768px）**：画布占据左侧大部分空间，右侧固定240px控制面板
- **移动端（<768px）**：画布全宽自适应屏幕，控制面板切换为底部折叠式，点击汉堡图标展开
- **触屏优化**：触屏绘制时禁止页面滚动，支持多点触控的单指绘制

### 4.4 动效指导

- 墨迹晕染：圆形波纹式扩散，使用径向渐变模拟，扩散速度随墨量递减
- 褪色：alpha值随时间线性递减，最终保留浅灰色痕迹(alpha=0.08)
- 爆散：墨点以扇形随机方向向外弹射，速度先快后慢(ease-out)，然后反向缓慢聚拢(ease-in-out)
- 画布背景：宣纸纹理通过多层噪声叠加实现微妙质感
