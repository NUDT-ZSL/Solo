## 1. 产品概述

音教管理系统是一款面向小型音乐教育机构的线上管理工具，旨在解决教师课表管理、学生练琴记录追踪、曲目库维护以及家长查看孩子学习进度的核心需求。目标用户为音乐机构管理员、教师、学生及学生家长。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 账号密码 | 创建教师/学生档案、设置每周课表、冲突检测 |
| 教师 | 账号密码 | 查看今日课表、查看学生练琴记录、提交练琴日志 |
| 学生 | 账号密码 | 查看今日课程、提交练琴日志（曲目、时长、心得） |
| 家长 | 输入孩子姓名或ID | 查看课程考勤率、查看练琴统计（7天柱状图） |

### 2.2 功能模块

1. **教师仪表盘页面**：今日课表列表（卡片式）、学生练琴日志查看与提交、课程卡片展开交互
2. **家长看板页面**：孩子搜索（姓名/ID）、课程出勤率环形图、7天练琴时长柱状图
3. **课表冲突检测模块**：新增/修改课表时自动检测时间冲突、冲突弹窗警告
4. **练琴数据统计模块**：按日分组统计练琴时长、周练琴时长趋势

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 教师仪表盘 | 今日课表列表 | 从后端获取今日课程数据，以卡片形式展示学生头像、姓名、时间、曲目 |
| 教师仪表盘 | 练琴日志面板 | 点击课程卡片展开，显示学生历史练琴记录，支持提交新日志 |
| 家长看板 | 孩子搜索 | 输入孩子姓名或ID查询学习数据 |
| 家长看板 | 出勤率环形图 | SVG绘制环形进度图，展示课程出勤百分比 |
| 家长看板 | 练琴柱状图 | 展示最近7天练琴时长，渐变柱体，无数据灰色虚线 |
| 冲突检测 | 冲突弹窗 | 检测到课表冲突时弹出警告对话框，含抖动动画 |

## 3. 核心流程

**教师查看课表并管理练琴日志流程**：
教师登录 → 进入教师仪表盘 → 查看今日课程卡片列表 → 点击某课程卡片展开 → 查看该学生历史练琴记录 → 可提交新的练琴日志（选择曲目、输入时长、填写心得）→ 提交成功显示绿色提示

**家长查看孩子学习进度流程**：
家长进入家长看板 → 输入孩子姓名或ID → 系统查询数据 → 展示出勤率环形图和7天练琴柱状图

**管理员排课冲突检测流程**：
管理员添加/修改课表 → 后端调用checkConflict检测 → 若冲突返回冲突详情 → 前端弹出警告对话框

## 4. 用户界面设计

### 4.1 设计风格

- 主色：#4A90D9（浅蓝色）
- 辅色：#F0F4F8（浅蓝灰背景）
- 强调色：#6C63FF（紫色）
- 成功色：#27AE60（绿色提示）
- 错误色：#E74C3C（红色警告）
- 按钮风格：圆角8px，hover上浮translateY(-4px) 0.2s，点击scale(0.98) 0.15s
- 字体：系统默认，标题20px/600，正文14px
- 布局：左右10%间距居中，顶部导航64px白色背景带阴影
- 卡片：圆角12px，背景#F0F4F8，左边框4px色条#4A90D9，hover上浮加深阴影

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 教师仪表盘 | 今日课表卡片 | 圆角12px卡片，左侧4px蓝色色条，圆形40px头像占位（随机色背景），学生姓名、课程时间、曲目信息，右侧展开箭头，hover上浮0.2s |
| 教师仪表盘 | 练琴日志面板 | 展开后显示日志列表（淡入0.3s），曲目下拉选择，时长数字输入5-120，心得多行输入，提交按钮，成功/失败提示条 |
| 家长看板 | 出勤率环形图 | SVG环形进度图，圆弧色#4A90D9，背景圆弧#E0E6ED，百分比居中显示 |
| 家长看板 | 练琴柱状图 | 柱体渐变#6C63FF→#4A90D9，无数据灰色虚线柱体 |
| 全局 | 冲突弹窗 | 圆角16px，背景#FFF5F5，边框红色#E74C3C 2px，抖动动画0.3s |

### 4.3 响应式适配

- 桌面优先设计（Desktop-first）
- 屏幕宽度<768px时：
  - 导航栏按钮变为汉堡菜单
  - 课程卡片改为单列布局，左侧色条隐藏
  - 日志面板从右侧展开改为下方折叠
- 触摸优化：按钮点击区域不小于44px

### 4.4 性能约束

- 课表列表渲染控制在50毫秒内（数据量≤100条）
- 日志提交请求响应时间≤500毫秒
