## 1. 产品概述

「幻灵棋盘」是一款基于8×8发光棋盘的回合制策略对战游戏。玩家通过拖动幻灵棋子占领格子、释放技能攻击对手，在霓虹符文风格的视觉体验中展开博弈。

- 目标用户：喜欢策略对战的休闲和硬核玩家
- 核心价值：将棋类策略与RPG技能系统结合，配合炫酷的霓虹粒子特效，打造沉浸式对战体验

## 2. 核心功能

### 2.1 玩家角色
| 角色 | 说明 |
|------|------|
| 蓝紫方（玩家1） | 操控蓝紫阵营幻灵，从棋盘下方出发 |
| 橙红方（玩家2/AI） | 操控橙红阵营幻灵，从棋盘上方出发 |

### 2.2 功能模块
1. **游戏主界面**：8×8发光棋盘、棋子展示、粒子特效、拖拽交互
2. **信息面板**：玩家信息、能量条、回合倒计时、行动日志
3. **控制面板**：技能按钮、结束回合按钮

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主界面 | 棋盘区域 | Canvas渲染8×8发光棋盘，格子随时间变换颜色和光效，占领格子持续发光并生成己方能量 |
| 游戏主界面 | 棋子系统 | 拖拽棋子移动，粒子拖尾特效，不同幻灵有不同属性和技能 |
| 游戏主界面 | 技能系统 | 技能释放带爆炸、闪电等粒子特效，帧率保持60fps |
| 游戏主界面 | 左侧信息栏 | 玩家头像/名称、能量条（蓝紫/橙红）、回合倒计时 |
| 游戏主界面 | 右侧日志栏 | 毛玻璃面板显示行动日志，记录每次操作 |
| 游戏主界面 | 底部控制栏 | 技能按钮（3-4个）、结束回合按钮 |

## 3. 核心流程

1. 游戏开始 → 双方各在己方区域放置3个幻灵棋子
2. 回合开始 → 回合计时器启动，玩家可拖拽棋子移动或释放技能
3. 拖拽移动 → 棋子跟随鼠标/手指移动，释放时吸附到最近合法格子，产生粒子拖尾
4. 技能释放 → 选择技能后点击目标格子/棋子，播放对应粒子特效，计算伤害和效果
5. 占领格子 → 棋子停留的格子变为己方颜色，每回合生成能量
6. 结束回合 → 点击结束回合按钮或倒计时归零，切换到对方回合
7. 胜负判定 → 一方棋子全部被消灭则对方获胜

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深紫（#1a0a2e）到黑（#0a0a0f）渐变背景
- **棋盘线条**：半透明发光蓝色（rgba(0,180,255,0.4)）
- **蓝紫阵营色**：#6c5ce7 → #a855f7 渐变
- **橙红阵营色**：#e17055 → #f97316 渐变
- **字体**：Orbitron（标题）+ Rajdhani（正文）
- **布局**：三栏布局（左信息 + 中棋盘 + 右日志），底部控制栏
- **特效风格**：霓虹发光、粒子拖尾、光晕扩散、毛玻璃面板

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主界面 | 棋盘区域 | Canvas全屏渲染，发光格线，阵营色格子填充，棋子发光图标+粒子拖尾 |
| 游戏主界面 | 左侧信息栏 | 半透明暗色面板，玩家头像框，能量条（渐变色+动画），回合计时器（数字倒计时+脉冲动画） |
| 游戏主界面 | 右侧日志栏 | 毛玻璃面板（backdrop-blur），滚动列表，操作文字+阵营色标记 |
| 游戏主界面 | 底部控制栏 | 技能图标按钮（发光边框+hover放大），结束回合按钮（脉冲发光） |

### 4.3 响应式适配
- 桌面端（≥1024px）：三栏布局，棋盘居中
- 平板端（768-1023px）：棋盘居中，左右面板折叠为可展开侧栏
- 手机端（<768px）：棋盘全宽，信息栏和控制栏上下排列，触控优化拖拽
