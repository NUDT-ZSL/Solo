## 1. 产品概述

「幻境棋局」是一款 2D 回合制策略战棋游戏，玩家控制一支由不同职业英雄组成的小队在 8×8 网格地图上对抗 AI 敌人。每回合玩家可移动和攻击，地图随机生成增益道具，击败所有敌人即可获胜。

- 目标用户：喜欢策略战棋类游戏的休闲及中度玩家
- 核心价值：易上手难精通的战术深度 + 精致的像素风视觉体验

## 2. 核心功能

### 2.1 玩家角色

| 职业 | 生命值 | 攻击力 | 移动范围 | 攻击范围 | 特殊技能 |
|------|--------|--------|----------|----------|----------|
| 战士 | 高(120) | 中(25) | 3格 | 近战(1格) | 「猛击」：对目标造成 1.5 倍伤害并击退 1 格 |
| 弓箭手 | 低(70) | 中(20) | 4格 | 远程(3格) | 「精准射击」：无视障碍对目标造成 2 倍伤害 |
| 法师 | 低(60) | 高(30) | 3格 | 范围(2格) | 「火球术」：对目标及周围 1 格范围内所有敌人造成伤害 |

### 2.2 功能模块

1. **战斗主界面**：网格地图渲染、单位显示、交互操作（选择/移动/攻击）
2. **回合系统**：玩家回合与敌方回合交替，回合切换有白光过渡动画
3. **AI 敌人系统**：优先攻击低血量单位，尝试包围玩家
4. **道具系统**：地图随机生成攻击增益和移动增益道具
5. **胜负判定**：击败所有敌人即胜，我方全灭即败

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 战斗主界面 | 网格地图 | 8×8 网格渲染，障碍物/道具/单位显示，移动路径虚线高亮，攻击范围红色半透明显示 |
| 战斗主界面 | 回合系统 | 回合计数器，玩家/敌方回合切换，白光过渡动画 |
| 战斗主界面 | 操作面板 | 结束回合按钮、技能按钮、毛玻璃质感 |
| 战斗主界面 | 状态栏 | 底部显示当前回合数、选中英雄信息（职业/血量/技能冷却） |
| 战斗主界面 | 单位交互 | 选中单位金色描边，可移动格子虚线，可攻击格子红色半透明 |
| 开始界面 | 标题画面 | 游戏标题、开始按钮、深色中世纪风格背景 |
| 结算界面 | 胜负显示 | 胜利/失败提示、重新开始按钮 |

## 3. 核心流程

玩家打开游戏 → 进入开始界面 → 点击开始 → 生成 8×8 地图（障碍物+道具+英雄+敌人）→ 玩家回合开始 → 选择英雄 → 显示移动范围 → 移动英雄 → 显示攻击范围 → 攻击敌人 → 所有英雄行动完毕/点击结束回合 → 敌方回合 → AI 控制敌人行动 → 检查胜负 → 循环至游戏结束

    "所有英雄行动完毕?" -- 是 --> "敌方回合"
    "敌方回合" --> "AI行动"
    "AI行动" --> "检查胜负"

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色中世纪风格，背景深灰蓝 `#1a1a2e`，辅色暗紫 `#16213e`
- **强调色**：金色 `#f0c040`（选中/高亮），银色发光 `#c0c8e0`（网格线）
- **危险色**：暗红 `#e74c3c`（攻击范围/敌人）
- **按钮风格**：毛玻璃质感（`backdrop-filter: blur`），悬停微缩放 1.05 倍
- **字体**：像素风格标题字体 + 清晰可读的 UI 字体
- **布局**：中央 Canvas 地图区 + 底部状态栏 + 右侧/底部操作面板
- **图标风格**：像素风角色图标，带呼吸光晕动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 战斗主界面 | 网格地图 | 深色背景、银色发光网格线、障碍物深色方块、道具发光图标 |
| 战斗主界面 | 英雄单位 | 像素风图标、呼吸光晕、选中金色描边、血条 |
| 战斗主界面 | 敌方单位 | 红色调像素图标、血条 |
| 战斗主界面 | 移动路径 | 蓝色虚线高亮格子 |
| 战斗主界面 | 攻击范围 | 红色半透明覆盖格子 |
| 战斗主界面 | 操作面板 | 毛玻璃按钮（移动/攻击/技能/结束回合）、悬停缩放 |
| 战斗主界面 | 状态栏 | 底部固定栏、当前回合数、选中英雄信息 |
| 开始界面 | 标题 | 大号像素风标题「幻境棋局」、发光效果 |
| 结算界面 | 结果 | 胜利金色/失败暗红大字、重新开始按钮 |

### 4.3 响应式适配

- 桌面端（≥1024px）：地图居中大尺寸显示，操作面板右侧
- 平板端（768px-1023px）：地图自适应缩放，操作面板底部
- 触摸优化：格子点击区域适当放大，按钮触摸友好

### 4.4 动画特效

| 场景 | 动画效果 |
|------|----------|
| 角色移动 | 沿路径缓动播放，每格约 150ms |
| 角色攻击 | 角色前冲动画 + 冲击波粒子特效 |
| 道具拾取 | 发光粒子飞向角色 |
| 回合切换 | 屏幕白光闪烁过渡 |
| 单位待机 | 呼吸光晕动画 |
| 单位选中 | 金色描边脉冲 |
| 英雄死亡 | 渐隐 + 粒子消散 |
