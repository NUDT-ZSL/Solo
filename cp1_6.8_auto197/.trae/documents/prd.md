## 1. 产品概述

「味觉梦境」是一个在线美食梦境记录平台，让用户记录和可视化关于食物的梦境体验。系统根据用户输入的情绪和食物关键词，通过粒子动画生成独特的「梦境食单」——模拟食物在盘中缓缓组合的奇幻效果，带来沉浸式的视觉体验。

- 目标用户：喜欢记录梦境、对美食与奇幻视觉有兴趣的创意人群
- 核心价值：将抽象的味觉梦境转化为可交互、可分享的粒子动画视觉作品

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 创建、浏览、搜索梦境记录 |

### 2.2 功能模块

1. **首页**：梦境瀑布流展示、搜索栏、创建入口、背景粒子动画
2. **创建梦境页**：表单提交梦境记录（标题、描述、情绪、食物关键词）
3. **梦境详情页**：Canvas 粒子动画场景、拖拽旋转视角、点击交互、味道描述卡片

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 瀑布流卡片 | 梦境记录以缓动淡入方式从底部向上浮动展示 |
| 首页 | 搜索栏 | 输入情绪或食物关键词动态过滤梦境记录 |
| 首页 | 背景粒子 | 缓慢飘浮的细小发光粒子（香料粉尘效果） |
| 首页 | 创建按钮 | 点击进入创建梦境表单页 |
| 创建梦境页 | 梦境表单 | 输入标题、描述，选择情绪标签，填写食物关键词 |
| 创建梦境页 | 粒子预览 | 实时预览当前情绪+关键词对应的粒子效果 |
| 梦境详情页 | Canvas 场景 | 完整的粒子动画梦境食单展示 |
| 梦境详情页 | 拖拽旋转 | 鼠标拖拽旋转视角，围绕中心食物元素 |
| 梦境详情页 | 点击交互 | 点击食物元素触发粒子爆散和毛玻璃信息卡片 |
| 梦境详情页 | 味道卡片 | 毛玻璃卡片显示味道描述和情绪浓度 |

## 3. 核心流程

用户打开平台 → 浏览首页瀑布流中的梦境记录 → 点击创建按钮 → 填写梦境表单（标题、描述、情绪、食物关键词）→ 系统生成粒子动画预览 → 提交后回到首页 → 点击任一卡片进入详情页 → 拖拽旋转查看梦境食单 → 点击食物元素触发爆散和味道描述 → 返回首页继续浏览或搜索。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：暖橙（#FF8C42）到淡紫（#C9A9FF）渐变
- 辅助色：深褐（#3D2B1F）用于文字，金色（#FFD700）用于强调
- 卡片样式：毛玻璃（backdrop-filter: blur），柔和圆角（16px），温暖阴影
- 字体：展示字体选用 Cormorant Garamond（奇幻优雅），正文选用 Noto Sans SC（清晰中文）
- 布局：瀑布流卡片式布局，顶部导航栏
- 图标：lucide-react 图标库
- 动画：缓动淡入（cubic-bezier）、卡片悬停放大、粒子飘浮、页面切换过渡

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 导航栏 | 毛玻璃背景，Logo居左，搜索框居中，创建按钮居右 |
| 首页 | 瀑布流 | 3列（桌面）/2列（平板）/1列（手机），卡片从底部缓动浮入 |
| 首页 | 背景 | 暖橙到淡紫渐变 + 飘浮发光粒子 |
| 首页 | DreamCard | 毛玻璃卡片，情绪标签，食物元素标签，粒子预览小窗 |
| 创建页 | 表单 | 毛玻璃卡片表单，情绪选择器（彩色圆形按钮），关键词输入 |
| 创建页 | 预览区 | 小型 Canvas 实时预览粒子效果 |
| 详情页 | Canvas | 全屏 Canvas 粒子场景，中心食物组合动画 |
| 详情页 | 味道卡片 | 毛玻璃弹出卡片，味道描述文字，情绪浓度条 |

### 4.3 响应式适配

- 桌面端（≥1024px）：3列瀑布流，全功能交互
- 平板端（768-1023px）：2列瀑布流，简化粒子效果
- 移动端（<768px）：1列瀑布流，触摸优化（滑动替代拖拽），降低粒子数量保帧率
- 所有断点保持 60fps 帧率

### 4.4 Canvas 场景指引

- 环境：深色半透明背景，暖色环境光，中心焦点
- 灯光：柔和暖色点光源 + 环境光
- 相机：透视投影，围绕中心食物旋转
- 粒子系统：根据情绪和食物类型配置颜色、速度、形态
  - 快乐+甜点：暖橙/金色发光粒子，缓慢旋转上升
  - 忧伤+热汤：蓝色粒子冒泡上升
  - 宁静+咖啡：棕色粒子烟雾缭绕
  - 激动+烧烤：红橙色粒子飞溅
- 交互：拖拽旋转视角，点击触发爆散动画 + 信息卡片
- 后处理：辉光效果增强粒子亮度
