## 1. 产品概述

手冲咖啡日志是一款面向咖啡爱好者的个人冲煮记录与复盘工具。用户可以在每次手冲时记录豆子产地、研磨度、水温、粉水比、注水时间、风味标签和评分，并通过历史卡片和可视化统计面板回顾和优化冲煮方案。

- 目标用户：家庭手冲咖啡爱好者，注重冲煮参数的精细化调整
- 核心价值：通过结构化记录和数据可视化帮助用户发现最佳冲煮方案

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 个人用户 | 无需注册 | 浏览和使用全部功能 |

### 2.2 功能模块

1. **冲煮记录页**：冲煮参数表单，支持产地、研磨度、水温、粉水比、注水时间、风味标签、评分录入
2. **历史记录页**：瀑布流卡片布局展示所有冲煮记录，支持滚动加载
3. **统计面板页**：折线图展示评分趋势，柱状图对比产地平均评分，支持时间范围筛选

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 冲煮记录页 | 参数表单 | 咖啡豆产地下拉（8选项）、研磨度5档、水温87-96℃、粉水比4档、注水时间120-300秒、风味标签多选（最多3个）、星级评分1-10 |
| 冲煮记录页 | 操作按钮 | 保存按钮（蓝色，涟漪动画）、重置按钮（灰色悬停变红，确认弹窗） |
| 冲煮记录页 | 表单校验 | 校验不通过时输入框红色边框+红色提示文字 |
| 历史记录页 | 瀑布流卡片 | 280px宽卡片，显示产地、图片占位、参数、风味标签、评分、日期 |
| 历史记录页 | 滚动加载 | 每次加载12张，距底部300px预取，帧率50fps+ |
| 统计面板页 | 评分趋势图 | 折线图，日期横轴+评分纵轴，蓝色数据点和连线 |
| 统计面板页 | 产地评分图 | 柱状图，渐变色柱体，点击柱体筛选对应产地记录 |
| 统计面板页 | 时间筛选 | 全部/近30天/近7天切换 |

## 3. 核心流程

用户打开应用 → 在冲煮记录页填写参数并保存 → 切换到历史记录页浏览已有记录 → 切换到统计面板分析评分趋势和产地差异 → 根据数据调整下次冲煮方案

    "D" --> "E[查看统计面板]"

## 4. 用户界面设计

### 4.1 设计风格

- 主色：咖啡棕 #6B4226，奶油白 #FFF8F0
- 辅助色：蓝色 #3B82F6（保存按钮）、灰色 #9CA3AF（重置按钮）
- 风味标签色：酸 #EF4444、甜 #F59E0B、苦 #8B5CF6、醇 #10B981
- 按钮风格：圆角6px，扁平化，带涟漪动画和触感反馈
- 字体：中文优先系统字体，数字使用等宽风格
- 布局：大屏居中1200px，小屏全宽自适应，底部固定导航栏
- 背景：渐变底色（奶油白到浅咖啡色过渡）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 冲煮记录页 | 参数表单 | 下拉选择器、滑块/单选组、数字输入、标签多选、星级评分组件 |
| 冲煮记录页 | 操作按钮 | 蓝色保存（涟漪动画）、灰色重置（悬停变红、确认弹窗） |
| 历史记录页 | 瀑布流卡片 | 280px卡片、阴影悬停加深上浮、圆形图片占位、彩色风味标签 |
| 统计面板页 | 评分趋势 | Recharts折线图，蓝色实心数据点+半透明连线 |
| 统计面板页 | 产地评分 | Recharts柱状图，紫色渐变柱体，可点击筛选 |
| 统计面板页 | 时间筛选 | 顶部切换按钮组（全部/近30天/近7天） |

### 4.3 响应式设计

- 桌面优先设计，大屏居中1200px最大宽度
- 小屏全宽自适应，卡片从多列切换为单列
- 移动端触摸反馈：按钮和卡片触摸时 transform: scale(0.97)，过渡0.15s
- 底部固定导航栏适配移动端操作习惯
