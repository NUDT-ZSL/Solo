## 2. 技术说明

- 前端：React 18 + TypeScript + Vite
- 后端：Express 4 + Multer + Sharp
- 构建工具：Vite
- 字体：Google Fonts - Press Start 2P

## 3. 路由定义

| 路由 | 用途 |
|-------|---------|
| / | 主编辑器页面 |
| /api/upload | 上传参考图片 |
| /api/export-gif | 导出GIF动画 |

## 4. API 定义

### 4.1 上传并像素化图片
```typescript
// POST /api/upload
// Request: multipart/form-data
// Request Body: { image: File (max 2MB, PNG/JPG }
interface UploadResponse {
  success: boolean;
  data: string; // Base64 24x24像素化后的图片
  error?: string;
}
```

### 4.2 生成GIF
```typescript
// POST /api/export-gif
// Request: application/json
interface ExportGifRequest {
  frames: string[]; // 3帧画布数据，每帧Base64
  delay: number; // 300ms
}
// Response: Buffer (GIF binary data)
```
