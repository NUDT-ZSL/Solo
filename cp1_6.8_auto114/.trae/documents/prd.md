## 1. 产品概述

「图腾之眼」是一款2D解谜冒险游戏，玩家在古老图腾构成的迷宫中，通过旋转和组合图腾上的眼睛图案来解锁通路。
- 核心目标：为玩家提供沉浸式的解谜体验，通过视觉与交互反馈增强成就感
- 目标用户：喜爱解谜游戏、休闲益智类游戏的玩家，追求美术氛围和策略思考

## 2. 核心功能

### 2.1 功能模块

1. **游戏主界面**：迷宫画布（Canvas渲染）、玩家角色、图腾交互
2. **UI覆盖层**：关卡信息、碎片计数、规则提示面板、重置按钮

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 游戏主界面 | 迷宫画布 | 网格地图渲染，包含墙壁、空地、图腾和出口，玩家用方向键移动角色 |
| 游戏主界面 | 图腾交互 | 点击图腾旋转90度，旋转带粒子拖尾和音效反馈 |
| 游戏主界面 | 关卡系统 | 3关递进规则，收集碎片解锁下一关 |
| UI覆盖层 | 信息面板 | 左上角显示关卡编号和碎片数量 |
| UI覆盖层 | 规则提示 | 右下角毛玻璃面板，点击展示当前关卡规则动画 |
| UI覆盖层 | 重置按钮 | 重置当前关卡状态 |

## 3. 核心流程

玩家启动游戏 → 进入第一关迷宫 → 用方向键移动 → 发现图腾 → 点击旋转图腾 → 满足关卡规则条件 → 图腾发光配对成功 → 收集碎片 → 集齐3个碎片 → 出口开启 → 进入下一关 → 通关第三关 → 游戏胜利

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深岩灰（#1a1a2e）为底，配以琥珀金（#d4a855）发光线条
- **辅助色**：暗青（#16213e）做迷宫墙壁，血红（#e74c3c）/ 深蓝（#3498db）/ 翠绿（#2ecc71）/ 亮黄（#f1c40f）用于图腾眼睛
- **按钮风格**：圆角石头质感按钮，hover时发光边缘
- **字体**：Cinzel（标题，古老神秘感）+ Noto Sans SC（正文，清晰可读）
- **布局风格**：全屏Canvas居中，UI元素浮于Canvas之上
- **视觉特效**：发光线条（glow shader效果）、粒子拖尾、毛玻璃面板

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|---------|
| 游戏主界面 | 迷宫画布 | 深色岩石纹理背景，墙壁用暗青色方块，图腾用发光线条绘制的石柱+眼睛图案，出口用金色光柱 |
| 游戏主界面 | 玩家角色 | 发光眼睛图标，带呼吸光效，移动时有轨迹残影 |
| 游戏主界面 | 粒子效果 | 图腾旋转时产生柔和粒子拖尾（最多150个），配对成功时爆发金色粒子 |
| UI覆盖层 | 信息面板 | 左上角半透明深色面板，显示"第X关"和"碎片 X/3"，Cinzel字体 |
| UI覆盖层 | 规则提示 | 右下角毛玻璃面板，点击展开当前关规则，带动画演示 |
| UI覆盖层 | 重置按钮 | 右上角石头质感按钮，hover发光 |

### 4.3 响应式设计

- 桌面优先，Canvas自适应窗口大小
- 方向键为默认操控方式
- 点击图腾交互自适应Canvas坐标映射

### 4.4 关卡规则设计

| 关卡 | 规则名称 | 规则描述 |
|------|---------|---------|
| 第一关 | 方向归一 | 所有图腾眼睛颜色一致且方向朝内（指向迷宫中心） |
| 第二关 | 色彩互补 | 相邻图腾颜色需互补（红-绿配对、蓝-黄配对），方向朝内 |
| 第三关 | 方向连锁 | 一个图腾旋转会带动相邻图腾同向旋转，所有图腾方向需朝内且颜色互补 |

### 4.5 音效设计

| 场景 | 音效 | 实现方式 |
|------|------|---------|
| 图腾旋转 | 短促咔嗒声 | Web Audio API，200ms正弦波衰减 |
| 配对成功 | 胜利轻音 | Web Audio API，上升音阶组合 |
| 收集碎片 | 金属叮声 | Web Audio API，高频短音 |
| 关卡完成 | 和弦音 | Web Audio API，三音和弦 |

## 5. 性能要求

- 帧率保持60fps
- 动画使用requestAnimationFrame
- 粒子系统上限150个
- Canvas渲染优化：仅重绘变化区域
