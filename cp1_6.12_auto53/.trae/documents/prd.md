## 1. 产品概述

CodeFlow 是一款面向前端开发者的交互式代码路径可视化与调试辅助工具，核心解决"理解代码函数调用关系"的痛点。用户上传 JavaScript/TypeScript 文件后，系统自动解析 AST 提取函数定义与调用关系，以有向图形式直观展示调用链路，并支持点击节点跳转源码行、高亮调用链、过滤入口函数和循环调用等功能。

- 目标用户：前端开发者、代码审查人员、技术面试者
- 核心价值：将隐式的函数调用关系显性化，加速代码理解与调试效率

## 2. 核心功能

### 2.1 功能模块

1. **文件上传页（主页面）**：拖拽/点击上传区、源码编辑面板、图可视化面板、工具栏

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 文件上传区 | 支持拖拽或点击上传单个 .js/.ts 文件，大小限制 500KB |
| 主页面 | 有向图可视化区 | 层次布局展示函数调用关系，节点可点击、悬停放大，连线带箭头 |
| 主页面 | 源码展示区 | 等宽字体展示源码，行号显示，点击节点后自动滚动并高亮函数行 |
| 主页面 | 工具栏 | 上传按钮、'仅显示入口函数'按钮、'显示循环调用'按钮 |
| 主页面 | 可拖拽分割线 | 左右面板间分割线，拖拽调整宽度比例 |

## 3. 核心流程

```mermaid
flowchart TD
    "用户上传JS/TS文件" --> "acorn解析AST"

```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暗色主题，背景 #1e1e2e，文字 #cdd6f4，图区背景 #181825
- **按钮样式**：圆角胶囊，背景 #313244，悬停 #45475a
- **字体**：源码区 Fira Code 14px，图节点 12px
- **布局**：左右分栏（50/50），可拖拽分割线
- **节点样式**：圆角矩形，颜色按复杂度渐变（绿→黄→红）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 主页面 | 顶部工具栏 | 胶囊按钮(上传/入口函数/循环调用)，暗色背景 |
| 主页面 | 左侧源码面板 | 等宽字体行号灰色，选中行黄色半透明高亮 |
| 主页面 | 右侧图面板 | 圆角矩形节点，箭头连线，选中节点蓝色发光边框 |
| 主页面 | 分割线 | 窄条，拖拽光标，拖动时两侧自适应 |

### 4.3 响应式

- 桌面优先设计，最小宽度 1024px
- 分割线可拖拽调整比例，面板内容自适应缩放
