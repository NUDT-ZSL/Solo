## 1. 产品概述

社区亲子活动管理平台，帮助活动组织者解决报名信息混乱、签到效率低下、活动照片分散的痛点。

- 核心目标：提供一站式活动报名、签到、照片分享的数字化管理工具
- 目标用户：社区亲子活动组织者、参与活动的家长用户
- 市场价值：提升活动组织效率，改善家长参与体验，建立活动档案库

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 活动组织者 | 后台管理入口 | 创建活动、查看报名名单、导出CSV、签到管理、上传照片、标注精彩瞬间 |
| 家长用户 | 无需注册 | 浏览活动列表、搜索筛选、在线报名、查看活动详情、浏览活动照片 |

### 2.2 功能模块

1. **首页**：活动列表展示、全局搜索筛选、活动卡片浏览
2. **活动详情页**：活动信息展示、报名表单、照片瀑布流展示
3. **后台管理页**：活动创建表单、报名名单表格、导出CSV功能
4. **签到管理页**：报名者卡片网格、签到状态切换、实时保存

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 全局搜索框 | 支持按活动名称、日期范围、年龄段筛选，空状态插画展示 |
| 首页 | 活动卡片列表 | 渐变背景卡片，展示封面、名称、年龄标签、日期、报名状态，悬停动效 |
| 活动详情页 | 报名表单 | 家长姓名、电话、儿童信息（最多2名），报名成功toast提示 |
| 活动详情页 | 照片瀑布流 | 自适应列宽，懒加载，悬停放大效果，爱心标注精彩瞬间 |
| 后台管理页 | 活动创建 | 表单填写活动信息，封面图上传（JPG/PNG，≤5MB） |
| 后台管理页 | 报名名单 | 表格展示报名详情，交替行背景，CSV导出 |
| 签到管理页 | 签到卡片网格 | 180x120px卡片，已/未签到状态区分，点击切换签到，淡入淡出过渡 |

## 3. 核心流程

### 3.1 组织者创建活动流程
组织者登录后台 → 填写活动信息（名称、时间、地点、年龄、名额、描述）→ 上传封面图 → 提交创建 → 活动展示在首页列表

### 3.2 家长报名流程
家长浏览首页活动 → 点击活动卡片 → 查看详情 → 填写报名表单（家长+儿童信息）→ 提交报名 → 成功toast提示

### 3.3 签到流程
活动当天组织者进入签到页 → 查看所有报名者卡片网格 → 点击卡片切换签到状态 → 状态实时保存到后端

### 3.4 照片上传与展示流程
活动结束组织者进入后台 → 批量上传照片（最多30张，≤8MB）→ 显示上传进度条 → 照片在详情页瀑布流展示 → 点击爱心标注精彩瞬间

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：薄荷绿 `#66bb6a`、浅橙 `#ffb74d`
- 背景色：极浅灰 `#fafafa`
- 卡片渐变：浅绿 `#e8f5e9` 到浅黄 `#fff8e1`
- 按钮：薄荷绿 `#66bb6a`，hover变深 `#43a047`，圆角20px，transition 0.2s
- 字体：系统无衬线字体（-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif）
- 圆角：卡片20px、按钮20px、标签8px、签到卡片12px、照片12px
- 阴影：悬停时 `box-shadow: 0 6px 20px rgba(0,0,0,0.1)`

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 搜索框 | 顶部居中，宽度50%，最小320px，圆角24px，focus边框#42a5f5 |
| 首页 | 活动卡片 | 300x260px，渐变背景，封面图140px高，年龄段标签绿色圆角，报名状态三色标识 |
| 报名表单 | Toast提示 | 280px宽，圆角8px，绿色#4caf50背景，2.5秒自动消失 |
| 报名名单 | 表格 | 表头背景#f5f5f5，行交替白/浅灰#fafafa |
| 签到卡片 | 状态切换 | 180x120px，已签到浅绿#c8e6c9背景，绿色对勾图标，0.3s淡入淡出 |
| 照片瀑布流 | 进度条 | 8px高，圆角4px，背景#e0e0e0，填充#42a5f5到#7c4dff渐变 |
| 照片瀑布流 | 爱心图标 | 空/实心切换，缩放1.2倍回弹效果 |

### 4.3 响应式设计
- 桌面端：最大宽度1200px居中，活动卡片多列网格
- 移动端（<768px）：卡片单列布局，搜索框宽度95%，导航栏汉堡菜单
- 触摸优化：按钮最小高度44px，卡片点击区域充足

### 4.4 交互动效
- 活动卡片悬停：上移4px + 阴影，transition 0.2s ease
- 签到状态切换：0.3s淡入淡出
- 照片悬停：放大1.05倍 + 阴影遮罩，transition 0.3s ease
- 爱心图标：点击时缩放1.2倍再回弹
- 页面元素：加载时错位渐入动画

## 5. 性能要求
- 活动列表首次加载 ≤1.5秒（含图片渲染）
- 照片上传后首批可见照片渲染延迟 ≤800ms
- 图片懒加载优化
- API请求统一错误处理
