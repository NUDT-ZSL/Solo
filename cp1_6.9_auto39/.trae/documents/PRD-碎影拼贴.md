
# 碎影拼贴 - 产品需求文档 (PRD)

## 1. 产品概述

「碎影拼贴」是一款面向拼贴艺术爱好者的浏览器端数字创作工具，用户可上传多张图片碎片，通过拖拽、旋转、缩放自由组合在画布上，每张碎片带有随机撕裂纹理与半透明阴影，形成具有手工质感的数字拼贴作品。

- **目标用户**：热爱手工拼贴艺术、数字艺术创作的设计师、学生及创意人群
- **核心价值**：在浏览器中零门槛复现手工撕纸拼贴的独特质感，无需安装软件，所见即所得

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，直接使用 | 上传图片、编辑碎片、导出作品 |

### 2.2 功能模块

1. **主画布页面**：
   - 多图上传模块（拖拽上传 + 点击上传）
   - 碎片交互模块（拖拽移动 / 滚轮缩放 / Alt+拖拽旋转）
   - 撕裂纹理渲染模块（贝塞尔曲线边缘 + 半透明阴影）
   - 图层管理模块（选中上浮 / 键盘调整层级）
   - 导出模块（2x分辨率 PNG 导出）
   - 工具栏模块（上传按钮 + 数量显示 + 导出按钮）

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主画布页面 | 多图上传模块 | 支持拖拽5-20张png/jpg（单张≤3MB），自动缩放至最大边长200px并保持比例，提取像素数据 |
| 主画布页面 | 碎片交互模块 | 鼠标拖拽移动、滚轮缩放(0.3x-2.0x)、Alt+拖拽旋转(0-360°)，实时响应≤16ms |
| 主画布页面 | 撕裂纹理渲染模块 | 贝塞尔曲线生成不规则撕裂轮廓(波动5-15px随机)，深灰色半透明阴影边缘(rgba(0,0,0,0.3)) |
| 主画布页面 | 图层管理模块 | 点击选中上浮到顶层，显示控制边框(深棕色#5A3E2B)+金色旋转手柄(#D4A84B)；[键下移一层、]键上移一层 |
| 主画布页面 | 导出模块 | 导出1024×768高清PNG(2x分辨率)，支持下载到本地 |
| 主画布页面 | 工具栏模块 | 左上角暗色半透明工具栏(rgba(0,0,0,0.2))，圆形上传按钮(悬停放大1.1倍+变色)，碎片数量显示；右上角导出按钮 |
| 主画布页面 | 视觉动效模块 | 选中光晕呼吸动画(透明度0.3↔0.6，周期2s)，米白色(#F5F0E8)纸张质感背景 |

## 3. 核心流程

### 3.1 创作主流程

用户打开页面 → 点击或拖拽上传5-20张图片 → 系统自动缩放并生成撕裂纹理 → 用户拖拽/旋转/缩放调整碎片位置 → 点击选中调整图层顺序 → 点击导出按钮 → 系统合成2x分辨率PNG → 浏览器触发下载

```mermaid

    A["用户进入页面"] --> B["上传图片<br/>(拖拽/点击 5-20张)"]
    B --> C["系统预处理<br/>缩放至200px + 提取像素"]
    C --> D["生成随机撕裂纹理<br/>贝塞尔曲线 + 阴影"]
    D --> E["碎片渲染到画布"]
    E --> F{"用户是否操作碎片?"}
    F -->|拖拽移动| G["更新位置坐标"]
    F -->|滚轮缩放| H["更新缩放系数<br/>(0.3x-2.0x)"]
    F -->|Alt+拖拽旋转| I["更新旋转角度<br/>(0-360°)"]
    F -->|点击选中| J["上浮顶层<br/>显示控制边框"]
    F -->|[/]键调整层级| K["调整z-index顺序"]
    G & H & I & J & K --> L["触发画布重绘"]
    L --> F
    F -->|点击导出| M["合成1024×768 2x PNG"]
    M --> N["浏览器下载保存"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米白色背景 `#F5F0E8`（旧纸张质感）
- **强调色**：深棕色控制边框 `#5A3E2B`、金色旋转手柄 `#D4A84B`
- **工具栏**：半透明暗色背景 `rgba(0,0,0,0.2)`，磨砂玻璃质感
- **按钮风格**：圆形上传按钮，悬停时 `transform: scale(1.1)` + 暖色调变化
- **字体**：标题使用手写风格字体（如 Ma Shan Zheng / Caveat），正文使用优雅衬线体（如 Noto Serif SC / Lora）
- **布局风格**：全屏沉浸式画布，工具栏浮动式布局，画布无边界感
- **图标风格**：线性极简图标，匹配纸张手工质感

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主画布页面 | 全屏画布 | 米白色背景 `#F5F0E8`，100vw×100vh，模拟纸张纹理的微弱噪点叠层 |
| 主画布页面 | 左上工具栏 | 半透明暗色浮层 `rgba(0,0,0,0.2)`，圆角12px，内边距16px；圆形上传按钮(φ44px)+碎片数量徽标 |
| 主画布页面 | 右上导出按钮 | 金色边框 `#D4A84B`，半透明米白背景，文字"导出 PNG"，悬停时金色填充 |
| 主画布页面 | 碎片控制边框 | 深棕色 `#5A3E2B` 虚线边框，2px宽，1px间距；旋转手柄φ10px金色圆点 |
| 主画布页面 | 选中光晕 | 碎片外缘径向渐变光晕，透明度0.3→0.6循环2s |
| 主画布页面 | 撕裂纹理 | 贝塞尔曲线不规则边缘，外缘8px宽深灰半透明阴影 `rgba(0,0,0,0.3)` |

### 4.3 响应式设计

- **桌面端（默认）**：工具栏左上角浮动，画布占满全屏
- **平板（<900px）**：工具栏缩小，按钮尺寸按比例调整至85%
- **移动端（<600px）**：工具栏改为底部横条布局（宽度96%，水平居中），控制手柄缩小至60%（φ6px），上传按钮调整为φ36px

### 4.4 性能指标

| 指标 | 目标值 | 测试条件 |
|------|--------|----------|
| 渲染帧率 | ≥55 FPS | 画布同时存在20张碎片 |
| 交互响应延迟 | ≤16ms | 拖拽、旋转、缩放操作 |
| 输入延迟 | ≤10ms | 鼠标操作跟随 |
| 首次加载 | ≤2s | 空画布状态 |
