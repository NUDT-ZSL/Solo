## 1. 产品概述

独立游戏工坊——一个面向独立游戏开发者和爱好者的作品集展示与社区评分解锁平台。玩家可浏览游戏项目卡片，查看截图、视频预告和开发日志，对游戏进行1-5星评分；当评分累计达到100分阈值时，解锁专属幕后彩蛋（概念图、开发者访谈片段、隐藏关卡小样）。

- 目标用户：独立游戏开发者（展示作品）、游戏爱好者（浏览与评分）
- 核心价值：通过社区评分驱动的解锁机制，为优质独立游戏提供曝光，同时让玩家获得深度幕后内容

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览游戏卡片、查看详情、评分、触发解锁 |

### 2.2 功能模块
1. **主页面**：瀑布流游戏卡片列表、分类筛选栏、评分排序
2. **详情页**：游戏截图轮播、项目信息、开发日志时间线、解锁弹窗

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 筛选栏 | 按类型筛选（全部、动作、解谜、模拟、角色扮演），选中高亮 |
| 主页面 | 瀑布流卡片区域 | 280px宽卡片，渐变背景，悬停上浮，星标评分交互 |
| 主页面 | 评分排序 | 卡片按评分从高到低排序 |
| 详情页 | 截图轮播 | 左侧水平滑动轮播，0.5秒切换动画 |
| 详情页 | 项目信息 | 开发者名、发布时间、平台标签（绿色圆角矩形） |
| 详情页 | 开发日志时间线 | 左侧圆点标记，最新实心红/过往空心灰，展开动画，点赞心形 |
| 详情页 | 解锁弹窗 | 评分达100分触发，展示概念图淡入切换+语音访谈播放按钮+分享链接 |

## 3. 核心流程

用户打开主页 → 浏览瀑布流卡片 → 点击筛选按钮切换分类 → 悬停卡片查看效果 → 点击星标评分 → 卡片按评分重新排序 → 点击卡片进入详情页 → 浏览截图轮播 → 阅读开发日志并点赞 → 评分累计达100分 → 解锁弹窗出现 → 查看幕后彩蛋

## 4. 用户界面设计

### 4.1 设计风格
- **主色**：#1E1E2E（深色背景）
- **强调色**：#E74C3C（红色高亮、选中态）
- **辅助色**：#F1C40F（金色星标）
- **卡片渐变**：#2C3E50 → #34495E
- **按钮风格**：圆角20px胶囊按钮，选中#E74C3C，未选中#7F8C8D
- **字体**：系统无衬线字体（-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif）
- **布局**：瀑布流卡片布局，响应式列数（1-4列）
- **图标**：lucide-react图标库

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 筛选栏 | 胶囊按钮横排，选中红/未选中灰，圆角20px |
| 主页面 | 游戏卡片 | 280px宽，渐变背景，16px圆角，3:2缩略图淡入，五星评分，悬停Y轴-8px上浮+阴影12px |
| 详情页 | 截图轮播 | 水平滑动切换，0.5秒过渡 |
| 详情页 | 平台标签 | 圆角矩形，#27AE60背景，白色文字 |
| 详情页 | 日志时间线 | 左侧圆点标记，纵向展开0.3秒，心形点赞#E74C3C |
| 详情页 | 解锁弹窗 | 圆角20px，背景#1A1A2E，边框#FFD700 2px，概念图淡入+播放按钮 |

### 4.3 响应式设计
- 桌面端（1200px+）：4列瀑布流
- 平板端（768px-1199px）：2-3列瀑布流
- 手机端（320px-767px）：1列瀑布流，卡片全宽
- 所有交互支持触摸操作

### 4.4 动画性能
- 所有动画保证30FPS以上
- 统一过渡动画0.2-0.5秒
- 图片加载淡入0.4秒
- 星标填充0.2秒
- 卡片悬停0.3秒
- 日志展开0.3秒
