## 1. 产品概述

社区菜园协作管理应用，帮助菜园成员协作管理作物种植、浇水、观察记录和任务分配，解决成员忘记浇水、不了解蔬菜成熟时间等问题。

- 目标用户：社区菜园的管理者和普通成员
- 核心价值：通过可视化的区域卡片、浇水冷却机制和积分激励，提升菜园协作效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理者 | 登录时指定 | 浏览菜园、浇水、记录日志、分配任务、管理成员 |
| 普通成员 | 登录时指定 | 浏览菜园、浇水、记录日志、查看和完成任务 |

### 2.2 功能模块

1. **登录页**：模拟用户登录，选择身份（管理者/成员）
2. **菜园概览页**：展示菜园各区域卡片网格，显示作物状态和倒计时
3. **区域详情页**：查看单区域作物详情、浇水操作、观察日志、任务列表
4. **个人资料页**：展示成员积分和等级进度条

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 登录页 | 登录表单 | 用户名输入、角色选择（管理者/成员）、登录按钮，模拟认证 |
| 菜园概览页 | 区域卡片网格 | 12列响应式网格展示各区域卡片，卡片显示区域名、作物名、种植日期、预计收获日期、浇水状态；卡片背景根据作物状态变色（绿/黄/红） |
| 菜园概览页 | 顶部导航栏 | 固定顶部，深绿色背景，显示菜园名称、用户头像、积分 |
| 区域详情页 | 作物信息面板 | 展示作物名称、种植日期、预计收获日期、收获倒计时 |
| 区域详情页 | 浇水按钮 | 含冷却倒计时（36小时冷却），点击触发水滴动画，冷却中显示剩余时间 |
| 区域详情页 | 观察日志 | 添加文字日志+照片上传（128x128预览），日志按时间倒序排列，左图右文 |
| 区域详情页 | 任务分配 | 管理者可选择成员分配任务（除草/施肥/收割），成员可完成领取积分 |
| 个人资料页 | 积分进度条 | 展示当前积分、等级、进度条（颜色渐变#8BC34A→#4CAF50→#388E3C） |

## 3. 核心流程

**用户登录流程**：用户输入用户名 → 选择角色 → 调用/api/login → 获取token和用户信息 → 跳转菜园概览

**浇水流程**：用户点击浇水按钮 → gardenLogic.ts校验冷却时间 → 若可浇则调用/api/water → 触发水滴动画 → 更新卡片状态

**观察日志流程**：用户输入文字+上传照片 → 调用gardenLogic.ts计算积分 → 保存日志 → 更新区域健康状态

**任务流程**：管理者选择成员 → 分配任务 → 成员点击完成 → gardenLogic.ts计算积分（+50） → 更新成员积分

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：绿色系自然园艺风格
  - 页面背景：#F1F8E9
  - 卡片背景：#FFFFFF
  - 标题色：#33691E（加粗）
  - 正文色：#4E342E
  - 导航栏背景：#2E7D32
- 按钮：浇水按钮蓝色渐变#64B5F6→#1E88E5，悬停放大1.05倍+2px发光
- 卡片：圆角12px，阴影4px rgba(0,0,0,0.08)，悬停边框#66BB6A+上浮6px（transition 0.3s ease）
- 布局：12列响应式网格（小屏单列、中屏两列、大屏三列）
- 表单：焦点边框#81C784，0.3秒过渡

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录页 | 登录表单 | 居中卡片，自然绿色背景，圆角输入框，绿色渐变登录按钮 |
| 菜园概览页 | 导航栏 | 固定顶部，深绿色#2E7D32，白色文字，右侧用户头像+积分 |
| 菜园概览页 | 区域卡片 | 白色卡片，圆角12px，状态色边框，悬停上浮，网格布局 |
| 区域详情页 | 浇水区域 | 蓝色渐变按钮，水滴动画容器，倒计时文字 |
| 区域详情页 | 日志列表 | 左图右文卡片，128x128缩略图，时间戳 |
| 区域详情页 | 任务列表 | 任务卡片，成员头像，完成按钮，积分标记 |
| 个人资料页 | 积分条 | 渐变进度条，等级标签，当前积分数字 |

### 4.3 响应式设计

- 桌面优先设计
- 大屏(≥1024px)：3列网格
- 中屏(768px-1023px)：2列网格
- 小屏(<768px)：单列，导航栏简化
- 触摸优化：按钮最小44px点击区域

### 4.4 动画效果

- 水滴动画：3个蓝色#42A5F5水滴从上往下掉落，大小递增，每滴0.3秒下落，间隔0.15秒
- 卡片悬停：边框变#66BB6A，上浮6px，0.3s ease
- 按钮悬停：放大1.05倍，2px发光
- 页面切换：≤300ms过渡
