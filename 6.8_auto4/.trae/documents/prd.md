## 1. 产品概述

一款2D平台跳跃小游戏，主角在随机生成的空中岛屿群上跳跃探险，收集星星并避开敌人。使用 Phaser 3 + TypeScript + Vite 构建，面向休闲玩家和移动端用户。

## 2. 核心功能

### 2.2 功能模块

1. **菜单场景**: 动态云朵背景、渐变天空、开始按钮、淡入转场
2. **游戏场景**: 随机岛屿生成、主角控制、敌人巡逻、星星收集、碰撞检测、粒子特效

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 菜单场景 | 动态背景 | 渐变天空 + 飘浮云朵动画 |
| 菜单场景 | 开始按钮 | 点击后淡入过渡到游戏场景 |
| 游戏场景 | 岛屿生成 | 随机生成不同尺寸绿色平台，布局确保可到达 |
| 游戏场景 | 主角控制 | 左右移动、跳跃、弹性动画 |
| 游戏场景 | 敌人系统 | 锯齿状敌人在岛屿上来回移动，对象池管理 |
| 游戏场景 | 星星收集 | 收集星星弹出粒子特效 + 得分文字 |
| 游戏场景 | 受伤反馈 | 主角闪红 + 短暂无敌 |
| 游戏场景 | 掉落重生 | 主角掉出屏幕重新来 |
| 游戏场景 | 移动端适配 | 触摸跳跃 + 左右按钮，响应式屏幕适配 |

## 3. 核心流程

用户进入菜单场景 → 点击开始 → 淡入过渡到游戏场景 → 主角在岛屿间跳跃 → 收集星星得分 → 碰到敌人受伤（闪红+无敌）→ 掉出屏幕重新开始


## 4. 用户界面设计

### 4.1 设计风格

- 主色系：糖果色系（粉蓝 #7EC8E3、亮绿 #7ED957、暖黄 #FFD93D）
- 辅助色：粉红 #FF6B9D、浅紫 #C9A0DC
- 背景风格：渐变天空（从暖黄到粉蓝过渡）
- 岛屿风格：圆润绿色平台，不同尺寸
- 敌人风格：简单锯齿状造型
- 按钮风格：圆角、带阴影、糖果色
- 字体：圆润可爱风格
- 动画：弹性跳跃动画、云朵飘浮、星星闪光

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 菜单场景 | 背景 | 渐变天空 + 飘浮云朵动画 |
| 菜单场景 | 标题 | 大号圆润字体，带轻微浮动动画 |
| 菜单场景 | 开始按钮 | 圆角矩形，暖黄色，悬停放大 |
| 游戏场景 | HUD | 左上角星星计数，右上角生命值 |
| 游戏场景 | 主角 | 圆润方块角色，跳跃时弹性缩放 |
| 游戏场景 | 岛屿 | 绿色圆角平台，不同宽度 |
| 游戏场景 | 敌人 | 锯齿状红色造型，来回移动 |
| 游戏场景 | 星星 | 五角星形，闪光旋转动画 |
| 游戏场景 | 移动端控制 | 左下方向按钮 + 右下跳跃按钮 |

### 4.3 响应式

- 桌面端：键盘控制（方向键/WASD + 空格跳跃）
- 移动端：触摸按钮控制，自动适配屏幕尺寸
- 游戏画布自动缩放适配不同分辨率
