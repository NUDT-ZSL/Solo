## 1. 产品概述

「镜像迷宫」是一款2D解谜游戏：玩家控制发光小球在由多面镜子组成的动态迷宫中探索，通过旋转镜子改变激光路径来解锁出口。整体视觉走赛博水晶风格，追求流畅的60fps交互体验。
- 核心目标：为玩家提供富有策略性和视觉冲击力的镜面反射解谜体验
- 目标用户：解谜游戏爱好者、独立游戏玩家、视觉美学追求者

## 2. 核心功能

### 2.1 功能模块

1. **游戏主界面**：迷宫场景渲染、HUD信息展示、小地图
2. **游戏交互**：玩家移动、镜子旋转、激光路径实时变化
3. **粒子特效系统**：拖尾、光斑爆散、金色光晕、炫光过渡

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主界面 | 迷宫场景 | 渲染赛博水晶风格迷宫，包括墙壁、镜子、激光、出口 |
| 游戏主界面 | 玩家控制 | WASD移动发光小球，鼠标拖拽旋转镜子 |
| 游戏主界面 | HUD-左上 | 显示步数和收集的水晶碎片数量 |
| 游戏主界面 | HUD-右上 | 毛玻璃小地图，显示当前区域和激光路径 |
| 游戏主界面 | HUD-底部 | 操作提示（WASD移动，鼠标拖拽旋转镜子） |
| 游戏主界面 | 激光系统 | 红色到金色渐变激光，多段反射，击中墙壁光斑爆散 |
| 游戏主界面 | 特效系统 | 镜子旋转微光脉冲、激光路径变化闪烁、水晶收集金色光晕、出口到达炫光过渡 |

## 3. 核心流程

玩家进入游戏 → 在迷宫中移动探索 → 发现镜子和激光源 → 拖拽旋转镜子改变激光方向 → 激光经过多次反射击中出口 → 出口解锁 → 过渡动画 → 下一关卡

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：纯黑(#000000)到深紫(#1a0033)渐变背景
- **辅助色**：半透明发光晶体蓝(#00ccff)、暖黄色(#ffcc00)、红色到金色渐变激光(#ff3333→#ffd700)
- **质感**：水晶半透明发光、镜面菱形光晕、毛玻璃(glassmorphism)
- **字体**：科技感像素/等宽字体，标题18px，HUD信息14px
- **布局**：全屏Canvas游戏画面，HUD叠加层

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主界面 | 迷宫墙壁 | 半透明发光晶体质感，深蓝到青色渐变边框 |
| 游戏主界面 | 镜子 | 菱形镜面，反射光晕，可拖拽旋转 |
| 游戏主界面 | 玩家小球 | 暖黄色发光球体，粒子拖尾效果 |
| 游戏主界面 | 激光 | 红色到金色渐变细光束，击中墙壁光斑爆散 |
| 游戏主界面 | 左上HUD | 步数计数器、水晶碎片图标+数量 |
| 游戏主界面 | 右上小地图 | 毛玻璃背景，显示区域缩略图和激光路径 |
| 游戏主界面 | 底部提示 | 半透明条，WASD/鼠标操作说明 |

### 4.3 响应式

- 桌面优先设计，Canvas自适应窗口大小
- HUD元素使用相对定位，适应不同分辨率

### 4.4 交互反馈

- 旋转镜子：屏幕边缘微光脉冲
- 激光路径变化：闪烁动画
- 收集水晶碎片：金色光晕+粒子上升效果
- 到达出口：全屏炫光过渡动画
