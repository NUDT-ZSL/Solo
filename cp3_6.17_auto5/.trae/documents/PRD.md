## 1. 产品概述

本产品是一个基于知识图谱的课程知识点关联与复习路径推荐应用，旨在解决在线教育学员课程知识点碎片化、复习效果不佳的问题。通过可视化知识图谱展示知识点间的依赖关系，并基于学生测评数据智能生成个性化复习路径，帮助学员高效巩固薄弱知识点。

- **目标用户**：教师用户（课程创建与知识点管理）、学生用户（知识图谱浏览与个性化复习）
- **产品价值**：将松散的知识点体系化呈现，通过智能推荐提升复习效率和学习效果

## 2. 核心功能

### 2.1 用户角色
| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 教师用户 | 用户名登录 | 创建课程、添加知识点、建立知识点关系、管理课程内容 |
| 学生用户 | 用户名登录 | 浏览知识图谱、标签过滤、查看测评结果、生成复习路径、完成复习打卡 |

### 2.2 功能模块
1. **图谱页面（MapPage）**：知识图谱可视化展示、标签过滤、节点拖拽、关系连线、复习路径高亮、节点详情弹窗
2. **用户页面（UserPage）**：用户登录注册、角色切换、课程列表、测评管理
3. **知识图谱组件（KnowledgeGraph）**：Canvas绘制、节点渲染、贝塞尔曲线连线、拖拽交互、高亮动画
4. **复习路径推荐Hook（useRecommendPath）**：薄弱点识别、拓扑排序、路径生成算法

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 图谱页面 | 顶部导航栏 | 显示课程名称、标签过滤下拉框、用户信息 |
| 图谱页面 | 图谱画布区域 | Canvas绘制知识点节点和关系连线，支持拖拽移动节点、拖拽创建关系、点击查看详情 |
| 图谱页面 | 右侧信息面板 | 课程信息展示、复习路径生成按钮、路径导航、复习进度统计 |
| 图谱页面 | 节点详情弹窗 | 展示知识点标题、难度、详情文本、标签，支持"完成复习"操作 |
| 用户页面 | 登录注册模块 | 用户身份认证、角色选择 |
| 用户页面 | 课程管理模块 | 教师创建课程、添加知识点、上传封面 |

## 3. 核心流程

### 3.1 教师课程创建流程
教师登录 → 创建课程（填写标题、简介、封面）→ 添加知识点（标题、详情、难度、标签）→ 在图谱上拖拽建立"前置-后续"关系 → 保存课程

### 3.2 学生复习流程
学生登录 → 选择课程 → 浏览知识图谱（可按标签过滤）→ 完成课后测评 → 点击"生成复习路径" → 系统基于薄弱点和依赖关系生成路径 → 学生沿路径逐一点击复习 → 标记完成后从路径移除

```mermaid
flowchart LR
    A["学生选择课程"] --> B["浏览知识图谱"]
    B --> C["完成课后测评"]
    C --> D["点击生成复习路径"]
    D --> E["DFS拓扑排序 + 薄弱点筛选"]
    E --> F["生成≤5节点的复习路径"]
    F --> G["图谱高亮显示路径"]
    G --> H["点击节点查看详情"]
    H --> I["标记完成复习"]
    I --> J["从路径移除节点"]
    J --> K["继续下一个节点"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝 #1a237e（用于导航栏、按钮、重要文字）
- **辅助色**：青蓝 #00bcd4（链接、图标、交互元素）
- **节点颜色**：初级 #81c784（绿）、中级 #ffb74d（橙）、高级 #e57373（红）
- **高亮色**：红色 #f44336（复习路径虚线）、金色 #ffd700（节点边框闪烁动画）
- **背景色**：白色 #ffffff、浅灰 #f5f5f5（右侧面板）
- **文本色**：深灰 #212121
- **布局风格**：卡片式布局，左侧70%图谱画布，右侧30%信息面板
- **按钮风格**：圆角8px，悬停有轻微阴影和缩放效果
- **字体**：使用现代无衬线字体，标题加粗，正文清晰易读
- **图标风格**：线性图标，简洁明了，与主色调一致

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 图谱页面 | 顶部导航栏 | 高度56px，白色背景，底部1px实线 #e0e0e0，左侧课程名，中间标签过滤下拉框，右侧用户信息 |
| 图谱页面 | 图谱画布 | 白色背景，Canvas绘制圆形节点（半径18px），贝塞尔曲线连线带箭头，节点悬停放大1.2倍（过渡0.2s），过滤动画0.3s ease |
| 图谱页面 | 右侧信息面板 | 浅灰 #f5f5f5 背景，圆角8px，包含课程信息、操作按钮、复习路径列表、进度统计 |
| 图谱页面 | 详情弹窗 | 宽380px，圆角12px，白色背景，阴影 0 8px 32px rgba(0,0,0,0.15)，从右侧滑入动画0.3s ease-out |

### 4.3 响应式设计
- **桌面端（≥1024px）**：左右布局，图谱70%，面板30%
- **平板端（768px-1024px）**：上下堆叠布局，图谱高度60vh，面板占剩余空间
- **交互优化**：触摸设备支持长按拖拽，点击区域足够大（≥44px）

### 4.4 动画效果
- **页面加载**： staggered 元素入场动画
- **节点悬停**：缩放1.2倍 + 轻微投影（0.2s过渡）
- **过滤切换**：未匹配节点渐变为半透明灰色（0.3s ease）
- **详情弹窗**：从右侧滑入（0.3s ease-out）
- **复习路径高亮**：金色边框闪烁动画（2s循环）
- **按钮交互**：悬停缩放1.03倍，点击反馈
