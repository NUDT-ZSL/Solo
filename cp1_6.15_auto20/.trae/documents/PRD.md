## 1. 产品概述

在线乐谱标记与分享应用，让用户能够创建、编辑和分享简谱格式的音乐乐谱，支持演奏标记和实时试听。面向音乐爱好者、音乐教师和学生，提供直观的简谱编辑体验和社区分享功能。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 创建/编辑/删除乐谱、播放试听、浏览社区乐谱、收藏 |

### 2.2 功能模块

1. **新建乐谱页**: 乐谱编辑器，五线网格编辑、音符填入、播放预览、撤销重做
2. **我的乐谱页**: 乐谱列表，卡片展示、搜索排序、删除管理
3. **社区乐谱页**: 社区乐谱浏览、只读预览、收藏功能

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 新建乐谱 | 五线网格编辑器 | 可编辑1-7音符（含升号#），最多16小节×4拍，支持点击/拖拽填入，右键删除，长按修改音长八度 |
| 新建乐谱 | 播放预览 | 使用tone.js合成音频，从左到右播放，当前音符高亮（橙色+0.15s脉冲），60fps |
| 新建乐谱 | 撤销重做 | 最多20步，Ctrl+Z/Ctrl+Shift+Z快捷键，0.15s闪烁过渡 |
| 新建乐谱 | 保存功能 | 保存到LocalStorage，UUID标识，成功后顶部绿色提示条（2s滑入消失） |
| 我的乐谱 | 卡片列表 | 卡片网格展示标题、创建时间、前八小节缩略图 |
| 我的乐谱 | 搜索排序 | 关键词搜索（0.2s淡入），按创建时间排序 |
| 我的乐谱 | 删除确认 | 确认对话框（0.2s淡入），确认后卡片0.3s缩放消失 |
| 社区乐谱 | 示例乐谱 | 10个预设示例乐谱，卡片列表展示 |
| 社区乐谱 | 只读预览 | 点击卡片打开预览，音符不可编辑但可播放 |
| 社区乐谱 | 收藏功能 | 心形图标点击变红，0.2s放大回弹动画 |

## 3. 核心流程

用户打开应用 → 选择"新建乐谱"标签 → 在五线网格上点击/拖拽填入音符 → 选择音长和八度 → 点击播放试听 → 满意后保存 → 切换到"我的乐谱"查看已保存列表 → 可搜索/排序/删除 → 切换到"社区乐谱"浏览示例 → 点击预览并收藏

## 4. 用户界面设计

### 4.1 设计风格

- 主色调: #e94560（玫红色），辅助色: #0f3460（蓝紫色）
- 背景: #1a1a2e（深色），卡片: #16213e
- 按钮: 圆角矩形，播放按钮渐变色(#e94560→#0f3460)
- 字体: 系统无衬线字体
- 布局: 顶部导航+内容区，最大宽度1200px居中
- 图标: lucide-react图标库

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 全局 | 顶部导航栏 | 毛玻璃效果(backdrop-filter:blur(10px))，三个标签，下划线滑动动画(0.2s) |
| 新建乐谱 | 五线网格 | 浅色虚线背景(#2a2a4e)，白色圆角矩形音符，选中边框#e94560 |
| 新建乐谱 | 播放按钮 | 渐变色(#e94560→#0f3460)，悬停光晕扩散动画 |
| 新建乐谱 | 提示条 | 绿色，从顶部滑入，2s后消失 |
| 新建乐谱 | 撤销重做图标 | 左上角，显示可撤销/重做步数 |
| 我的乐谱 | 卡片 | 圆角12px，阴影0 4px 12px rgba(0,0,0,0.3)，悬停上移2px+阴影加深 |
| 我的乐谱 | 确认对话框 | 透明背景0.2s淡入 |
| 社区乐谱 | 收藏按钮 | 心形图标，填充红色+0.2s放大回弹 |
| 社区乐谱 | 只读预览 | 覆盖层展示，音符不可点击 |

### 4.3 响应式

- 桌面端优先（1024px-1920px）
- 内容区域最大宽度1200px居中
- 超出部分显示滚动条

### 4.4 动画规范

- 标签切换: 内容区0.3s水平滑入
- 搜索结果: 0.2s淡入
- 删除确认: 0.2s透明背景淡入
- 卡片删除: 0.3s缩放消失
- 播放高亮: 橙色背景+0.15s亮度脉冲
- 撤销重做: 0.15s闪烁过渡
- 导航下划线: 0.2s滑动
- 卡片悬停: 0.2s上移+阴影加深
