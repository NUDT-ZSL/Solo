# AeroScope 三维城市空气质量时空可视化应用 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 背景
环境科学家在分析城市空气质量监测数据时，发现传统的二维图表难以直观呈现污染物浓度的空间分布与时间变化趋势，需要更具沉浸感的可视化手段。

### 1.2 产品定位
AeroScope 是一款面向环境科研人员的三维城市空气质量时空可视化应用，通过三维地图、粒子动画和交互式图表，直观呈现城市监测站点的污染物浓度分布、时间变化趋势及空气流动状况。

### 1.3 目标用户
- 环境科学家与研究员
- 城市规划与环境监测部门
- 大气污染研究相关学者

---

## 2. 核心功能需求

### 2.1 三维场景可视化（F1）
- **功能描述**：在三维空间中渲染城市监测区域，以柱状图形式直观展示各站点污染物浓度
- **详细规格**：
  - 300×300 单位的矩形区域作为地图基座
  - 半透明网格地面（浅灰#e2e8f0，网格线#cbd5e1）
  - 每个站点：圆形基座（半径2单位，#64748b）+ 垂直柱体 + 顶部半透明小球
  - 柱体高度：浓度0-300映射到高度0-50单位
  - 柱体颜色：低浓度浅绿#22c55e → 高浓度深红#dc2626 插值过渡

### 2.2 时间回溯与动画（F2）
- **功能描述**：通过时间滑块查看任意小时的历史数据，支持自动播放
- **详细规格**：
  - 水平时间滑块：范围0-23小时，步长1小时，宽度300px
  - 轨道颜色#475569，滑块圆形#3b82f6
  - 数据过渡动画：500ms，easeInOut缓动
  - 自动播放：每1.5秒前进1小时，播放按钮可切换为暂停

### 2.3 详情弹窗与图表（F3）
- **功能描述**：点击站点显示该站点的详细信息和24小时污染物变化曲线
- **详细规格**：
  - 浮动信息卡片：400×320px，背景#1e293b，圆角12px，阴影0 8px 32px rgba(0,0,0,0.4)
  - 卡片顶部：站点名称 + 当前时间
  - 卡片中部：Chart.js折线图展示24小时4种污染物（NO2、PM2.5、PM10、O3）变化，数据点显示具体数值，图表背景#0f172a
  - 卡片底部：风向（箭头图标）+ 风速数值

### 2.4 污染物筛选与全局联动（F4）
- **功能描述**：通过Toggle按钮切换显示不同污染物，3D场景与图表实时联动
- **详细规格**：
  - 4个Toggle按钮：PM2.5、PM10、O3、NO2，默认全部选中
  - 选中状态：填充色#3b82f6，文字白色
  - 未选中状态：背景#1e293b，文字#94a3b8
  - 按钮圆角8px，间距8px
  - 取消选中：3D场景柱子切换为该污染物浓度显示，图表隐藏对应曲线

### 2.5 风向粒子系统（F5）
- **功能描述**：通过流动粒子直观展示空气流动方向
- **详细规格**：
  - 500个粒子，大小0.05单位，颜色#94a3b8透明度0.3
  - 粒子沿风向飘移，速度2单位/秒
  - 风向随当前选定时间点变化

## 3. UI/UX 设计规范

### 3.1 整体风格
- **主题**：深色科技风格
- **主背景**：#0f172a
- **辅助背景**：#1e293b
- **文字主色**：#f1f5f9
- **强调色**：#3b82f6（蓝）、#8b5cf6（紫）

### 3.2 左侧UI面板
- 固定宽度320px
- 背景：#0f172a + backdrop-blur模糊效果
- 圆角16px，内边距20px
- 悬浮于场景左上角
- 包含：站点列表、时间滑块、污染物筛选Toggle组

### 3.3 站点列表样式
- 条目圆角8px
- Hover：背景#1e293b，translateY(-2px)，阴影加深
- 选中：背景#1e293b + 左侧3px蓝色#3b82f6边线

### 3.4 右上角信息面板
- 半透明背景#1e293b，圆角8px
- 显示：当前模拟时间、污染物综合指数（4种污染物归一化平均值）

### 3.5 全局动效
- 所有UI组件过渡平滑
- 悬停效果：translateY(-2px) + 阴影加深

## 4. 性能要求
| 指标 | 要求 |
|------|------|
| 3D场景渲染帧率 | ≥ 30 FPS |
| 时间滑块响应时间 | ≤ 100ms |
| 图表重绘时间 | ≤ 200ms |
