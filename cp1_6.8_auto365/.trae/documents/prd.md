## 1. 产品概述

「回声胶囊」是一个匿名语音社交 Web 应用——用户录制 15 秒语音留言，系统随机匹配另一位在线用户进行三轮语音对话，对话结束后生成加密的「回声胶囊」链接，仅当双方均点击同意后方可解锁收听完整对话。
- 目标用户：追求匿名社交体验、喜欢声音互动的年轻互联网用户
- 核心价值：用声音建立短暂而神秘的连接，让匿名对话拥有"时间胶囊"般的仪式感

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 匿名用户 | 无需注册，进入即用 | 匹配、录音、对话、胶囊解锁 |

### 2.2 功能模块

1. **首页（匹配页）**：匹配按钮、粒子动画、在线状态
2. **对话房间**：语音气泡、录音控制、倒计时、轮次状态
3. **胶囊页**：3D 胶囊展示、双方同意解锁、完整对话回放

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 匹配按钮 | 中央大圆形按钮，点击触发粒子汇聚动画并发起匹配请求 |
| 首页 | 漂浮粒子 | 半透明发光胶囊形状粒子在背景中漂浮 |
| 首页 | 匹配成功提示 | 显示对方匿名头像（随机渐变色圆形）和"连接成功"脉冲光效 |
| 对话房间 | 语音气泡 | 自己的气泡偏右蓝色毛玻璃，对方气泡偏左紫色毛玻璃 |
| 对话房间 | 录音按钮 | 圆形按钮，录音时脉动光环，顶部 15 秒倒计时进度条 |
| 对话房间 | 轮次状态 | 显示当前第几轮（共三轮），等待/录制/播放状态 |
| 对话房间 | 波形预览 | 录音时实时显示音频波形 |
| 胶囊页 | 3D 胶囊模型 | CSS 3D 或轻量模型旋转发光胶囊 |
| 胶囊页 | 解锁控制 | "解开需双方同意"提示 + 勾选同意按钮 |
| 胶囊页 | 对话回放 | 解锁后可收听完整三轮语音对话 |

## 3. 核心流程

用户进入首页 → 点击匹配按钮 → 粒子汇聚动画 → WebSocket 加入在线队列 → 系统随机匹配 → 显示匹配成功 → 进入对话房间 → 双方轮流录音（15秒/条，共三轮）→ 三轮结束 → 系统生成加密胶囊链接 → 双方进入胶囊页 → 各自点击"同意解锁"→ 双方均同意后解锁 → 收听完整对话

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝 (#0a0e27) → 墨黑 (#020408) 渐变背景
- 强调色：青蓝 (#00d4ff)、电紫 (#8b5cf6)
- 按钮：圆形，发光边缘，3D 玻璃质感
- 字体：展示字体选用 "Orbitron"（科技感），正文用 "Noto Sans SC"
- 布局：全屏沉浸式，居中对齐
- 动画：粒子漂浮、脉冲光效、毛玻璃卡片、3D 旋转胶囊

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 背景 | 深蓝→墨黑线性渐变，CSS 星点散布 |
| 首页 | 粒子系统 | Canvas 渲染半透明发光胶囊形状粒子，60fps 漂浮 |
| 首页 | 匹配按钮 | 120px 圆形，青蓝发光边缘，hover 放大 + 光晕增强 |
| 首页 | 匹配成功 | 随机渐变色圆形头像 + "连接成功"脉冲扩散动画 |
| 对话房间 | 顶部状态栏 | 轮次指示器（3个圆点），当前轮高亮 |
| 对话房间 | 录音区 | 80px 圆形录音按钮，录音态红色脉动环，15秒倒计时弧形进度条 |
| 对话房间 | 语音气泡 | 毛玻璃卡片(backdrop-blur)，己方右偏蓝色，对方左偏紫色 |
| 对话房间 | 波形预览 | Canvas 实时波形，青蓝色渐变 |
| 胶囊页 | 3D胶囊 | CSS 3D transform 旋转胶囊体，青蓝→紫色渐变光晕 |
| 胶囊页 | 解锁区 | 半透明勾选框 + "同意解锁"按钮，双方状态指示灯 |

### 4.3 响应式设计

- 桌面优先设计（1920×1080 基准）
- 移动端自适应：按钮和气泡尺寸缩放，布局纵向排列
- 触屏优化：录音按钮触摸区域 ≥ 44px

### 4.4 3D 场景指引

- 环境：深色太空感背景，微弱星点粒子
- 光照：胶囊体自发光 + 环境光，青蓝色点光源
- 相机：正前方固定视角，胶囊缓慢 Y 轴旋转
- 交互：鼠标悬停胶囊轻微浮动，解锁时光效爆发
- 性能：CSS 3D transform 优先，避免重渲染；60fps 锁定
