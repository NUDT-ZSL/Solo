## 1. 产品概述

「诗影随行」是一个在线诗歌分享与可视化平台，用户可以创作或上传短诗，系统根据诗句的情感倾向和关键词自动生成抽象的动态插画，让诗歌从文字升华为视听体验。
- 目标用户：诗歌爱好者、文艺创作者、追求诗意表达的年轻人
- 核心价值：将诗歌与视觉艺术融合，用动态插画赋予文字生命力，以「回声」机制构建诗意社交

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 前端注册（localStorage模拟） | 浏览诗歌、创作诗歌、添加回声评论 |
| 未登录访客 | 无 | 仅浏览诗歌列表和详情 |

### 2.2 功能模块

1. **登录/注册页**：用户名+密码简单注册登录，localStorage存储会话
2. **诗歌列表页**：瀑布流布局展示所有诗歌卡片，每张卡片含动态插画Canvas
3. **创作页/弹窗**：创作新诗歌，输入标题、正文（≤200字）、选择情感标签
4. **诗歌详情页**：全屏插画展示、完整诗歌、回声评论列表、添加回声表单

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 登录/注册页 | 注册表单 | 用户名、密码输入，前端验证，localStorage模拟存储 |
| 登录/注册页 | 登录表单 | 用户名、密码验证，成功后跳转诗歌列表 |
| 诗歌列表页 | 顶部导航栏 | Logo「诗影随行」、我的创作、全部诗歌、退出按钮 |
| 诗歌列表页 | 瀑布流诗歌卡片 | 每张卡片含毛玻璃效果、诗歌文字、底部动态插画Canvas |
| 诗歌列表页 | 创作入口 | 浮动按钮，点击打开创作弹窗 |
| 创作弹窗 | 诗歌编辑器 | 标题输入、正文编辑（≤200字计数）、情感标签选择（快乐/悲伤/宁静/愤怒/惊奇） |
| 创作弹窗 | 提交按钮 | 提交后自动生成插画，卡片添加到列表 |
| 诗歌详情页 | 全屏插画 | 诗歌插画全屏展示，动画更丰富 |
| 诗歌详情页 | 诗歌内容 | 标题、正文、情感标签、作者、时间 |
| 诗歌详情页 | 回声评论列表 | 展示所有回声评论，含评论者头像/名、文字、情感emoji |
| 诗歌详情页 | 添加回声表单 | 文字输入（≤200字）、emoji情感选择器、提交后插画闪烁 |

## 3. 核心流程

用户注册登录后进入诗歌列表页，以瀑布流浏览所有诗歌卡片（每张卡片含动态插画），点击卡片进入详情页查看完整诗歌和回声评论，也可以通过创作入口撰写新诗。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米白 (#FAF8F5) 和淡灰 (#E8E6E3) 作为底色
- **强调色**：柔和暖光琥珀 (#C8956C) 用于按钮和高亮，淡赭 (#A67C52) 用于交互状态
- **毛玻璃面板**：`backdrop-filter: blur(16px) saturate(180%)` + 半透明白色背景
- **阴影**：柔和暖光阴影，`box-shadow: 0 8px 32px rgba(200, 149, 108, 0.12)`
- **字体**：正文使用 Noto Serif SC（宋体诗意感），标题使用 ZCOOL XiaoWei（站酷小薇体）
- **布局**：卡片瀑布流布局，顶部固定导航，详情页毛玻璃弹窗覆盖
- **图标**：lucide-react 图标库
- **动画**：页面切换缓动淡入（300ms ease-out），背景飘浮落花粒子，插画Canvas 60fps

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录/注册页 | 表单区域 | 毛玻璃卡片居中，标题用站酷小薇体，输入框淡灰边框，琥珀色提交按钮，淡入动画 |
| 诗歌列表页 | 导航栏 | 固定顶部，半透明背景，Logo左侧，导航项右侧，毛玻璃效果 |
| 诗歌列表页 | 瀑布流区域 | 多列卡片布局，卡片毛玻璃效果，底部Canvas插画，hover轻微上浮+阴影增强 |
| 诗歌列表页 | 创作按钮 | 右下角浮动按钮，琥珀色圆形，hover旋转动画 |
| 诗歌列表页 | 背景粒子 | 全屏Canvas，飘浮落花粒子，缓慢下落+左右飘荡 |
| 创作弹窗 | 编辑表单 | 居中毛玻璃弹窗，标题/正文输入，情感标签彩色圆点选择，字数计数器 |
| 诗歌详情页 | 全屏插画 | Canvas占满背景，诗歌文字覆盖其上，毛玻璃文字容器 |
| 诗歌详情页 | 回声评论 | 列表式布局，每条评论左侧emoji，右侧文字，毛玻璃卡片样式 |
| 诗歌详情页 | 添加回声 | 底部固定输入区域，emoji选择器弹出面板，提交按钮 |

### 4.3 响应式适配

- 桌面端（≥1024px）：瀑布流4列，详情页左右分栏（插画/文字+评论）
- 平板端（768px-1023px）：瀑布流2列，详情页上下布局
- 移动端（<768px）：瀑布流1列，简化动画粒子数量

### 4.4 性能要求

- 插画动画帧率保持60fps
- 瀑布流卡片懒加载，仅可视区域Canvas绘制
- 背景粒子数量根据设备性能动态调整
- 页面切换使用CSS transform动画，避免布局抖动
