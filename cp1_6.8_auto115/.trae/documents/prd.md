## 1. 产品概述

「星图占卜」是一款基于 Canvas + React 的交互式星座探索工具，用户可在深色星空背景上通过鼠标悬停和点击，探索全部88个现代星座的轮廓、亮星和神话故事。
- 面向天文爱好者、占星学爱好者及普通用户，提供沉浸式星座知识浏览体验
- 以神秘优雅的占星学视觉调性，打造差异化的星座探索产品

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览全部星座、搜索、切换显示模式 |

### 2.2 功能模块

1. **星图主场景页**：全屏 Canvas 星空背景，88星座连线轮廓与亮星渲染，悬停高亮与点击交互
2. **信息面板**：星座详情弹出面板，含中英文名、亮星名称、神话故事
3. **控制栏**：搜索框、可见性切换、重置视角、响应式折叠

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 星图主场景 | Canvas 星空渲染 | 径向渐变背景、随机微弱星光粒子、极慢自转动画（30秒一圈） |
| 星图主场景 | 星座连线与亮星 | 淡蓝色连线轮廓、2-3等亮星带光晕渲染 |
| 星图主场景 | 悬停高亮交互 | 悬停星座连线/亮星区域时高亮（亮黄色连线、星点放大闪烁），其余星座变暗 |
| 星图主场景 | 点击选中交互 | 点击高亮星座弹出信息面板，再次点击或点击空白关闭 |
| 信息面板 | 毛玻璃弹出面板 | 右下角半透明毛玻璃样式，展示星座中英文名、主要亮星、神话故事摘要 |
| 信息面板 | 缓动动画 | 弹出平滑展开，关闭缓动淡出 |
| 控制栏 | 搜索框 | 支持中文/英文星座名称实时过滤，匹配星座闪烁光晕提示，不匹配暗淡 |
| 控制栏 | 可见性切换 | 全部/仅亮星模式切换 |
| 控制栏 | 重置视角 | 恢复默认视角 |
| 控制栏 | 展开收起动画 | 搜索框平滑展开/收起，平板端折叠为汉堡菜单 |

## 3. 核心流程

用户打开页面 → 看到星空全景 → 悬停探索各星座（高亮反馈）→ 点击感兴趣的星座 → 信息面板弹出查看详情 → 使用搜索框查找特定星座 → 切换显示模式或重置视角

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝（#0a0e2a）到紫黑（#1a0a2e）径向渐变背景
- **强调色**：亮黄色（#ffd700）用于高亮星座，淡蓝色（#6eb5ff）用于默认星座连线
- **辅助色**：半透明白色用于毛玻璃面板（rgba(255,255,255,0.08)），暗淡灰色用于非活跃星座
- **字体**：展示字体用 Cinzel（优雅衬线），正文用 Noto Sans SC（中文支持）
- **布局**：全屏 Canvas + 覆盖层 UI（控制栏顶部、信息面板右下角）
- **动效**：极慢自转、悬停闪烁光晕、面板缓动弹出/淡出、搜索框展开动画
- **视觉调性**：神秘、优雅的占星学风格，星空深邃、星光闪烁

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 星图主场景 | Canvas 星空 | 深蓝-紫黑径向渐变、随机星光粒子（微弱闪烁）、30秒自转动画 |
| 星图主场景 | 星座连线 | 淡蓝色半透明线条（默认）、亮黄色粗线条（高亮）、灰色半透明（暗淡） |
| 星图主场景 | 亮星 | 2-3等星带径向光晕（默认蓝色），高亮时放大+闪烁（金黄色） |
| 信息面板 | 面板容器 | 右下角固定、backdrop-blur 毛玻璃、圆角、阴影、滑入动画 |
| 信息面板 | 星座名称 | Cinzel 字体英文名 + Noto Sans SC 中文名，金色大标题 |
| 信息面板 | 亮星列表 | 小型标签式亮星名称列表 |
| 信息面板 | 神话故事 | 半透明文本区域，可滚动 |
| 控制栏 | 搜索框 | 半透明输入框，Cinzel 字体占位符，展开/收起动画 |
| 控制栏 | 切换按钮 | 图标按钮（眼睛图标），半透明背景 |
| 控制栏 | 重置按钮 | 图标按钮（指南针图标），半透明背景 |
| 控制栏 | 平板折叠 | ≤768px 时折叠为汉堡菜单图标 |

### 4.3 响应式

- **桌面端**（>768px）：全屏 Canvas、控制栏完整展示、信息面板固定右下角
- **平板端**（≤768px）：控制栏折叠为汉堡菜单、信息面板宽度自适应（max-width: 90vw）
- **触控优化**：增大星座点击区域、信息面板可滑动关闭

### 4.4 3D 场景指导

- **环境**：深空星云氛围，深蓝到紫黑径向渐变
- **光照**：无明显光源，星点自发光+光晕效果
- **相机**：2D 俯视投影，极慢自转（30秒/圈）
- **交互**：悬停高亮、点击选中、搜索闪烁
- **后处理**：星点辉光（bloom）效果通过 Canvas radialGradient 模拟
