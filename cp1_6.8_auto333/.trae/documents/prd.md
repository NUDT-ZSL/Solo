## 1. 产品概述

「情绪回声」是一个情绪记录与共鸣的全栈 Web 应用，用户可以记录每日情绪片段（文字、表情符号、颜色标签），这些情绪碎片以声波波形的形式展示在个人时间线上。用户还可以浏览其他用户匿名分享的情绪波形，并给予「共鸣」反馈。

- 目标用户：希望记录和表达情绪、寻找情感共鸣的人群
- 核心价值：将抽象情绪具象化为视觉波形与声音，创造沉浸式的情绪表达与连接体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 匿名用户 | 无需注册 | 浏览全局情绪波形 |
| 注册用户 | 用户名注册 | 记录情绪、编辑/删除自己的情绪碎片、给予共鸣 |

### 2.2 功能模块

1. **首页**：全局情绪波形图展示、情绪记录入口
2. **个人主页**：个人情绪时间线、情绪碎片管理

### 2.3 页面详情

| 页面 | 模块 | 功能描述 |
|------|------|----------|
| 首页 | 全局情绪波形图 | 展示所有用户匿名分享的情绪碎片，每个碎片为半透明发光波形脉冲，按时间排列；点击波形播放对应情绪声音和显示文字 |
| 首页 | 情绪记录面板 | 选择颜色（预设色板）、选择表情符号、输入文字描述，保存后用 Web Audio API 合成短音效 |
| 首页 | 波形交互 | 点击波形播放情绪声音、显示文字内容、查看共鸣列表、给予共鸣 |
| 个人主页 | 情绪时间线 | 按日期排列用户的情绪碎片，每个碎片带日期标题；点击可编辑或删除 |
| 个人主页 | 情绪编辑 | 编辑已有情绪碎片的文字、颜色、表情；删除情绪碎片 |
| 共鸣弹窗 | 共鸣列表 | 查看给予共鸣的用户列表及标签；给予共鸣并选择标签（"同感"、"想听更多"、"心疼"、"加油"等） |

## 3. 核心流程

用户打开应用 → 浏览全局情绪波形图 → 点击"记录情绪" → 选择颜色和表情、输入文字 → 保存并生成情绪声音 → 波形出现在时间线上 → 其他用户点击波形 → 查看内容并给予共鸣 → 用户查看共鸣列表

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：极光风格（Aurora Borealis）
- **主色调**：深空蓝 (#0a0e27) 到紫色 (#1a0533) 渐变背景
- **辅助色**：青色光晕 (#00d4ff)、紫粉光晕 (#ff00ff)、绿色极光 (#00ff88)
- **卡片风格**：毛玻璃圆角卡片（白色半透明质感，backdrop-blur）
- **波形视觉**：半透明发光曲线带光晕（glow effect），悬停时微微脉动
- **按钮风格**：柔和光晕扩散动画，hover 时发光扩散
- **字体**：标题使用 "Orbitron" 科技感字体，正文使用 "Noto Sans SC" 中文友好字体
- **布局风格**：居中卡片式布局，波形占据主体视觉空间
- **动画**：60fps 流畅动画，波形呼吸脉动、光晕扩散、卡片淡入

### 4.2 页面设计概览

| 页面 | 模块 | UI 元素 |
|------|------|---------|
| 首页 | 顶部导航 | Logo "情绪回声" + 导航链接（首页/个人主页）+ 记录情绪按钮，毛玻璃背景 |
| 首页 | 全局波形区域 | Canvas 绘制的声波波形图，半透明发光脉冲，点击交互，深空蓝渐变背景 |
| 首页 | 情绪记录面板 | 模态弹窗，毛玻璃卡片，预设色板（8色），表情选择器（12个表情），文本输入框，保存按钮 |
| 首页 | 波形详情弹窗 | 毛玻璃卡片，显示情绪文字、颜色、表情、时间，播放按钮，共鸣列表，给予共鸣按钮 |
| 个人主页 | 情绪时间线 | 垂直时间线布局，按日期分组，每个碎片为波形卡片，带日期标题 |
| 个人主页 | 编辑/删除操作 | 点击卡片弹出编辑面板，支持修改颜色/表情/文字，删除需确认 |

### 4.3 响应式设计

- 桌面端（≥1024px）：波形区域宽度 80%，卡片最大宽度 600px
- 平板端（768px-1023px）：波形区域宽度 90%，卡片自适应
- 移动端（<768px）：全宽布局，波形纵向排列，触摸优化
