## 1. 产品概述

「茶语时光」是一款品茶记录与分享的 Web 应用，旨在为茶爱好者提供一个温润优雅的数字空间，记录每一次品茶体验——从茶名、品种、冲泡参数到心情感受，并以卡片墙形式分享发现。目标用户为热爱品茶、追求生活美学的用户群体，核心价值在于将品茶的仪式感延伸至数字世界。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 默认用户（无鉴权） | 创建、编辑、删除自己的品茶记录；浏览所有公开记录；查看个人统计 |

### 2.2 功能模块
1. **首页**：品茶卡片墙（瀑布流布局）、品种/心情标签筛选
2. **详情页**：品茶完整记录、温度曲线图、浸泡时间进度条、评论区
3. **个人页**：品茶日历、月度品茶次数柱状图、偏好品种饼图
4. **创建/编辑**：品茶记录表单（茶名、品种、温度、浸泡时间、笔记、心情标签、星级评分、图片上传）

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 卡片墙 | 瀑布流展示所有公开品茶记录卡片，按时间倒序排列，支持按品种和心情标签筛选 |
| 首页 | 筛选栏 | 品种下拉筛选 + 心情标签多选筛选 |
| 详情页 | 品茶信息 | 展示茶名、品种、冲泡温度、浸泡时间、品鉴笔记、心情标签、星级评分 |
| 详情页 | 温度曲线图 | Canvas 绘制冲泡温度折线图 |
| 详情页 | 浸泡时间进度条 | 可视化展示浸泡时间 |
| 详情页 | 评论区 | 查看和发表评论 |
| 详情页 | 操作按钮 | 编辑、删除品茶记录 |
| 个人页 | 品茶日历 | 月历视图，标红有记录的日子 |
| 个人页 | 月度统计 | Canvas 柱状图展示每月品茶次数 |
| 个人页 | 品种偏好 | Canvas 饼图展示偏好品种分布 |
| 创建/编辑页 | 表单 | 输入茶名、品种、温度、浸泡时间、品鉴笔记、心情标签、星级评分，上传茶汤照片 |

## 3. 核心流程

用户打开应用后进入首页卡片墙，浏览所有公开品茶记录；点击卡片进入详情页查看完整信息和图表；用户可创建新的品茶记录（含图片上传）；在个人页查看自己的品茶日历和统计图表。

    "B" --> "L[进入个人页]"

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：米白（#FAF6F0）到浅茶色（#D4B896）渐变背景
- **辅助色**：深茶褐（#6B4226）、暖金（#C8A96E）、淡绿（#8BA888）
- **卡片风格**：半透明磨砂玻璃（backdrop-filter: blur）圆角方块，柔和阴影和微光晕
- **按钮风格**：圆角胶囊按钮，悬停时光晕扩散效果（box-shadow 扩散）
- **字体**：标题使用衬线体（Noto Serif SC），正文使用无衬线体（Noto Sans SC）
- **布局风格**：顶部毛玻璃导航栏，首页瀑布流卡片墙，个人页图表居中
- **图标风格**：lucide-react 图标库
- **动画**：卡片悬停微微上浮（translateY），页面切换缓动过渡（CSS transition），交互反馈流畅

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 导航栏 | 毛玻璃材质，发光底边，居中 Logo + 导航链接 |
| 首页 | 筛选栏 | 品种下拉 + 心情标签胶囊按钮组 |
| 首页 | 卡片墙 | 瀑布流三列（桌面）/ 两列（平板）/ 单列（手机），卡片磨砂玻璃 + 上浮动效 |
| 详情页 | 信息区 | 大图 + 茶名标题 + 星级 + 心情标签胶囊 |
| 详情页 | 图表区 | 温度曲线折线图 + 浸泡时间进度条 |
| 详情页 | 评论区 | 评论列表 + 输入框 |
| 个人页 | 日历区 | 月历网格，有记录日期标红 |
| 个人页 | 图表区 | 柱状图（月度次数）+ 饼图（品种偏好）并排居中 |
| 创建/编辑页 | 表单 | 分组表单，星级评分交互，图片上传预览 |

### 4.3 响应式适配
- 桌面端（≥1024px）：卡片三列瀑布流，图表并排，宽屏舒适布局
- 平板端（768-1023px）：卡片两列，图表上下排列
- 移动端（<768px）：卡片单列，汉堡菜单导航，触摸优化
