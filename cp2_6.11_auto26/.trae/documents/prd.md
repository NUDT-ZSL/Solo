## 1. 产品概述

「森语棋局」是一款基于浏览器的策略战棋游戏，玩家控制由森林精魂幻化的棋子，在随机生成的迷雾棋盘上探索并占领灵脉节点，与AI对手竞逐领地控制权。

- 目标用户：策略游戏爱好者、休闲棋类玩家
- 核心价值：迷雾探索+领土争夺的独特策略体验，森林主题暗色系沉浸式视觉

## 2. 核心功能

### 2.1 用户角色
| 角色 | 控制方式 | 核心权限 |
|------|----------|----------|
| 玩家（翠绿方） | 鼠标点击操控 | 移动棋子、召唤棋子、施放法术、结束回合 |
| AI（琥珀方） | 自动决策 | 移动棋子、召唤棋子、施放法术 |

### 2.2 功能模块
1. **游戏主界面**：8x8迷雾棋盘、棋子渲染、灵脉节点、回合信息、操作按钮
2. **迷雾探索系统**：视野机制、迷雾消散/回覆盖动画
3. **棋子与占领系统**：棋子移动、灵脉占领倒计时、脉冲光效
4. **召唤与法术系统**：法力值管理、召唤新棋子、藤蔓缠绕法术
5. **回合与胜负系统**：20回合制、30秒限时、胜负判定

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主界面 | 迷雾棋盘 | 8x8网格，格子类型随机生成（灵脉/空地/荆棘），迷雾覆盖与消散 |
| 游戏主界面 | 棋子系统 | 4个初始棋子，上下左右移动一格，灵脉占领倒计时3秒 |
| 游戏主界面 | 召唤法术 | 消耗2法力召唤棋子（粒子动画0.6秒），消耗3法力施放藤蔓缠绕（动画0.8秒） |
| 游戏主界面 | 回合管理 | 20回合制，30秒限时，圆形进度条倒计时，回合切换提示 |
| 游戏主界面 | 胜负判定 | 比较双方占领灵脉数，多者获胜 |

## 3. 核心流程

## 4. 界面设计

### 4.1 设计风格
- 主色调：深林绿(#0B1A10)到暗松石绿(#0D2B20)渐变
- 点缀色：翠绿(#3CB371)、琥珀(#FFB347)、淡金色(#FFD700)、浅绿色(#A8E6CF)
- 按钮风格：圆角矩形，深绿到墨绿渐变背景，悬停亮度+20%，点击缩小5%
- 字体：水墨风格标题，游戏UI字体
- 布局：居中棋盘，右上角倒计时，底部操作按钮
- 边框：2px深褐色(#2C1810)格子边框

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主界面 | 棋盘区域 | 70x70px格子，径向渐变迷雾，灵脉脉冲光，选中光晕旋转 |
| 游戏主界面 | 棋子 | 50px直径圆形，荧光效果，0.5px阴影(模糊4px) |
| 游戏主界面 | 回合倒计时 | 右上角60px圆形进度条，绿→红渐变，中央秒数 |
| 游戏主界面 | 操作按钮 | 圆角矩形，深绿渐变，悬停亮度提升，点击缩放 |
| 游戏主界面 | 回合提示 | 居中淡入淡出，浅绿色#A8E6CF，1.2秒 |

### 4.3 响应式
- 桌面优先设计，棋盘固定8x8布局
- Canvas尺寸根据窗口自适应缩放

### 4.4 动画规格
| 动画 | 持续时间 | 说明 |
|------|----------|------|
| 迷雾消散 | 500ms | 从#1A2A1A到#2A3A2A再透明 |
| 迷雾回覆盖 | 250ms | 消散速度的2倍 |
| 灵脉脉冲 | 1.5秒周期 | 亮度50%-100%循环 |
| 灵脉占领 | 3秒 | 停留3秒后占领 |
| 召唤粒子 | 0.6秒 | 从中心向外扩散 |
| 藤蔓缠绕 | 0.8秒 | 绿色藤蔓包裹动画 |
| 回合提示 | 1.2秒 | 淡入淡出 |
| 选中光晕 | 1.5秒周期 | 旋转淡金色#FFD700 |
| 符文光晕 | 持续 | 回合切换时棋盘边缘旋转 |
| 按钮点击 | 100ms | 缩小5%再弹回 |
