# 星尘矿脉 - 多人协作太空采矿模拟器

一款基于HTML5 Canvas和WebSocket的多人协作太空采矿游戏。

## 游戏特色

- 🚀 **4人多人协作**：最多支持4名玩家同时游戏
- 🌌 **实时星云背景**：使用Perlin噪声算法实时生成的星云背景，缓慢漂移效果
- ⛏️ **采矿系统**：三种矿石类型（铁质#b5651d、铜质#cd7f32、稀有晶矿#00e5ff）
- 💎 **升级系统**：货舱扩容、护盾强化、采矿效率升级
- ☄️ **流星空袭**：每30秒一次流星警报，小心陨石撞击！
- 💬 **实时聊天**：游戏内聊天系统，按Enter发送消息
- 🏆 **结算系统**：30分钟倒计时，游戏结束后显示详细排名和评语

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 鼠标左键 | 点击小行星进行开采 |
| Enter | 打开/发送聊天 |
| ESC | 关闭升级面板 / 取消聊天 |

## 游戏玩法

1. **探索**：驾驶飞船在随机生成的星云矿区中探索
2. **采矿**：鼠标左键点击小行星，飞船会自动靠近并开采
3. **运输**：将开采的矿石运回屏幕中央的基地
4. **升级**：靠近基地自动弹出升级界面，消耗矿石升级飞船
5. **躲避**：注意流星空袭警报，及时躲避流星

## 升级选项

| 升级类型 | 消耗 | 效果 |
|---------|------|------|
| 货舱扩容 | 10铁矿 + 5铜矿 | 货舱容量+5 |
| 护盾强化 | 8晶矿 | 护盾上限100→150 |
| 采矿效率 | 6晶矿 + 4铁矿 | 开采速度10%/s→15%/s |

## 技术栈

- **前端**：TypeScript + HTML5 Canvas + Vite
- **后端**：Node.js + Express + ws (WebSocket)
- **渲染**：2D Canvas API，requestAnimationFrame驱动
- **同步**：WebSocket实时状态同步

## 运行要求

- Node.js >= 16.0.0
- npm >= 8.0.0
- 支持HTML5 Canvas的现代浏览器

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器（同时启动后端和前端）
npm run dev
```

启动后在浏览器中打开 `http://localhost:5173` 即可开始游戏。
