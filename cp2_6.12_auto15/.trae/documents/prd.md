## 1. 产品概述

徒步路线记录与分享社区——一个面向户外爱好者的轻量级平台，用户可在地图上绘制轨迹、上传沿途照片、浏览和收藏他人路线，解决目前缺乏可在地图上标注轨迹并上传照片的社区工具的问题。

- 目标用户：户外徒步爱好者、登山者、自然探索者
- 核心价值：让户外路线记录变得简单直观，通过社区分享发现更多精彩路线

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 默认用户（无需注册，使用浏览器标识） | 创建轨迹、上传照片、浏览和收藏路线、管理个人轨迹 |

### 2.2 功能模块

1. **地图页面**：Leaflet地图渲染、轨迹绘制工具栏（折线/多边形模式）、轨迹保存与加载、节点拖动微调
2. **探索页面**：公开轨迹卡片网格、搜索框（标题模糊搜索）、筛选下拉框（时间/收藏/距离）、收藏按钮
3. **轨迹详情页**：完整轨迹地图、照片标记点、标题/创建者/描述、照片画廊（灯箱模式）、收藏按钮与动画
4. **个人空间**：我的轨迹列表、收藏的轨迹列表、编辑标题、删除轨迹（弹窗确认）

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 地图页面 | 地图组件 | 渲染Leaflet地图，支持缩放和平移，地图占页面主体60%以上 |

## 3. 核心流程

### 3.1 轨迹绘制与保存流程
用户打开地图页面 → 选择绘制模式（折线/多边形）→ 在地图上点击创建轨迹点 → 按住Shift连续绘制 → 双击结束绘制 → 拖动节点微调 → 点击保存 → 输入标题 → POST GeoJSON数据到后端 → Toast提示保存成功

### 3.2 照片上传流程
进入轨迹详情页 → 点击上传照片 → 选择文件（最多5张，每张5MB）→ 显示进度条动画 → 上传完成 → 缩略图显示在照片画廊中

### 3.3 浏览与收藏流程
用户打开探索页面 → 浏览轨迹卡片列表 → 使用搜索框和筛选条件 → 点击卡片进入详情页 → 查看完整轨迹和照片 → 点击收藏按钮 → 按钮高亮变红，数字动画增加 → Toast提示收藏成功

### 3.4 个人空间管理流程
进入个人空间 → 查看我的轨迹/收藏列表 → 编辑标题（内联编辑）→ 删除轨迹（弹窗确认）→ 后端删除轨迹及关联照片文件 → 列表更新 → Toast提示删除成功

## 4. 界面设计

### 4.1 设计风格

- **主色调**：墨绿（#2d5016）和米白（#f5f0e8），辅以橙黄（#e8a838）强调色
- **风格**：户外自然风格，温暖质朴，有机纹理感
- **按钮风格**：圆角矩形按钮，墨绿底色白色文字，强调按钮使用橙黄色
- **字体**：正文使用系统无衬线字体 14px-16px，标题粗体 18px，页面大标题 24px
- **布局风格**：左侧导航/工具栏 + 主内容区，卡片网格布局
- **图标风格**：使用Lucide图标库，线条风格，与Leaflet内置风格协调

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 地图页面 | 地图区域 | 占页面60%以上，Leaflet地图，默认缩放级别12 |
| 地图页面 | 绘制工具栏 | 左侧浮动，半透明毛玻璃效果（backdrop-filter: blur），包含折线/多边形/保存按钮 |
| 地图页面 | 保存弹窗 | 居中模态框，输入标题和描述，墨绿边框 |
| 探索页面 | 搜索筛选栏 | 顶部固定，搜索框+下拉筛选，米白背景墨绿边框 |
| 探索页面 | 卡片网格 | 每张280px宽，圆角12px，阴影0 2px 8px rgba(0,0,0,0.1)，hover阴影加深上浮5px，淡入动画 |
| 轨迹详情页 | 照片画廊 | 左侧缩略图列表，全屏灯箱查看，暗色背景，左右滑动切换动画，右上角关闭按钮 |
| 轨迹详情页 | 收藏按钮 | 底部固定，橙黄色心形图标，已收藏变红色填充，数字变化动画 |
| 个人空间 | 轨迹列表 | 时间倒序卡片列表，内联编辑标题，删除按钮带确认弹窗 |
| 全局 | Toast提示 | 右上角出现，2.5秒后自动消失，消失时缩放淡出动画 |

### 4.3 响应式设计

- 桌面优先设计，中屏（768px）以下自动调整为单列布局
- 地图工具栏在小屏下折叠为底部抽屉式菜单
- 卡片网格在小屏下变为单列
- 照片画廊灯箱在小屏下支持滑动手势切换

### 4.4 动效设计

- 轨迹线渐变色：随海拔变化从绿到红
- 卡片hover：阴影加深+上浮5px（transition 0.3s ease）
- 卡片列表更新：淡入动画（fadeIn 0.4s ease）
- 收藏按钮：数字变化时弹跳动画（scale 1.2 → 1.0）
- Toast提示：滑入→2.5秒停留→缩放淡出（scale 0.8 + opacity 0）
- 照片灯箱：左右滑动切换（translateX动画）
- 进度条：上传时从0%到100%平滑填充动画
