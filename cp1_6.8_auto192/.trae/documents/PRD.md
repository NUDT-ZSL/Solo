## 1. 产品概述

「墨光流影」是一个基于 Three.js 的 3D 交互式水墨扩散可视化项目，模拟墨水在水中扩散与流动的过程。用户通过鼠标点击和拖拽投放不同颜色的墨滴，观察其在水流中扩散、交织形成抽象图案。

- 目标用户：艺术爱好者、交互设计从业者、创意编程学习者
- 核心价值：提供沉浸式的水墨交互体验，将东方美学与数字技术融合

## 2. 核心功能

### 2.1 功能模块

1. **主画布页面**：全屏 3D 画布，墨滴交互与水流模拟
2. **控制面板**：颜色选择器（预设三色）和清除按钮

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主画布 | 墨滴投放 | 鼠标点击/拖拽在画布上投放墨滴，生成粒子 |
| 主画布 | 粒子扩散 | 粒子受水流影响扩散，颜色渐变混合，形成水墨效果 |
| 主画布 | 水流模拟 | 基于向量场生成水流，随时间变化产生涡旋和涌动 |
| 控制面板 | 颜色选择 | 预设红橙、蓝紫、青绿三色，点击切换当前墨色 |
| 控制面板 | 清除按钮 | 一键清除所有粒子，恢复静水状态 |

## 3. 核心流程

用户打开页面 → 看到深蓝渐变静水背景 → 右下角毛玻璃控制面板选择墨色 → 在画布上点击/拖拽投放墨滴 → 墨滴粒子受水流影响扩散 → 多色墨滴交叉混合 → 形成抽象水墨图案 → 可随时切换颜色或清除

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝渐变（#0a0e27 → #1a1a4e）模拟静水深潭
- 强调色：红橙（#ff6b35）、蓝紫（#7b68ee）、青绿（#00d4aa）
- 按钮/面板风格：半透明毛玻璃（backdrop-filter: blur），圆角，发光边框
- 字体：Noto Serif SC（衬线，东方美学）
- 布局：全屏画布，右下角浮动控制面板

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主画布 | 3D画布 | 全屏 WebGL 画布，深蓝渐变背景 |
| 主画布 | 粒子渲染 | 半透明发光圆点，拖尾光晕，additive blending |
| 控制面板 | 颜色选择器 | 三个圆形色块，选中态带发光边框 |
| 控制面板 | 清除按钮 | 毛玻璃背景，圆角，hover 发光 |

### 4.3 响应式

- 桌面优先，全屏画布自动适配窗口尺寸
- 控制面板在小屏幕上缩小并移至底部中央
- 窗口 resize 时自动更新相机和渲染器

### 4.4 3D 场景指引

- 环境：深蓝渐变背景，模拟静水深潭
- 灯光：无需复杂灯光，粒子使用自发光材质
- 相机：正交相机，俯视视角，适合 2D 水面效果
- 交互：鼠标点击/拖拽投放墨滴，粒子受水流影响运动
- 后处理：粒子发光效果（additive blending + 半透明）
- 性能预算：粒子数 ≤ 5000，帧率 ≥ 60fps
