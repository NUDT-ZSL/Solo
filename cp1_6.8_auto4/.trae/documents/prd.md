## 1. 产品概述

「万象拼图」是一款 2D 重组解谜游戏。玩家面对一幅被打散成不规则碎片的动态抽象油画，通过拖拽碎片到正确位置还原完整画面。每个碎片带有微弱色调偏移和随机旋转干扰，还原后整幅画闪烁并触发下一关更复杂的画作。

- 目标用户：休闲解谜游戏爱好者、视觉艺术爱好者
- 核心价值：将艺术欣赏与解谜玩法融合，提供沉浸式的视觉体验和逐步升级的挑战

## 2. 核心功能

### 2.1 功能模块

1. **游戏画板页面**：碎片拼图区域、画作渲染、拖拽交互
2. **关卡选择页面**：关卡列表、已通关标记、难度预览

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏画板 | 碎片区域 | 显示打散的碎片，支持拖拽、弹性缓动、吸附锁定 |
| 游戏画板 | 画作渲染 | ArtGenerator 生成的抽象油画，持续缓慢变化 |
| 游戏画板 | 毛玻璃控制面板 | 显示当前关卡、耗时、得分，右下角悬浮 |
| 游戏画板 | 完成动画 | 碎片微光脉动 → 画面闪烁 → 淡入淡出过渡到下一关 |
| 关卡选择 | 关卡列表 | 展示可用关卡，碎片数量从4到36递增 |
| 关卡选择 | 难度预览 | 显示每关碎片数量和风格类型 |

## 3. 核心流程

玩家进入游戏 → 选择关卡或从第1关开始 → ArtGenerator生成随机抽象油画 → FragmentManager将画作切割为不规则碎片并打乱（含色调偏移和随机旋转） → 玩家拖拽碎片到正确位置 → 接近正确位置时自动吸附锁定 → 全部碎片归位后触发完成动画（微光脉动→闪烁→1秒淡入淡出） → 进入下一关（碎片更多、形状更不规则）

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：极简黑灰主题 — 背景 #0a0a0a，面板 rgba(20,20,20,0.8)
- **辅助色**：碎片悬停发光边框 — 柔和白色光晕 rgba(255,255,255,0.3)，锁定成功闪光 rgba(100,200,255,0.6)
- **按钮风格**：圆角矩形，半透明背景，微妙投影
- **字体**：显示字体 'Cormorant Garamond'（优雅衬线），UI字体 'DM Sans'（现代无衬线）
- **布局风格**：中心画板布局，右下角毛玻璃控制面板
- **特效**：画板柔光边缘和微妙投影、碎片悬停发光边框、完成时微光脉动

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏画板 | 碎片区域 | 黑色背景，中心画板600x600px，碎片带投影和发光边框 |
| 游戏画板 | 控制面板 | 毛玻璃效果面板，显示关卡/耗时/得分，右下角定位 |
| 游戏画板 | 完成动画 | 碎片白色脉动光晕 → 画面白闪 → 1秒淡入淡出过渡 |
| 关卡选择 | 关卡网格 | 3列网格，每格显示关卡编号、碎片数、风格标签 |

### 4.3 响应式

- 桌面优先设计，最小宽度 1024px
- 画板尺寸随窗口缩放，保持正方形比例
- 控制面板固定在右下角
