## 1. 产品概述

社区共享工坊工具管理系统，解决工坊工具借用登记混乱、逾期未还追踪困难、损坏报修无记录等痛点。通过数字化管理实现工具状态透明化、借用流程规范化、维修记录可追溯，提升社区工坊运营效率。

- 目标用户：社区共享工坊会员（工具借用者）和工坊管理员
- 核心价值：替代纸质登记本，实现工具全生命周期管理（借出→使用→归还→维修）

## 2. 核心功能

### 2.1 用户角色

| 角色 | 说明 | 核心权限 |
|------|------|----------|
| 会员 | 注册工坊的社区成员 | 浏览工具、发起借用、查看个人记录、提交使用反馈 |
| 管理员 | 工坊运营人员 | 查看所有维修工单、标记工具已修复 |

### 2.2 功能模块

1. **首页（工具浏览）**：工具搜索、工具卡片列表、工具详情弹窗
2. **借用申请页**：工具信息展示、借用时长选择、提交借用申请
3. **个人借用记录页**：借用记录列表、倒计时显示、逾期提醒、使用反馈提交
4. **管理员维修工单页**：待修工单列表、工单详情、标记修复操作

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 搜索栏 | 关键词搜索工具名称/分类 |
| 首页 | 工具卡片 | 宽220px高280px，渐变背景，显示工具照片（圆形60px）、名称、状态标签 |
| 首页 | 详情弹窗 | 底部滑入，模糊遮罩0.5，显示使用说明和最近三次借用记录 |
| 借用申请页 | 工具信息区 | 大图展示工具详情、状态、基本信息 |
| 借用申请页 | 时长选择器 | 按小时选择：1/2/4/8/24小时选项 |
| 借用申请页 | 提交表单 | 确认借用信息，提交后生成借用记录 |
| 个人借用记录页 | 记录卡片 | 显示借用工具、时长、精确到秒的倒计时（红色#e53935） |
| 个人借用记录页 | 逾期状态 | 超时未还状态变逾期，每30秒闪动红色警告边框 |
| 个人借用记录页 | 反馈表单 | 状态选择（正常/轻微磨损/损坏需修理）+ 100字评价 |
| 管理员维修页 | 工单列表 | 显示待修工具、报告人、描述、生成时间 |
| 管理员维修页 | 标记修复 | 点击后工具恢复可用，从维修列表移除 |

## 3. 核心流程

会员浏览首页工具 → 点击工具卡片查看详情 → 发起借用申请选择时长 → 系统生成带倒计时的借用记录 → 会员使用后归还提交状态反馈 → 如标记损坏则自动生成维修工单 → 管理员查看工单并标记修复 → 工具恢复可用状态

    I -->|正常/磨损| J["记录归档"]

    K --> L["管理员查看工单"]

    M --> N["工具恢复可用"]

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：浅绿色（#e8f5e9 → #c8e6c9渐变）+ 白色为主
- **状态色**：可用绿色#4caf50、已借出橙色#ff9800、维修中红色#f44336、倒计时红色#e53935
- **卡片样式**：圆角12px，悬浮时阴影加深+上移3px，过渡动画0.3s ease-out
- **导航栏**：居中高度56px，白色背景，底部1px浅灰#e0e0e0分割线
- **弹窗**：底部滑入动画，模糊背景遮罩透明度0.5

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 导航栏 | 白色导航，Logo居中，移动端汉堡菜单 |
| 首页 | 搜索框 | 圆角输入框，带搜索图标 |
| 首页 | 工具卡片网格 | 响应式布局，等间距排列 |
| 首页 | 详情弹窗 | 底部抽屉式，背景模糊遮罩 |
| 借用申请页 | 工具展示区 | 大图+详情信息卡片 |
| 借用申请页 | 时长选择 | 胶囊式按钮组，选中高亮 |
| 借用申请页 | 提交按钮 | 绿色主按钮，悬浮动效 |
| 个人记录页 | 倒计时显示 | 大号红色字体，精确到秒 |
| 个人记录页 | 逾期边框 | 红色边框每30秒闪烁动画 |
| 个人记录页 | 反馈表单 | 下拉选择+文本域+提交按钮 |
| 管理员页 | 工单卡片 | 带时间戳，操作按钮右对齐 |

### 4.3 响应式设计
- 桌面端：顶部居中导航栏，网格布局多列展示
- 移动端（<768px）：顶部导航栏变为汉堡菜单，卡片单列布局
- 触控优化：按钮最小点击区域44px，列表项间距适配拇指操作

### 4.4 性能要求
- 所有列表页渲染必须在200ms内完成
- 倒计时使用requestAnimationFrame或setInterval优化
- 图片懒加载，工具照片使用合适尺寸
